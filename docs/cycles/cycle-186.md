# サイクル-186

このサイクルでは、移行計画 Phase 5（B-331）として、新デザイン用の検索コンポーネント（`src/components/Search*/`）を実装し、(new) ヘッダーの actions スロットへ結線する。Cmd+K / Ctrl+K キーバインドへの対応、モーダル維持／独立ページ化（`/search?q=...`）／両立のいずれを採るかの実装方針判断、ランキングロジックの `src/lib/search/` 共有層への切り出し、B-340 計測コードの新コンポーネントへの接続、(legacy) との共存パスの設計を含む。
