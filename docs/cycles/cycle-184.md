# サイクル-184

このサイクルでは、B-389「ブログのタグ UI 完全廃止」の本体実装を行います。cycle-183 で導入したホットフィックス（>=3 件タグのみ表示する UI フィルタ）を撤去し、ブログのタグ pill UI とタグ系ルートを完全削除することで、404 リンク混在の暫定状態を解消します。frontmatter の `tags` フィールドは残置するため、将来の検索精度向上やシリーズ・カテゴリ的な情報源としての再活用余地は保たれます。

着手前に GA 実測 2 軸を計測し、想定通り X1（タグ UI 完全廃止）が来訪者価値の観点で最適か、それとも X4（タグ機能を能動的に強化）に切り替えるべきかを最終判定します。
