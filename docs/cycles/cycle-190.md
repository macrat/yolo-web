# サイクル-190

`docs/design-migration-plan.md` の Phase 7（ツール・遊び詳細ページの新デザイン移行 + タイル化）に着手する。Phase 7 は 30+ ツールと 20+ 遊び（result ページ含む）を「1 コンテンツ 1 サイクル」で順次移行する長期工程で、本サイクルではその第 1 サイクル分として 1 コンテンツ（ツール 1 件または遊び 1 件）を移行する。

本サイクルは Phase 7 の最初のサイクルとなるため、後続サイクルのテンプレ／観察手順／チェック項目を確立する役割も持つ。具体的にどのコンテンツから着手するか（visitor 価値の高さ・タイル化適合性・PV / GA4 データ・移行リスクの組み合わせで判断）、および Phase 7 のテンプレ手順をどう構造化するかは `/cycle-planning` で確定する。

併せて Phase 7 着手前の整合作業として、`docs/design-migration-plan.md` Phase 7.1 / 7.2 の数値（L170「30 ルート」→ 実体 34 ルート、L173「13 ルート + result ページ」→ 実体 20 ルート、ゲーム列挙 13 件 → quizEntries 15 件等）を実体ベースで訂正する（B-365）。
