## 締め(tighten)

現行 `eslint .` の jsx-a11y 警告は0(next core-web-vitals 既定)。F1–F4是正後、clean な core a11y ルールを warn→error に昇格して回帰を**ビルド時ブロック**で固定する(散文でなく構造で塞ぐ=研究doc準拠)。誤検知を含むルール(`<dialog>` の click-events-have-key-events 等=実操作でキーボード操作完遂を確認済の誤検知)は昇格対象外。
