# サイクル-212

このサイクルでは、`image-resizer` ツールの詳細ページを新デザインに移行し、ツールタイルを新規実装する。**画像入力型タイル 2 件目**として、cycle-211（image-base64）で確立した補足事項 (i)〜(x) の SSoT を初めて引用適用し、N≥2 件の蓄積によって画像入力型タイル AP-P21 (v) 基準値（経験的暫定値 ±15%）の妥当性を一段階検証する。FileReader 非同期 + ファイル選択 UI + 「未選択時 / 選択後」UI 構造変化という画像入力型の構造的特徴を再現しつつ、Canvas 描画 + blob 出力（リサイズ後画像の DL）という image-base64 と異なる出力経路を扱う。
