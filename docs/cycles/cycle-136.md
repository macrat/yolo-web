# サイクル-136

GAデータ分析の結果、play系コンテンツ（20種）はインタラクティブな構造のためテキストコンテンツが少なく、オーガニック検索からの流入がほぼゼロであることが判明。直近2週間の107セッション中、play個別ページへの検索流入は確認できない。

一方、辞典ページやブログ記事はテキスト量が豊富なため、相対的に検索からの流入がある（/dictionary 12PV、/blog/memo-system-rise-and-fall 9PV）。

このサイクルでは、play系コンテンツページに説明テキスト・FAQ・構造化データを追加し、検索エンジンからの発見可能性を高める。メインターゲット（手軽で面白い占い・診断を楽しみたい人）が使う検索クエリ（「性格診断 無料」「心理テスト おもしろ」等）での表示を目指す。
