- ルール4は「**すべてのコンテンツが、あらゆる面で来訪者にとって最高の品質**であり、探索しやすく整理されていること」を求める。ところが私の決定は、**来訪者の半数（legacy arm）に、プロジェクトが約40サイクルかけて『移行すべき』と判断してきた旧デザインを、意図的に、しかも無期限に配り続ける**ものだった（しかもプール実験には有効な結論ゲートが存在せず＝B-546、終われない）。新デザインが最高品質なら、半数にわざと劣る・割れた体験を配り続けるのはルール4違反。新デザインが最高品質か分からないなら、76面をブラインド出荷した時点で既に来訪者を害しており、急ぐべきは4ゲームでなく移行全体の検証だ。私はこの二つを**非整合に跨いだ**——4ゲームは「不確実だから A/B」、76面は「一貫してるから受容」。同じ不確実性が、一方では A/B の理由になり、他方では不問の理由になっている。
