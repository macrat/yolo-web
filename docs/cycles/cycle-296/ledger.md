### 2-1. archetypeBreakdown の共有文型(cycle-284「24/24 同一文型」を現物で再測)

**結論: 「あなたの中で(は)…二つが同時に◯◯している」の**厳密な**同一文型は 15/24(異型のみ)。同型6・逆順3 は別の共有文型で、合計は「3つのテンプレ族に24/24が収まる」構造。** つまり異型の文型だけ潰しても同型・逆順のテンプレが残る(書き直し注意点)。

**族A(異型15 = #1-#15)**: 「(あなたの中では)その二つが〔動詞〕ている」。動詞を類語で回すだけ。

- 「同時に動いている」#1(L22 "この二つが同時に動いている"), #2(L47), #12(L52), #13(L75)
- 「同時に働いている」#4(L97), #11(L29), #14(L98), #15(L121)
- 「同時に走っている」#3(L71)
- 「同時に立ち上がる」#8(L194)
- 「ぶつかり合っている」#5(L122) / 「絡み合っている」#6(L146) / 「共存している」#7(L170) / 「同居している(ますの)」#9(L218), #10(L242)
- 開始句「あなたの中で(は)」は #1-#15 全てに存在。#1 のみ「この二つ」、他は「その二つ」。

**族B(同型6 = #16-#21)**: 「あなたの〔X〕と〔Y〕が極限まで強まると、…(概念が)消える/溶ける + その〔一面〕だけが二重に振り切れているのがあなた」。

- 「極限まで強まると…が消える/溶ける」共有: #16,#17,#18,#19,#20。

**族C(逆順3 = #22,#23,#24)**: 「(誰かを守りたい)本能と…力が、あなたの中で溶け合っている/同居している」。

- 「あなたの中で溶け合っている」#22(L61), #23(L87)。「あなたの中で同居している」#24(L114)。

→ **書き直し指針**: AC1 が消すべきは族Aの「あなたの中で(は)…二つが同時に◯◯」だが、**族B「二重に振り切れているのがあなた」・族C「溶け合っている/同居している」も同格の共有テンプレとして 0/24 にする**。3族すべてを解体しないとテンプレ感は残る。

## 4. 固有核の配分表(最重要)— 各タイプに「他の23には言えない核」

各タイプ: **(a)中心観察1つ**(主軸×副軸の対にだけ固有)/ **(b)behaviors が向かう場面の方向**/ **(c)助言1つの方向**(§2-2の共有助言族を割って重複ゼロに)/ **(d)キャッチの芯**。
**衝突回避の設計原理: 同一主軸内は副軸で差を出す。** 特に衝突しやすい群を先に示す。

### 4-0. 最も衝突しやすい群と差別化の骨子(先に宣言)

1. **commander主軸6(#1,2,3,4,5,16)**——全て「まず動く」。→ **副軸=動きの行き先**で割る: #1 動き=分析加速(圧力で頭が回る) / #2 動き=内省へワープ(号令直後に空を見る) / #3 動き=裏道へ強行 / #4 動き=全員牽引 / #5 動き=感性の着火(制作) / #16 動き=思考ゼロの純推進。
2. **guardian主軸4(#15,20,22,24)+ guardian副軸(#4,#8,#11,#13)**——全て「守る/備える」。→ **守り方の手段**で割る: #15 感情察知でケア / #20 物理的備え / #22 不安をデータ化 / #24 決断して前に出る / (#4 牽引で守る=commander主軸 / #8 確認で守る=professor主軸 / #11 理想世界を設計 / #13 脆弱性を塞ぐ)。
3. **「逆張り」3(#10,#14,#23)**——→ #10 夢への自己ツッコミ(照れ隠し・dreamer) / #14 **意図的**に王道を外し美しく仕上げる(計算・trickster) / #23 **逆張りのつもりなく**本能で走った結果の異端(artist)。(#14 と #23 の差別化はデータコメント L74-76 が既に宣言済=保持)
