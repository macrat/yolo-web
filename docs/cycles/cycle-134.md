## 作業計画

### 目的

play系コンテンツへの検索流入を週3PVから改善するため、検索需要が高く競合が手薄なジャンルで新しい性格診断コンテンツを1つ追加する。サイトの「言葉・日本語」というアイデンティティを活かしつつ、SNSシェアされやすい設計にすることで、検索流入とソーシャル流入の両方を狙う。

### コンテンツの選定: 言葉センス診断

5つの候補の中から「言葉センス診断」を選定する。理由は以下の通り。

**1. サイトアイデンティティとの整合性が最も高い（星5評価）**
サイトには既に漢字力診断、四字熟語力診断、ことわざ力診断、四字熟語性格診断といった「言葉・日本語」系コンテンツが複数存在する。言葉センス診断はこの系譜に自然に位置づけられ、既存コンテンツからの回遊導線が最も効果的に機能する。keywords設計により推薦ロジックが既存の言語系コンテンツと相互に推薦し合える。

**2. 競合がブルーオーシャン**
調査の結果、「語彙力診断」は正誤判定型（知識テスト）が主流で、「言葉のセンス・感性」を性格診断として分類するエンタメフォーマットはほぼ存在しない。マイナビの語彙力診断は10問の知識テスト、ことばの学校は25問の語彙量推定。いずれも「正解がある」形式であり、「言葉の選び方の好み・傾向から性格タイプを分類する」という切り口は差別化できる。

**3. SNSシェアとの相性**
「あなたの言葉センスは○○タイプ」という結果形式は、自己表現としてシェアしやすい。言葉の選び方は日常に密着しているため共感を得やすく、会話のネタになりやすい。

**4. 季節性に依存しない通年コンテンツ**
コミュニケーションスタイル診断は4月新生活シーズンに強いが季節を過ぎると需要が落ちる。言葉センス診断は季節に依存せず、通年で検索される。

**5. 既存コンテンツとの重複がない**
知識テスト系（漢字力・四字熟語力・ことわざ力）は「正解がある」形式。言葉センス診断は「好み・感性」による性格分類であり、カテゴリも形式も完全に異なる。

### コンテンツの詳細設計

**タイトル**: あなたの言葉センス診断
**slug**: `word-sense-personality`
**shortTitle**: 言葉センス診断
**type**: personality
**category**: personality
**questionCount**: 10問
**icon**: 言葉・文字をイメージするもの（実装時に決定）
**accentColor**: 実装時に既存コンテンツと被らない色を選定

**コンセプト**: 日常の場面で「どんな言葉を選ぶか」「どんな表現に惹かれるか」から、あなたの言葉センスのタイプを診断する。正解・不正解はなく、言葉の好みや使い方の傾向から性格的な特徴を分類する。

**結果タイプ一覧（8タイプ）**:

| ID                 | タイプ名（四字熟語）                 | 概要                                                                         |
| ------------------ | ------------------------------------ | ---------------------------------------------------------------------------- |
| `elegant-precise`  | 一字千金（いちじせんきん）タイプ     | 的確で洗練された言葉を選ぶ。少ない言葉で多くを伝える、精密な言葉使いの達人   |
| `warm-empathy`     | 和顔愛語（わがんあいご）タイプ       | 相手の気持ちに寄り添う温かい言葉を選ぶ。共感力が高く、言葉で人を癒す         |
| `creative-playful` | 奇想天外（きそうてんがい）タイプ     | 予想外の表現や言い回しを好む。独創的で遊び心あふれる言葉のクリエイター       |
| `logical-clear`    | 理路整然（りろせいぜん）タイプ       | 論理的で明快な表現を好む。曖昧さを排し、クリアに伝えることを重視する         |
| `poetic-sensory`   | 花鳥風月（かちょうふうげつ）タイプ   | 情景や感覚を豊かに描写する言葉を好む。五感に訴える詩的な表現が得意           |
| `bold-impact`      | 疾風迅雷（しっぷうじんらい）タイプ   | インパクトのある力強い言葉を選ぶ。短くても心に刺さる表現で人を動かす         |
| `humor-wit`        | 抱腹絶倒（ほうふくぜっとう）タイプ   | ユーモアと機知に富んだ言葉を好む。場を和ませ、笑いを生み出す言葉の芸人       |
| `gentle-indirect`  | 柔和温順（にゅうわおんじゅん）タイプ | 柔らかく間接的な表現を好む。角を立てず、丁寧に気持ちを伝えることを大切にする |

タイプ名を四字熟語で表現することで、サイトの言語系アイデンティティを強化し、四字熟語性格診断や四字熟語力診断への回遊導線を自然に設けられる。

**質問の設計方針**:

- 全10問。各問は日常の具体的な場面を提示し、4つの選択肢から最も自分に合うものを選ぶ形式
- 場面例: メッセージの書き方、褒め方、謝り方、SNS投稿のスタイル、本の選び方、会話の切り出し方など
- 各選択肢は2つの結果タイプにポイントを付与（primary: 2点、secondary: 1点）
- ポイント配分: 8タイプ x primaryスロット5回 = 40（10問 x 4選択肢）。各タイプがprimaryとして5回ずつ均等に出現するよう設計する
- 質問文は親しみやすく、読むだけでも楽しいトーンにする
- 意外性のあるシチュエーション設問を積極的に取り入れる。ありきたりな「友達への返信」だけでなく、「AIに人生相談したら返事が四字熟語だけだった。あなたの反応は?」「上司が会議中に突然キラキラネームで呼んできた。どう切り返す?」のような、思わず笑ってしまうシチュエーションを設問の軸にする
- 選択肢の文面にもクスリとくる表現を織り込む。真面目に見せかけて少しズラした選択肢（例: 「まず深呼吸して、心の中で三回ツッコむ」）を混ぜることで、選ぶ行為自体を楽しめるようにする

**相性診断**:

- 8タイプ x (8+1)/2 = 36組の相性を定義する（推奨基準に準拠）
- 各相性は言葉のやりとりの観点から記述し、「このタイプ同士だと会話はこうなる」という具体的なイメージを提示する

**結果ページの設計**:

- 結果タイプの四字熟語名と概要説明（300-500文字）
- 「あなたの言葉センスの特徴」セクション
- 「こんな場面で力を発揮する」セクション
- 相性の良いタイプ・補い合えるタイプの紹介
- 関連コンテンツへの自然な導線（例:「このタイプの人が得意な四字熟語TOP3 → 四字熟語力診断へ」）
- 結果説明の中に「あるある」的なクスリと笑える一文を必ず含める。例: 一字千金タイプなら「LINEの返信が短すぎて既読スルーと間違われがち」、抱腹絶倒タイプなら「大事な場面でもボケずにはいられない宿命を背負っている」など、自分のタイプに思わず「わかる!」と共感しつつ笑える描写を入れることで、シェア欲を刺激する

### keywordsの設計

推薦ロジック（keywordsに基づくスコアリング）を最大限活用するため、以下の方針でキーワードを選定する。

```
keywords: [
  "言葉センス",
  "言葉選び",
  "性格診断",
  "四字熟語",
  "コミュニケーション",
  "言葉の使い方",
  "表現力",
  "語彙",
  "日本語"
]
```

**キーワード選定の意図**:

- 「言葉センス」「言葉選び」「言葉の使い方」: 本コンテンツ固有の検索キーワード。直接的な検索流入を狙う
- 「性格診断」: 診断系コンテンツ全般との相互推薦を可能にする。動物性格診断、キャラクター性格診断などとの回遊
- 「四字熟語」: 結果タイプが四字熟語で表現されるため、四字熟語性格診断・四字熟語力診断との強い関連性を持たせる
- 「コミュニケーション」: コミュニケーション関連の検索需要を取り込む
- 「表現力」「語彙」: 知識テスト系（漢字力・ことわざ力）との橋渡し
- 「日本語」: 日本文化適性診断との相互推薦
