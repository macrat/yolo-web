# サイクル-215

このサイクルでは、B-314（ツール・遊び詳細ページの新デザイン移行 + タイル化 / 移行計画 Phase 8）の **第 16 弾** として **regex-tester** の新デザイン移行 + タイル化を行う。

regex-tester は **複合入力型タイル 3 件目**（cycle-210 text-replace = N=1 / cycle-214 text-diff = N=2 / 本サイクル = N=3）であり、cycle-210 で確立し cycle-214 で初回引用検証された **複合入力型タイル AP-P21 SSoT 4 項目（(i) 二分類運用 / (ii) input 高さ ±15% / (v) (d)→(e) 変化率 ±15% / (vi) エラー文言枠）** を 3 回目の引用検証で固める。これにより B-452（複合入力型タイル AP-P21 (v) 基準値見直し / 着手条件 N≥3）が達成され、経験的暫定値 ±15% の妥当性に客観的決着をつけられる。

来訪者価値の観点では、M1b プログラマの普遍的需要「正規表現の動作確認・デバッグ」（フォーム validation / URL ルーティング / ログ抽出 / メール文字列パターンマッチ）に直結する。M1a 文書編集者の検索置換補助としても刺さる（text-replace との二重提供）。タイル動線で「パターンを書いてテストテキストに即時マッチ表示」最短 UX を提供することを目的とする。
