## 補足事項

### キャンセルの経緯

B-245の当初の前提は「静的結果ページの来訪者がシェアボタンを見つけにくい位置にある → fold上に移動すればシェア率が向上する」だった。しかし調査の結果、以下が判明した:

1. **静的結果ページの来訪者はクイズを受けた本人ではない**: クイズ完了後、本人は `/play/[slug]` 上の ResultCard（動的コンポーネント）で結果を見てシェアする。静的結果ページには遷移しない。
2. **静的結果ページの来訪者はシェアリンクや検索から来た第三者**: 彼らにとっての主要アクションは「自分もクイズを受けてみる」（CTA）であり、他人の結果をシェアすることではない。
3. **静的結果ページのトラフィックは実質ゼロ**: GA過去30日でPV 6件（開発テストと推定）、ランディングセッション0件。
4. **ページコンテンツが来訪者に合っていない**: 「あなたの特徴」「こんなところ、ありませんか？」等の文言はクイズ受験者向けであり、第三者には不適切。

シェアボタンの配置最適化は、(1)来訪者が実質ゼロ (2)来訪者にとってシェアが主要アクションではない (3)コンテンツ自体が来訪者に合っていない の3点から、現時点では効果を発揮しない。ページ全体の設計を根本からやり直す必要があり、シェアボタンの配置はその中で検討すべき。
