# サイクル-22

このサイクルでは、SNS拡散によるページビュー向上を目指し、以下の3つの関連タスクに一括で取り組みます。まずSNSベストプラクティスを調査し、その結果を踏まえてシェアボタンの全コンテンツ設置とOGP情報の最適化を実施します。

## 実施する作業

- [ ] B-067: SNSベストプラクティス調査・適用
  - X(Twitter)、Facebook、LINE等の主要SNSでの拡散最適化を調査
  - シェアボタンのUI/UXベストプラクティス（配置、デザイン、挙動）を調査
  - OGPタグ・Twitter Card等のメタ情報ベストプラクティスを調査
  - 調査結果をもとにB-065, B-066の実装方針を決定
- [ ] B-065: シェアボタンの全コンテンツ設置
  - ブログ、ツール、ゲーム、クイズ、診断等の全コンテンツタイプにシェアボタンを設置
  - B-067の調査結果に基づいたUI/UX設計
  - 対応SNS: X(Twitter)、LINE、リンクコピー等
- [ ] B-066: OGP画像・情報の全ページ最適化
  - ブログ、ゲーム結果、辞書、ツール等の個別ページOGP最適化
  - Twitter Card、og:image等のメタ情報を適切に設定
  - 各コンテンツタイプに合ったタイトル・説明文の自動生成
