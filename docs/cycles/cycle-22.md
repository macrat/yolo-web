# サイクル-22

このサイクルでは、SNS拡散によるページビュー向上を目指し、3つの関連タスクに一括で取り組みました。SNSベストプラクティスを調査し、その結果を踏まえて全コンテンツにシェアボタンを設置し、全ページのOGP画像を最適化しました。また、SNS最適化について日本のWebサイト管理者向けのブログ記事を執筆しました。

## 実施する作業

- [x] B-067: SNSベストプラクティス調査・適用
  - 日本の主要SNS（X、LINE、はてなブックマーク）の利用動向を調査
  - シェアボタンのUI/UXベストプラクティス（配置、デザイン、挙動）を調査
  - OGPタグ・Twitter Card等のメタ情報ベストプラクティスを調査
  - 各SNSのシェアIntent URL仕様を文書化
  - 調査結果をもとにB-065, B-066の実装方針を決定
- [x] B-065: シェアボタンの全コンテンツ設置
  - 共通ShareButtonsコンポーネント新規作成（src/components/common/ShareButtons.tsx）
  - 対応SNS: X(Twitter)、LINE、はてなブックマーク、コピー（コンテンツタイプ別に選択可能）
  - ブログ記事、ツール、チートシート、漢字辞典、四字熟語辞典、色辞典にシェアボタン設置
  - 既存ゲーム・クイズのX intent URLをtext+url分離形式に修正

- [x] B-066: OGP画像・情報の全ページ最適化
  - 共通OGP画像ヘルパー新規作成（src/lib/ogp-image.tsx）
  - 日本語フォント対応（Google FontsからNoto Sans JP動的fetch）
  - ブログ、ツール、チートシート、4ゲームにopengraph-image.tsx + twitter-image.tsx追加（計16ファイル）
  - ルートOGP画像の日本語化
  - テスト10件追加（ogp-image.test.tsx）
- [x] ブログ記事「日本のWebサイト管理者のためのSNS最適化ガイド」執筆
