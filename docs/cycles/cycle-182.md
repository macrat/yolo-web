# サイクル-182

このサイクルでは、デザイン移行計画 Phase 4 のうち **Phase 4.2（遊び一覧）** を実施する。`/play` と `/play/page/[page]` を `src/app/(new)/` 配下に移行し、新デザイン体系（DESIGN.md）に沿った再設計と、遊びコンテンツのリスト内絞り込み機能を追加する。

Phase 4 全体（4.1 ツール / 4.2 遊び / 4.3 ブログ / 4.4 トップ）は Owner 指示によりサブフェーズごとに独立サイクルとして実施する。本サイクルは Phase 4 の 2 番目のサイクルとなる（4.1 は cycle-181 で完了）。

来訪者から見える主な変化は、遊び一覧ページの新デザイン適用と絞り込み機能の追加。「軽くゲームで休憩したい」「ちょっとした診断系の遊びを探したい」といった来訪者が、目的の遊びにすばやく辿り着けるようになる。
