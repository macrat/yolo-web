# サイクル-217

このサイクルでは、デザイン移行計画 Phase 8.1（ツール群の新デザイン移行 + タイル化）の第 18 弾として、`traditional-color-palette`（伝統色カラーパレット）ツールを対象に、(1) ロジック CSS Module の新トークン置換、(2) 詳細ページ（`/tools/traditional-color-palette`）の `(new)/` 配下への移行とデザイン適用、(3) 道具箱で単独完結するタイルウィジェットの実装を行う。

移行順序原則「GA4 ページビュー高い順 + 構造単純な順」に基づき、残り 17 ツールの中から GA4 直近 30 日 PV 最高（10 PV / 2 位 cron-parser 1 PV の 10 倍）の `traditional-color-palette` を選定した。これまでの 17 件の多くが「テキスト等を入力 → 変換/生成 → コピー」型であったのに対し、本ツールは「伝統色 250 色から 1 色を選択 → 補色・類似色・トライアド・テトラド・分裂補色の 5 種配色パレットを一括生成 → 各色の HEX/RGB/HSL をコピー」という**色選択 → 複数配色一括生成型**の生成系アーキタイプであり、タイル UI の設計には専用の検討が必要。
