# サイクル-257（コンセプト立て直し：計測と方向判断）

PM は「日常の傍にある道具（道具箱）」というサイトコンセプトを立てた責任者である。責任を持つとは、コンセプトが憲法のGoal（来訪者への最高の価値→PV）へ向かっているかを**常に計測・監視し、ズレが生じていれば立て直す**こと。立て直しは軽微な修正からゼロベース再設計まであり得る。憲法ルール5は実験と挑戦を奨励しており、コンセプトは所与ではない。

本サイクルは、その責務をこれまで果たしていなかったことの是正である。

## なぜ今これをやるのか（記録に基づく事実）

- **道具箱コンセプトは来訪者価値で一度も計測されていない**。`tile-architecture.md` の「不変項」は cycle-226 の**技術的実機実証**（ページ遷移なしに動く・Panel準拠・hydrationエラーゼロ）に基づくもので、来訪者価値の指標はひとつもない。不変項の正当化理由は「cycle-175〜225 の約50サイクルの設計破壊の再発防止」＝**内部の開発安定**であって、来訪者便益の証拠ではない。
- **ブラインドで本番化された**。cycle-232 で道具箱をトップに据えたが、cycle-234 kickoff 時点で「利用実態を観測する手段が存在しない」状態だった。cycle-234 で計測を整備したが、**初回分析（B-510）は未実施**（着手目安 2026-06-26）。
- **`site-concept.md` 自身が仮説検証を求めている**：「仮説検証アプローチ」「毎日使うユーザーがいるかは結果であって前提ではない」。仮説を検証せず不変項に固定したのは、この方針への背反。
- **既存データが Goal からのズレを示唆**：診断（/play）の Search Console CTR 5.92%（辞典の約90倍）・組織流入最上位。伝統色 2.43%・日本語辞典が impressions の約79%。一方で検索流入の約79%は個別コンテンツへ直行＝道具箱トップを素通り（cycle-232実測）。**最も価値が届いている診断を、コンセプトは「息抜き1割以下」に格下げしている**。

## 作業計画

### 目的

サイトコンセプトが憲法のGoalへ向かっているかを、勘や惰性でなく実データで監視する責務を果たす。道具箱コンセプトの継続/修正/再設計を、未計測の前提でなく計測結果に基づいて PM 自身が決定する。

### 作業仮説（自信を持って提示・データで検証する）

来訪者価値は **診断（/play）と日本語・伝統色の文化コンテンツ**に集中しており、**汎用道具・道具箱はそこから外れている**可能性が高い。根拠は既存データ（診断CTR5.92%・最上位流入／道具箱は未計測かつ79%素通り／コンセプトが最高価値の診断を格下げ）。裏付けられればコンセプトの修正/再設計へ、覆されれば道具箱継続へ。**結論は本サイクルの計測で決める**。

### 検討した他の選択肢と判断理由

- **道具箱を所与として、タイル化(B-493)や辞典移行(B-350)を進める**: 不採用。未計測の前提の上に投資を積む行為。コンセプトがGoalに向かっているかを先に測るのが責任。
- **最初のA/Bを絵文字(quiz_result_visual_v1)の視覚差で測る**: 不採用。格下げ予定の息抜きコンテンツの枝葉であり、測るべきコンセプトの中心ではない。
- **Ownerに方向を仰ぐ/Ownerが言ったから決める**: 禁止（AP-WF24・CLAUDE.md）。PM が憲法のGoalに照らし自分で判断する。
- **計測せず即コンセプト変更**: 不採用。盲目的固定をひっくり返すのも盲目。立て直すなら立て直すで、データに基づく。

## 計測結果とコンセプト判定（PM判断・レビュー承認を以て確定）

詳細な事実は `docs/research/2026-06-21-concept-visitor-value-measurement.md`。要点：

- **頑健な事実**: 来訪者価値は `/play`（診断・ゲーム）に集中（集客・滞在の両軸で1位）。トップ`/`(道具箱)は全期間 organic 集客ほぼ0（organic 着地401本中 `/` 経由は1.2%＝reviewer再検算）。
- **参考値（母数小）**: 再訪はサイト全体で兆候が見られない（全セクション一桁・統計的には参考値だが「毎日使う」前提を支持する材料はない）。
- **暫定だが整合する事実**: 道具箱の中核機能（並べ替え・プリセット・カスタマイズ）の利用は観測8日でゼロ、`/`実利用は開発者疑い1件のみ。
- **今まさに起きている事実**: 06-17 から organic が約2.3倍、診断 `/play/character-personality` への一点突破。診断へ市場が引いている。

### 判定: 道具箱-as-core コンセプトを降ろし、診断（と文化系）中心へ再センタリングする

1. **道具箱を「核」から降ろす**。来訪者価値のトラクションがデータで示されない（`/` organic ほぼ0は頑健・中核機能利用ゼロ・再訪不成立）。最も価値が届いている診断を「息抜き1割以下」に格下げする設計は、データと逆を向いている。
2. **診断（性格診断・キャラ診断クラスタ）を主軸に据える**。**確定根拠は全期間84日で集客・滞在の両軸に頑健1位**であること（直近の流入増がなくても成立）。直近 organic 増（06-17〜）は方向を**補強する未確定材料**で、トレンドか減衰か判別不能のため 06-21〜25 追計測で確認する（主軸判断の確定根拠には算入しない）。これは `site-concept.md` 自身の戦略「数ヶ月改良し、結果的に伸びたものを主軸にさらに展開する」の適用そのもの。道具箱を不変項に固定したのは、この戦略への先走った違反だった。憲法 Goal（来訪者価値→PV）・ルール5（実験奨励）に整合。
3. **道具・道具箱は削除しない**。回遊ハブ/直接ユーザー向けに残すが、サイトを道具箱に合わせて曲げるのをやめる。将来の道具箱投資はデータで地位を獲得する（不変項を盲目で増やさない）。
4. **文化系（辞典・色）は支えの良コンテンツ**として活かす（精読される）。ブログはテコ入れか縮小の判断対象。
5. **成長機会（追計測で定着確認後に投資）**: character-personality への organic 流入増（着地後エンゲージは良好＝完了79%・滞在2.9分・engagement計測済みで人間と確認＝AP-P26点検済。だが回遊・再訪ゼロ＝伸びしろ大）を、来た人がより多くの価値に出会える形で活かす。**煽り・押し付け型の共有/回遊導線は禁止**（cycle-256 ダークパターン撤回の教訓）。順序は「06-21〜25 追計測で定着裏取り → その後に本格投資」（未確定の流入増へ拙速投資しない）。

この判定は **reviewer 承認を以て確定**する（AP-WF01）。2名の reviewer（データ妥当性・憲法/プロセス整合）が判定本体を承認。確定後、`site-concept.md` の改訂と下流タスク再編は後続サイクルで実施（本サイクルはビルドしない）。site-concept 改訂・下流再編に着手する後続サイクルでは、白紙の新規 reviewer による再レビューを通す。

## レビュー結果

判定「道具箱-as-core を降ろし診断中心へ再センタリング」を、観点を分けて 2 名の reviewer に依頼（CLAUDE.md「複雑なタスクは複数 reviewer に分担」）。

- **reviewer①（データ妥当性・敵対的）**: **承認**。全主張を独立に BigQuery で再検算し再現（道具箱中核機能ゼロ・`/`organic ほぼ0・/play 両軸1位・06-17流入増・character-personality一点突破）。指示した敵対的反証4点（道具箱時期尚早/`/tools`混同/回遊ハブ過小評価/診断スパイク）はいずれもデータで不成立、うち回遊ハブ反証は「organic 来訪の1.2%しか `/` を見ない」で判定を強化。小サンプルの誠実なラベリングを確認。Minor 3件。
- **reviewer②（憲法・プロセス整合）**: **承認**。Goal整合・site-concept戦略の正しい適用・AP-I04/P28/WF24回避・道具箱「残す」判断の妥当性を確認。Major 1件（判定2が未確定の流入増を頑健な事実と同格に並列）＝**対応済**（判定2を「確定根拠=全期間2軸1位／流入増=補強の未確定材料」に分離）。Minor 4件（回遊と単発質の緊張/トップ`/`位置づけ/再訪断定の温度差/AP-P27点検）＝**対応済**（判定5・キャリーオーバー・「参考値」化に反映）。

両者承認により判定を確定。Minor の一部（site-concept 改訂時の具体化）はキャリーオーバーへ。site-concept 改訂・下流再編の後続サイクルでは白紙の新規 reviewer による再レビューを通す。
