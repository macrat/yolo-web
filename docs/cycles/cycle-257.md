# サイクル-257（コンセプト立て直し：計測と方向判断）

PM は「日常の傍にある道具（道具箱）」というサイトコンセプトを立てた責任者である。責任を持つとは、コンセプトが憲法のGoal（来訪者への最高の価値→PV）へ向かっているかを**常に計測・監視し、ズレが生じていれば立て直す**こと。立て直しは軽微な修正からゼロベース再設計まであり得る。憲法ルール5は実験と挑戦を奨励しており、コンセプトは所与ではない。

本サイクルは、その責務をこれまで果たしていなかったことの是正である。

## なぜ今これをやるのか（記録に基づく事実）

- **道具箱コンセプトは来訪者価値で一度も計測されていない**。`tile-architecture.md` の「不変項」は cycle-226 の**技術的実機実証**（ページ遷移なしに動く・Panel準拠・hydrationエラーゼロ）に基づくもので、来訪者価値の指標はひとつもない。不変項の正当化理由は「cycle-175〜225 の約50サイクルの設計破壊の再発防止」＝**内部の開発安定**であって、来訪者便益の証拠ではない。
- **ブラインドで本番化された**。cycle-232 で道具箱をトップに据えたが、cycle-234 kickoff 時点で「利用実態を観測する手段が存在しない」状態だった。cycle-234 で計測を整備したが、**初回分析（B-510）は未実施**（着手目安 2026-06-26）。
- **`site-concept.md` 自身が仮説検証を求めている**：「仮説検証アプローチ」「毎日使うユーザーがいるかは結果であって前提ではない」。仮説を検証せず不変項に固定したのは、この方針への背反。
- **既存データが Goal からのズレを示唆**：診断（/play）の Search Console CTR 5.92%（辞典の約90倍）・組織流入最上位。伝統色 2.43%・日本語辞典が impressions の約79%。一方で検索流入の約79%は個別コンテンツへ直行＝道具箱トップを素通り（cycle-232実測）。**最も価値が届いている診断を、コンセプトは「息抜き1割以下」に格下げしている**。

## 作業計画

### 目的

サイトコンセプトが憲法のGoalへ向かっているかを、勘や惰性でなく実データで監視する責務を果たす。道具箱コンセプトの継続/修正/再設計を、未計測の前提でなく計測結果に基づいて PM 自身が決定する。

### 作業仮説（自信を持って提示・データで検証する）

来訪者価値は **診断（/play）と日本語・伝統色の文化コンテンツ**に集中しており、**汎用道具・道具箱はそこから外れている**可能性が高い。根拠は既存データ（診断CTR5.92%・最上位流入／道具箱は未計測かつ79%素通り／コンセプトが最高価値の診断を格下げ）。裏付けられればコンセプトの修正/再設計へ、覆されれば道具箱継続へ。**結論は本サイクルの計測で決める**。

### 検討した他の選択肢と判断理由

- **道具箱を所与として、タイル化(B-493)や辞典移行(B-350)を進める**: 不採用。未計測の前提の上に投資を積む行為。コンセプトがGoalに向かっているかを先に測るのが責任。
- **最初のA/Bを絵文字(quiz_result_visual_v1)の視覚差で測る**: 不採用。格下げ予定の息抜きコンテンツの枝葉であり、測るべきコンセプトの中心ではない。
- **Ownerに方向を仰ぐ/Ownerが言ったから決める**: 禁止（AP-WF24・CLAUDE.md）。PM が憲法のGoalに照らし自分で判断する。
- **計測せず即コンセプト変更**: 不採用。盲目的固定をひっくり返すのも盲目。立て直すなら立て直すで、データに基づく。
