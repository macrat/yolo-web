# サイクル-216

このサイクルでは、B-314（ツール・遊び詳細ページの新デザイン移行 + タイル化 / 移行計画 Phase 8.1）の第 17 弾として、**keigo-reference（敬語早見表）** を新デザインへ移行し、タイル化する。

keigo-reference は移行未完了の 18 ツールの中で GA4 直近 30 日 PV が最高（22 PV / 2 位の traditional-color-palette 10 PV の倍以上）であり、移行順序の原則「GA4 PV 高い順 + 構造単純な順」に合致する。これまで移行した 16 件はすべて変換・生成系（入力 → 出力）のツールだったが、keigo-reference は **参照・検索型** という新しいアーキタイプである。「この言い方の敬語は何だっけ」とつまずいた来訪者が、調べてさっと使える「傍にある道具」になることを目指す。参照・検索型タイルの設計指針をここで確立できれば、残る辞典系ツール群（traditional-color-palette ほか）の移行にも効く。
