# サイクル-246

このサイクルでは、実測（GA4 / Search Console）に基づき、サイトで最大の検索露出を稼ぎながらクリックに変換できていない**四字熟語辞書ページ（/dictionary/yoji）の CTR を改善**します。具体的には、全 400 ページが共通利用するメタデータ生成（`generateYojiPageMetadata`）のタイトル・説明文・構造化データを検索意図に合わせて最適化し、検索 1 ページ目に出ているのにクリックされない取りこぼしを是正します（B-515）。

## 実施する作業

- [ ] 競合（コトバンク等の四字熟語辞典）の検索結果スニペット・タイトルを調査し、なぜ無名ドメインの当サイトが選ばれないかの仮説を立てる（take-screenshot / Web 調査）
- [ ] 現状の `generateYojiPageMetadata` の title / description / keywords を、実測で取りこぼしている検索意図（「○○ 意味」「○○ 読み方」「○○ とは」「○○（単体）」）を網羅し、クリック動機（読み方を前置・由来・例文の存在示唆）を持たせる形に改善
- [ ] `generateYojiJsonLd` の構造化データ強化を検討（DefinedTerm に追加可能なプロパティ・データで埋められる範囲で）

- [ ] reviewer エージェントによるレビューと指摘対応

## 作業計画

### 目的

四字熟語辞書ページはサイト全体の検索露出の **約 40.8%（4,798 impression / 30日）** を稼ぐ最大の露出源でありながら、CTR が **0.42%** と壊滅的に低い。露出の **74%（3,547 impr）が検索 1 ページ目（4〜10 位）** に入っているにもかかわらずクリックされていない。クエリ「至誠通天 読み方」は平均 **4.2 位なのに CTR 0%**。これは順位や被リンクの問題ではなく、**タイトル・スニペットが検索意図に響いていない CTR の問題**である。

メタデータは `src/lib/seo.ts` の `generateYojiPageMetadata`（全 400 ページ共通）に集約されているため、1 箇所の改善で全ページに波及する。**最小の変更で最大の波及**が見込める、来訪者価値・PV 最大化に最も効く一手。

来訪者の困りごとは具体的だ。四字熟語の意味・読み方を調べようと検索し、検索結果に当サイトが 1 ページ目で出ているのに、スニペットが「ここに来れば知りたいことが分かる」と伝えられていないため、別の辞書サイトを選んでいる。せっかくたどり着ける場所にいるのに選ばれていない、この取りこぼしを是正する。

### 作業内容

1. **競合スニペット調査**: 「至誠通天 意味」「冷汗三斗 意味」等の実取りこぼしクエリで実際に検索し、上位の四字熟語辞典がどんなタイトル・説明文で表示されているかを把握。当サイトとの差分から、クリックされる要因の仮説を立てる。
2. **title の改善**: 現状 `「至誠通天」の意味・読み方 - 四字熟語辞典 | yolos.net`。検索意図（意味/読み方/とは/単体）を網羅しつつ、読みやすく差別化できる形へ。読み方クエリ（平均 4.2 位 CTR0%）への訴求を強化。
3. **description の改善**: 現状 `四字熟語「至誠通天」（しせいつうてん）の意味: 誠を尽くせば天に通じること`。読み方を前置し、意味＋由来（中国/日本）＋例文の存在を情報密度高く示し、クリック動機を持たせる。`src/data/yoji-data.json` が持つ reading / meaning / origin / structure / example を活用。
4. **構造化データの強化検討**: `generateYojiJsonLd` に、保有データで埋められる範囲のプロパティを追加し、リッチな表示の可能性を高める。
5. **検証**: データ駆動でエッジケース（reading 欠落、長い meaning による文字数超過等）に破綻がないか確認。実レンダリングの title/meta を確認し、競合と並べて評価。

### 検討した他の選択肢と判断理由

- **当初の第一候補 B-088（敬語早見表の収録拡充）**: keigo-reference はツールとして 30 日 13 PV（ツール 2 位）と健闘するが、SC 実測で敬語関連クエリの impression が実質 0。課題は「収録の量」ではなく「検索露出の不在」と判明。収録を増やしても流入に直結しないため見送り。backlog の B-088 Notes に実測を注記。
- **B-435 / B-437（QR ツール強化）**: /tools/qr-code は 30 日 PV 0・全期間累計 1 PV・QR 検索 impression 0。流入皆無で「作っても誰も来ない」投資。P3→P4 へ降格。
- **B-388（Pagination 44px 化）**: 2 ページ目以降への到達が 30 日 0 PV。ほぼ未使用のため P3→P4 へ降格（a11y 是正の価値自体は残す）。
- **B-323 そのものの全面着手**: スコープが大きく 1 サイクルに収まらない。実測で前提（ツール系へ刷新）が逆転したことを Notes に反映し、最も効果が確実で最小の「四字熟語 CTR 改善」を B-515 として切り出した。
- **コンテンツ拡充（meaning 詳説・由来・類義語の追記）による順位押し上げ**: 400 件の本格拡充は大規模で、かつ順位は既に十分（1 ページ目 74%）。まず CTR 改善が費用対効果で勝る。コンテンツ拡充・診断への内部導線強化は二次的施策として backlog に残す。

## 補足事項

- メタ改善の SEO 効果は遅効性（再クロール待ち）であり、来週の SC データでは確認できない。効果検証は数週間後（次の四字熟語 SC 実測サイクルで確認）。本サイクルでは「変更の質（検索意図への適合・競合との差別化）」を確認することで妥当性を担保する。
- 診断系（/play/yoji-level）は唯一クリックが立っている成功パターン（CTR 2.0〜3.5%）。辞書ページの高 impression を診断クリックへ転換する内部導線強化は、効果が見込めるが本サイクルのスコープ外とし、二次施策として backlog（B-323 残スコープ）に残す。
- 憲法 3（AI 運営の明示）に関わる表示は本変更の対象外（メタデータのみの改善）。
