# サイクル-183

このサイクルでは、デザイン移行計画 Phase 4 のうち **Phase 4.3（ブログ一覧）** を実施する。`/blog` 一覧および派生ルート（ページネーション・カテゴリ・タグ）を `src/app/(new)/` 配下に移行し、新デザイン体系（DESIGN.md）に沿って再設計したうえで、ブログ一覧内の絞り込み機能（タイトル全文 + タグ + カテゴリの組み合わせ）を追加する。

Phase 4 全体（4.1 ツール / 4.2 遊び / 4.3 ブログ / 4.4 トップ）は Owner 指示によりサブフェーズごとに独立サイクルとして実施する。本サイクルは Phase 4 の 3 番目のサイクルとなる（4.1 は cycle-181、4.2 は cycle-182 で完了）。

来訪者から見える主な変化は、ブログ一覧ページの新デザイン適用と絞り込み機能の追加。「特定のテーマで記事を探したい」M1a と、「興味のあるカテゴリの記事を俯瞰したい」M1b の双方が、100 件超のブログ記事から目当ての 1 本に少ない手順で辿り着けるようになる。

なお、Phase 4.3 はサブルート数が多い（一覧本体・ページネーション・カテゴリ・タグの 4 系統）ため、本サイクル内で全部を扱うか分割するかは作業計画フェーズで判断する（規模次第では一部を後続サイクルへ送る選択肢を持つ）。
