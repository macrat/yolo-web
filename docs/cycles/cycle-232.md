### 目的

**派生元**: B-336（backlog・P1）＝ `docs/design-migration-plan.md` Phase 10.3「道具箱ページの本公開」。その原典は `docs/site-concept.md` L7「気に入ったツールを1つの画面にまとめられるダッシュボード的な機能を通じて、『日常の傍にある』存在を目指す」。

cycle-226〜231 で道具箱は「39枚の生きたタイル・タイル選択・localStorage 永続化・ペルソナ別プリセット5本」まで完成したが、`/toolbox` は noindex のままで、ターゲット「気に入った道具を繰り返し使っている人」（docs/targets/気に入った道具を繰り返し使っている人.yaml）が実際にこの価値に触れる経路が存在しない。プリセットが「迷わない出発点」として「特定の作業に使えるツールをさっと探している人」に価値を発現するのも本公開時点（cycle-231 計画 r1 SF-1 で正確化済み）。**作った価値が来訪者に届いていない状態を解消することが本サイクルの目的**であり、constitution の Goal（来訪者への価値提供による PV 向上）への最短経路である。
