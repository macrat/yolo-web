# ツールアイデア集

これはツール提案の集約ドキュメントです。実装確約ではなく、着手時に visitor need を再評価して採否を判断します。

各エントリは「ツール名 / 来訪者の用途 / 含む機能の要点 / 技術的実現性 / 対応 backlog 番号」の形式で記載します。

---

## 時間関係ツール群（B-317）

**来訪者の用途**: 日常生活で時間を管理・計測したい。  
**含む機能の要点**: アナログ時計・タイマー・ストップウォッチ・ポモドーロタイマーなど。具体的なツール選定は PM が着手時に判断する。  
**対応 backlog 番号**: B-317

## 画像を共有用に整えるワークフローツール（B-318）

**来訪者の用途**: 写真や画像を SNS/LINE で共有する前に、安全で見栄え良く整えたい。  
**含む機能の要点**: 色味調整 / 切り抜き・回転 / EXIF 削除 / 顔ぼかし / リサイズ / フォーマット変換を 1 ページで完結するワークフロー。  
**技術的実現性**: docs/research/2026-05-05-browser-image-editor-feasibility.md および docs/research/2026-05-05-browser-ai-tool-ideas.md 参照。  
**対応 backlog 番号**: B-318

---

## 動画/音声を共有用に整えるワークフローツール（B-319）

**来訪者の用途**: 動画や音声ファイルを共有しやすい形に整えたい。  
**含む機能の要点**: 音声抽出 / トリミング / 解像度変更・圧縮 / フォーマット変換を 1 ページで完結するワークフロー。技術構成は WebCodecs + Mediabunny（gzip 17〜70KB）。  
**改題経緯**: 旧提案ではホワイトノイズ・サイン波・ボイスレコーダー等の散発的な音声ツールを例示していたが、サイトコンセプト変更後の来訪者ニーズ（既存ファイルを共有可能な形に整えて持ち帰る）に合わせて「動画/音声を共有用に整えるワークフローツール」に再定義した。  
**技術的実現性**: docs/research/2026-05-05-webcodecs-browser-media-converter-feasibility.md 参照。  
**対応 backlog 番号**: B-319

## 動画→GIF 変換ツール（B-368）

## プッシュ・トゥ・トーク（PTT）メモツール（B-369）

**来訪者の用途**: 話した内容をその場でテキストにして手軽に蓄積したい（会議・アイデアメモ等）。  
**含む機能の要点**: ボタンを押している間だけ録音し、離した瞬間に文字起こしして蓄積するリアルタイムメモ。  
**技術的実現性**: transformers.js v4 + moonshine-tiny-ja-ONNX（28MB int8）。docs/research/2026-05-05-browser-stt-transcription-tool-feasibility.md 参照。  
**対応 backlog 番号**: B-369

## 録音ファイル文字起こしツール（会議録・長尺対応）（B-370）

**来訪者の用途**: 録音済みの音声・動画ファイルをテキストに文字起こしして記録を残したい。  
**含む機能の要点**: 音声/動画ファイルのアップロードから文字起こしテキストの生成・書き出し。着手条件: B-369 完了後を推奨（STT 統合基盤の再利用）。  
**技術的実現性**: transformers.js + whisper-base（77MB int8）+ vad-web。docs/research/2026-05-05-browser-stt-transcription-tool-feasibility.md 参照。  
**対応 backlog 番号**: B-370

## 画像合成ワークフローツール（背景除去 + 色合わせ + 重ね合わせ）（B-371）

**来訪者の用途**: 背景を除去して別の背景と合成した画像を手軽に作りたい。  
**含む機能の要点**: 背景除去 → 前景・背景独立に色味調整 → 位置・サイズ・回転調整 → ブレンドモード → 書き出しの 1 ページワークフロー。  
**技術的実現性**: MODNet（MIT）+ Canvas API。docs/research/2026-05-05-browser-image-editor-feasibility.md 参照。  
**対応 backlog 番号**: B-371

## 写真カラーパレット抽出ツール（B-372）
