## 2. 長期目標: 月間6万PVへの逆算

### 2-1. 6万PV/月に必要な条件の逆算

月間6万PVを達成するために必要な条件を、トラフィックソース別に逆算する。

**前提**: 6万PV = 2,000PV/日。チャネル構成は成熟した診断サイトの一般的な比率を参考にする。

| 条件                              | 必要水準                       | 現状                         | ギャップ                               |
| --------------------------------- | ------------------------------ | ---------------------------- | -------------------------------------- |
| **条件1: SEO流入（日本語）**      | 1,500PV/日（45,000PV/月）      | 約10PV/日（294PV/月）        | 150倍                                  |
| **条件2: ドメインオーソリティ**   | DR20以上（推定）               | 新規ドメイン（DR0相当）      | 被リンク獲得が必要                     |
| **条件3: コンテンツ量と質**       | 検索上位表示される診断50種以上 | 20種（いずれも上位表示なし） | 30種以上の追加+SEO最適化               |
| **条件4: バイラル/SNS流入**       | 500PV/日（15,000PV/月）        | ほぼゼロ                     | フォロワー基盤またはバイラル発生が必要 |
| **条件5: 英語圏流入（代替経路）** | 条件4の代替として検討可        | 未着手                       | 実験で検証が必要                       |

### 2-2. 各条件の獲得可能性分析

**条件1（SEO流入45,000PV/月）**: 獲得には時間がかかるが、段階的に可能。

- 必要なこと: ドメイン年齢6ヶ月以上 + 検索需要のあるキーワードで上位表示 + 十分なコンテンツ量
- 最大のボトルネック: ドメインオーソリティ（条件2に依存）
- 参考: uranaino.net（診断特化サイト）が低ドメインオーソリティ期から検索流入を獲得した事例あり（出典: cycle-162 Step 2調査）

**条件2（ドメインオーソリティ）**: AI自律での能動的獲得は困難だが、長期的には自然獲得が可能。

- 被リンクの能動的獲得はAI単独では困難（メディアへの売り込み等ができない）
- 自然な被リンク獲得（良質コンテンツが引用される）は時間がかかるが可能
- はてブ経由の話題化で加速できる可能性あり（ただし自然な話題化が必要条件）

**条件3（コンテンツ量と質）**: AI自律で獲得可能。

- 診断コンテンツの制作はAIの得意分野
- 検索需要のあるキーワードを狙った制作が可能

**条件4（バイラル/SNS流入）**: 現時点では獲得不可能。

- フォロワーゼロの独自サイトがバイラルで大規模PVを達成した事例は、2回の独立調査で確認できなかった（出典: cycle-162 Step 2調査）
- バイラル係数K>1にはフォロワー200-500人またはインフルエンサー1名の介入が必要（出典: cycle-162 Step 2調査）
- ただし、英語圏展開（条件5）で代替可能な可能性がある

**条件5（英語圏流入）**: 未検証だが、高ポテンシャル。

- 英語圏診断市場は月間数千万規模（16Personalities: 月間1,633万訪問）
- 「日本の伝統色 x 性格診断」は英語圏で直接競合ゼロ（出典: cycle-162 WebSearch調査）
- KTestOne（韓国発パーソナルカラー診断）がTikTokで4.9億回再生を達成（出典: cycle-162 Step 2調査）
- 英語圏はフォロワーゼロでもSNSバイラルの可能性がある（KTestOneの事例: プラットフォーム拡散型で個人フォロワー不要）

### 2-3. 6万PV到達の段階的計画

**結論: 6万PVは現時点で確約できないが、条件を段階的に獲得していくことで到達可能性を高められる。最終目標として維持する。**

到達に向けた段階:

1. **Phase 0-1（0-6ヶ月）**: 条件3（コンテンツ量）を充足 + 条件5（英語圏）を実験で検証
2. **Phase 1-2（6-12ヶ月）**: 条件2（ドメインオーソリティ）の自然向上を待ちつつ、条件1（SEO流入）を積み上げ
3. **Phase 2-3（12ヶ月以降）**: 英語圏の検証結果に基づき、条件4（バイラル/SNS）または条件5（英語圏拡大）で突破を図る

5,000PV/月は「12ヶ月後の中間マイルストーン」として設定する（最終目標ではない）。

## 3. 施策別PV積み上げ計算（12ヶ月後 5,000PV/月の根拠）

### 3-1. 第1層: 診断SEO（日本語）

**計算方法**: キーワード検索ボリューム x 想定CTR x 想定順位補正

| キーワード                                        | 月間検索数（実測）   | 想定順位 | 想定CTR  | 月間PV推計 | 備考                                                                                                                                                                                                                                                                               |
| ------------------------------------------------- | -------------------- | -------- | -------- | ---------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| おもしろ診断                                      | 2,400                | 20位以下 | 0.5%以下 | 12以下     | 競合強度「非常に高い（5つ星中5）」、yolos.netの勝算「極めて低い」（出典: docs/research/2026-04-06-serp-competition-and-search-demand-analysis.md §1.3）。ShindanMaker・BuzzFeed JP等が上位を占め参入余地なし                                                                       |
| 動物 性格診断                                     | 4,400                | 5-10位   | 2-5%     | 88-220     | SERP調査で競合強度「中程度」、yolos.netの勝算「中程度〜高い」と評価。hitostat.com・ロルモ等の個人・小規模サイトが上位に入っており参入余地あり（出典: docs/research/2026-04-06-serp-competition-and-search-demand-analysis.md §1.5）。yolos.netは既存の動物性格診断コンテンツを保有 |
| 四字熟語 診断                                     | 390                  | 1-3位    | 10-30%   | 39-117     | SEO難易度22（低い）、上位獲得見込み高                                                                                                                                                                                                                                              |
| MBTI 診断（ロングテール）                         | 2,240,000のうち0.01% | 圏外     | -        | 約224      | 超競合。ロングテールの派生クエリからの漏れ流入のみ                                                                                                                                                                                                                                 |
| 占い 無料（ロングテール）                         | 201,000のうち0.05%   | 圏外     | -        | 約100      | 超競合。ロングテール漏れ流入のみ                                                                                                                                                                                                                                                   |
| その他ロングテール（20-30種の診断からの流入合計） | -                    | -        | -        | 200-400    | 1コンテンツあたり10-15PV/月と推定（現状の1.25PVから成長）                                                                                                                                                                                                                          |

**第1層 合計推計: 700-1,200PV/月**

前提条件（未検証を含む）:

- ドメイン年齢が6ヶ月を超え、Googleの新規ドメイン評価期間を脱する（未検証）
- SEO最適化（タイプ別解説ページ等）により現状の1.25PV/コンテンツが10-15PV/コンテンツに向上する（未検証）
- 「四字熟語 診断」等のSEO難易度が低いキーワードで上位表示される（未検証だが蓋然性あり）

### 3-2. 第2層: 英語圏コンテンツ（主軸候補）

**計算方法**: 英語圏市場規模から逆算

**保守的シナリオ（SEOのみ）**:

- ニッチキーワード "Japanese color personality quiz" 等で月間1,000-5,000検索（推定、有償ツール未使用のため未検証）
- 上位表示で CTR 5-15%
- 推計: 50-750PV/月

**楽観的シナリオ（SNSバイラル発生時）**:

- KTestOneの事例を参考: TikTok/Instagram等でシェアが拡散
- 月間5,000-50,000PV（KTestOneの規模からの類推、ただし同等の拡散は保証されない）

**第2層 合計推計: 50-750PV/月（保守的）、最大50,000PV/月（バイラル時、未検証）**

重要: バイラルシナリオは予測不可能であり、計画上は保守的シナリオで計算する。ただし英語圏は条件4（バイラル/SNS）を代替できる唯一の経路であり、実験の優先度は高い。

**英語圏を主軸候補とする根拠**:

- 16Personalities: 月間1,633万訪問（英語圏市場の巨大さ）
- KTestOne: TikTok 4.9億回再生（東アジア発カラー診断の英語圏バイラル実績）
- 「日本の伝統色 x 性格診断」は英語圏で直接競合ゼロ（出典: cycle-162調査）
- 日本文化への英語圏需要が急拡大中（"Japanese vintage art" 検索数270%増等）
- 日本発サイトとしてのオーセンティシティ（本物らしさ）が差別化要因

### 3-3. 第3層: はてブ経由の初期トラクション

**計算方法**: はてブ掲載時の一般的な流入量から推定

- はてブ新着掲載: 100-500PV/回（推定）
- ホットエントリ掲載: 1,000-10,000PV/回（推定）
- 継続的な効果: 被リンクによるドメインオーソリティ向上（第1層・第2層を間接的に強化）

**第3層 合計推計: 直接PVは単発的。主たる効果はドメインオーソリティ向上による第1層への間接貢献。**

制約: AI自律では「自分でブックマーク」は不可能。自然な話題化が必要条件。

### 3-4. 積み上げ合計

| 施策                       | 保守的推計   | 楽観的推計     | 推計の信頼度                               |
| -------------------------- | ------------ | -------------- | ------------------------------------------ |
| 第1層: 診断SEO（日本語）   | 700PV/月     | 1,200PV/月     | 中（検索ボリュームは実測、CTR/順位は推定） |
| 第2層: 英語圏コンテンツ    | 50PV/月      | 750PV/月       | 低（英語キーワードボリューム未実測）       |
| 第3層: はてブ効果          | 0PV/月       | 500PV/月       | 低（話題化の可否が予測不能）               |
| Direct（既存ユーザー回帰） | 200PV/月     | 400PV/月       | 中（現状240PVからの推定）                  |
| **合計**                   | **950PV/月** | **2,850PV/月** | -                                          |

**5,000PV/月到達の条件**: 保守的推計（950PV）では5,000PVに届かない。5,000PV到達には以下のいずれかが必要:

1. 英語圏でバイラルが発生（第2層が5,000PV以上に跳ねる）
2. ドメインオーソリティが大幅に向上し、中競合キーワードでも上位表示される
3. 上記の複合効果

したがって、5,000PV/月は「条件が揃えば到達可能な目標」であり、確実に到達できる数字ではない。ただし6万PVへの逆算上、この水準を12ヶ月後に通過する必要がある。

## 4. 成長エンジン: 三層戦略

### 第1層（主エンジン）: 診断・占いSEO特化

**方針**: 診断・占いジャンルに完全特化してドメインの専門性を高め、ロングテールキーワードからの検索流入を積み上げる。

**根拠（検証済み）**:

| キーワード    | 月間検索数 | 出典                                        |
| ------------- | ---------- | ------------------------------------------- |
| MBTI 診断     | 2,240,000  | cycle-162 aramakijake.jp実測                |
| 動物占い      | 450,000    | cycle-162 aramakijake.jp実測                |
| 占い 無料     | 201,000    | cycle-162 aramakijake.jp実測                |
| 動物 性格診断 | 4,400      | cycle-162 aramakijake.jp実測                |
| おもしろ診断  | 2,400      | cycle-162 aramakijake.jp実測                |
| 四字熟語 診断 | 390        | cycle-162 aramakijake.jp実測（SEO難易度22） |

**参考事例**: uranaino.net（診断・占い特化サイト）が低ドメインオーソリティ期から検索流入を獲得した成功事例として確認（出典: cycle-162 Step 2調査）。

**具体的施策**:

- 検索需要のあるロングテールキーワードを狙った新規診断コンテンツの制作
- 既存20種の診断の結果ページのSEO最適化（タイプ別解説ページの追加等）
- 辞典・ツール等の新規追加は停止し、診断に集中（cycle-161の方針を維持）

### 第2層（主軸候補）: 英語圏展開

**方針**: 「日本の伝統色 x 性格診断」の英語版を制作し、英語圏からの流入を検証する。成功した場合は主エンジンへ昇格させる。

**根拠（検証済み）**:

- 英語圏で「日本の伝統色 x 性格診断」に直接競合なし（出典: cycle-162 WebSearch調査）
- KTestOne（韓国発パーソナルカラー診断）がTikTokで4.9億回再生を達成（出典: cycle-162 Step 2調査）
- 英語圏の診断コンテンツ市場はUSD 25億規模で成長中（CAGR 9.8%、出典: Verified Market Reports）
- 日本文化への英語圏需要が急拡大中（"Japanese vintage art" 検索数270%増、出典: cycle-162 Step 2調査）
- Ownerがcycle-158で英語圏向けローカライズの小実験を提案（出典: cycle-158サイクルドキュメント）

**6万PV到達における位置づけ**: 日本語SEOだけでは6万PVに必要な条件4（バイラル/SNS流入15,000PV/月）を満たせない。英語圏展開は、フォロワーゼロの制約を迂回して大規模流入を獲得できる唯一の現実的経路。KTestOneの事例は、東アジア発カラー診断が英語圏でプラットフォーム拡散型（個人フォロワー不要）のバイラルを起こせることを証明している。

**未検証の前提**: yolos.netの英語版が実際に英語圏からの流入を獲得できるかは未検証。まず1コンテンツで実験し、結果を見て判断する。

### 第3層（補助施策）: はてブ経由の初期トラクション

**方針**: 「AI自律運営サイトの実験記録」コンテンツではてなブックマークでの話題化を狙い、被リンク獲得からドメインオーソリティ向上につなげる。

**制約**: AI自律では「自分でブックマーク」は不可能。自然な話題化が必要条件。

**根拠**: 被リンクがドメインオーソリティに寄与することは一般論として確立（ただし向上速度は未検証）。

## 5. フェーズ別マイルストーン

### Phase 0（現在〜3ヶ月後）: 基盤構築期

**目標**: 月間800PV

**800PV内訳（施策別推計）**:

| 施策                                            | 推計PV        | 前提                                                               |
| ----------------------------------------------- | ------------- | ------------------------------------------------------------------ |
| 診断SEO（「動物 性格診断」「四字熟語 診断」等） | 200-350PV     | ドメイン年齢向上と既存コンテンツのSEO最適化による改善              |
| その他ロングテール（20-30種の既存診断）         | 100-200PV     | 現状1.25PV/コンテンツから5PV/コンテンツへの部分的改善              |
| 英語版コンテンツ実験                            | 50-100PV      | 1コンテンツ公開後の初期流入                                        |
| Direct（既存ユーザー回帰）                      | 200-250PV     | 現状240PVを概ね維持                                                |
| **合計**                                        | **550-900PV** | 保守的中央値で約720PV。800PVは楽観的シナリオ寄りだが実現可能な範囲 |

注: 「おもしろ診断」はSERP競合が「非常に高い（5つ星中5）」のため積み上げ計算から除外（出典: docs/research/2026-04-06-serp-competition-and-search-demand-analysis.md §1.3）。

**主要施策**:

1. 既存診断コンテンツのSEO最適化（結果ページ・タイプ別解説の追加）
2. 英語版1コンテンツ実験の制作・公開（「日本の伝統色 x 性格診断」英語版）
3. はてブ話題化を狙うブログ記事の制作

**判断ポイント（3ヶ月後）**:

- 月間600PV以上（現状維持以上）→ Phase 1へ継続
- 月間600PV未満 → 戦略を根本から再検討
- 英語版: 月間100PV以上 → Phase 1で主軸として拡大
- 英語版: 月間100PV未満 → 英語版の改善策を検討（即撤退はしない。市場の大きさを考慮）

### Phase 1（3〜6ヶ月後）: 検証期

**目標**: 月間1,500PV

**主要施策**:

1. Phase 0の結果に基づく拡大/改善判断
2. 検索需要が確認できた新規診断コンテンツの制作
3. 英語版が好調であれば追加コンテンツの制作
4. ドメインオーソリティの経過観察

**判断ポイント（6ヶ月後）**:

- 月間1,000PV以上 → Phase 2へ継続
- 月間1,000PV未満 → 戦略を見直し

### Phase 2（6〜12ヶ月後）: 成長期

**目標**: 月間5,000PV（中間マイルストーン）

**主要施策**:

1. ドメインパワー向上効果（6-12ヶ月での自然な向上を期待）の活用
2. 英語版が成功していた場合はコンテンツを本格拡大
3. SEO流入の継続的な積み上げ

**判断ポイント（12ヶ月後）**:

- 月間3,000PV以上 → Phase 3（突破期）へ
- 月間3,000PV未満 → Phase 3に必要な条件の欠如を分析し戦略を再構築

### Phase 3（12ヶ月以降）: 突破期

**最終目標**: 月間6万PV

**到達に必要な条件（段階的に獲得する）**:

- 条件1: SEO流入が安定的に成長している（Phase 0-2で獲得）
- 条件2: ドメインオーソリティが向上している（Phase 1-2で自然獲得を期待）
- 条件3: 十分なコンテンツ量がある（Phase 0-2で獲得）
- 条件4/5: バイラル発生 または 英語圏での大規模拡大（Phase 2の結果次第）

**重要な留意点**: Phase 3以降はコントロールできない要因（バイラル発生、英語圏でのバイラル）に部分的に依存する。Phase 2終了時点での状況を見て、6万PVの実現可能性を改めて評価する。

## 6. 検証済みと未検証の前提一覧

## 9. cycle-161中期計画との関係

**cycle-161の中期計画（OS PV 294→350PV/月）は破棄する。**

**理由（具体的データに基づく）**:

1. OS PVがGA4実測で明確な下降トレンド: W9:141 → W10:55 → W11:54 → W12:81 → W13:61 → W14:42。6週間で141→42と70%減少しており、月間350PV/月への増加目標は現実と乖離している
2. OS PVだけに固執すると英語圏等の他チャネルを見落とす。6万PV到達には日本語SEO以外のチャネル（特に英語圏）が不可欠
3. 350PV/月は保守的すぎて6万PVへの道筋が見えない。逆算で必要な45,000PV/月のSEO流入と比較して約130倍の開きがある

**ただし以下はcycle-161の決定を維持する**:

- 診断・占いをサイトの中心に置く方針
- 辞典・ツール・チートシートの新規追加は停止する方針
