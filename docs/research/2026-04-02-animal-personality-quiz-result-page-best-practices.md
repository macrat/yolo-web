# 動物性格診断 結果ページ ベストプラクティス調査

## 1. 主要性格診断サイトの結果ページ構成

### 16Personalities（16personalities.com）

世界で10億回以上受験されている最大規模の性格診断サイト。日本語対応済み。

**結果ページの構成（セクション順）:**

1. **タイプ名とキャッチコピー** - 例: "弁護士型 (INFJ)" など、4文字のコード＋日本語タイプ名
2. **ビジュアルヒーロー** - タイプを象徴したイラスト（色・雰囲気でタイプを視覚化）
3. **イントロダクション** - 約200〜300字のタイプ概要文。肯定的・共感的な書き出し
4. **軸スコア表示** - E/I、N/S、T/F、J/P の各軸にパーセンテージを表示（例: "外向型 62%"）
5. **長文解説** - タイプの本質、思考様式、強み・弱みを深掘り
6. **強みと弱み** - 箇条書き形式
7. **恋愛・人間関係** - ロマンス、友情、子育てなど関係性別セクション
8. **職業・キャリア** - 向いている仕事・職場行動傾向
9. **有名人リスト** - 同タイプの著名人・歴史的人物
10. **シェアボタン** - 各セクション下部に常駐

**特徴的なUX設計:**

- 結果URLが固定（例: `/ja/infj-性格`）であり、受検者以外も訪問できる
- 結果ページ自体が充実した読み物コンテンツになっており、シェアリンク経由でも十分な価値がある
- プレミアム機能（キャリアキット等）への誘導が自然に組み込まれている

---

### 16TEST（16test.uranaino.net/animals/）

日本の精密性格診断テスト。動物16タイプを採用。累計2,380万人以上が受験。

**動物タイプの一覧:**
オウム、フクロウ、ウマ、シカ、シマウマ、ネコ、イヌ、キリン、ウシ、ヒツジ、ライオン、トラ、ゾウ、ウサギ、クマ、カバ（16種）

**各タイプページの構成:**

1. タイプ名 + MBTIコード（例: "オウム型 (ENTP)"）
2. タイプキャッチコピー（例: "議論が大好きな発明家"）
3. 動物イラスト
4. 長文性格説明（3〜4段落。強みと弱みを両方記述）
5. 他タイプとの相性（16パターン全てを関係名ラベル付きで表示: "双対関係"、"鏡像関係" 等）
6. 同タイプの偉人・アニメキャラクター（画像付き、Amazonアフィリエイトリンク含む）
7. 「診断スタートボタン」（スティッキーCTA）
8. アプリDLへの誘導（AppStore/Google Play）

**特徴的な要素:**

- キャラクター一覧が豊富（レオナルド・ダ・ビンチ、ヒソカ、JOKERなど多様な有名人・キャラ）
- 相性情報が全タイプ網羅されており、「友達・パートナーのタイプも知りたい」という動機を生み出す
- ページ上部にはCTAバナー（"どうぶつたちがあなたを待っています"）が表示され、未受検者へ診断を誘導

### 動物キャラ占い うらなえる（unkoi.com/special/animal/）

生年月日から60種類の動物キャラを診断する占い系サイト。

**特徴:**

- 生年月日入力という低ハードルな入口
- 60タイプという豊富なバリエーション（希少感・ユニーク感が生まれる）
- 占い文脈で結果を提示（運勢・相性・恋愛傾向を含む）

---

### マイナビウーマン 動物タイプ診断（woman.mynavi.jp）

10問の設問に答える形式。女性向けメディアの記事型診断。

**特徴:**

- フローチャート形式（設問への回答でルーティング）
- 記事構成の中に診断が組み込まれている
- X（旧Twitter）・Facebook・LINEのシェアボタンをページ上部に配置
- 全9種類の動物タイプ

---

### CanCam 自分を動物に例えると診断（cancam.jp）

女性向けファッション誌系サイトの診断コンテンツ。10問形式。

**特徴:**

- 画像主導の記事型フォーマット
- 設問が画像カルーセル形式（スライド型）
- X・Facebook・LINEシェアボタンをタイトル直下に配置
- 結果も画像で提示

---

### BuzzFeed Quizzes

日本版は展開縮小しているが、グローバル版では依然高トラフィック。

**特徴:**

- 結果を短文テキスト＋画像でまとめ、SNSシェアに最適化
- 平均シェア回数1,900回以上（BuzzSumo調査）
- 「あなたは〇〇です！」という断言スタイルで共有しやすいテキストを生成
- 結果ページのOGP画像が「答え」を直接表示するため、SNSタイムラインで目立つ

---

### vonvon（jp.vonvon.me）

2015年開始、14言語対応。1日最高500万PVを記録した診断特化サービス。

**特徴:**

- 結果がSNSシェア前提の設計（シェアしないと結果が見づらい仕掛けも過去に存在）
- ランダム性のある結果テンプレート（毎回異なる結果が出る）
- Facebook連携を活用した初期の爆発的拡散モデル
- 現在はプライバシー問題での信頼低下あり

## 3. エンゲージメントを高める要素

### 3-1. 「まさに自分だ！」と感じさせる心理設計

**バーナム効果（フォラー効果）の活用:**
バーナム効果とは、誰にでも当てはまるような曖昧で一般的な表現を「自分だけに当てはまる」と感じてしまう心理現象。性格診断コンテンツが「当たっている！」と感じられる核心的なメカニズム。

具体的な設計手法:

- 「あなたは外では明るく振る舞いますが、一人になると疲れを感じることがあります」などの二面性の記述
- 「〇〇な面もある一方で、〇〇な部分も持ち合わせている」という矛盾を含む描写
- 普遍的な欲求（承認、成長、愛、独自性）への言及

**具体性と抽象性のバランス:**

- 抽象的すぎると「自分のことではない」と感じる
- 具体的すぎると当たらなくなる
- 日常的なシーン（職場、恋愛、友人関係）に落とし込んだ具体例が有効

**ポジティブなフレーミング:**

- 弱みや欠点も「成長の余地」「個性」として肯定的に表現
- 「〇〇なあなたは実は〇〇な才能がある」という発見型の表現
- 自己肯定感を高める文章が共感と満足度を上げる

**適度な自虐要素:**

- 全肯定ではなく「私ってこうかも（笑）」と共感できる弱みの記述
- 完璧すぎる自己像より、欠点を含む人間らしいキャラクター描写
- ホットリンクの調査でも「少しの自虐要素がシェアを促進する」と指摘

---

### 3-2. シェアされやすくする要素

**心理的動機:**

| 動機                 | 設計への示唆                                           |
| -------------------- | ------------------------------------------------------ |
| 自己表現・自己開示   | 「私はこういう人」を間接的に伝えられる結果テキスト     |
| 承認欲求             | ポジティブな結果でシェアしたくなる自己像               |
| 話題提供             | 「これやってみて！」と友人に勧める口実になるコンテンツ |
| アイデンティティ確認 | 友人の結果と比べることで関係性を深めたい欲求           |
| 希少性・ユニーク感   | 「〇〇タイプは全体の5%」など希少性を示す               |

**技術的・UI的要素:**

- **SNSシェアボタン（X・LINE・Facebook）** を結果表示直後に配置
- **シェア用テキストを自動生成** - ハッシュタグ付きの文章を事前に用意
- **OGP画像の最適化** - タイプ名・動物イラストをSNSカード画像に含める
- **Twitter Cardをsummary_large_imageに設定** - タイムラインで目立つ大型画像
- **URLシェア** - 結果ページのURLが固定で、シェア後も同じ内容が見られる
- **シェアを促すコピー** - 「友達の結果も見てみよう！」などの誘導文

**結果の視覚化:**

- 動物イラストが印象的で「絵として」シェアしやすい
- タイプ名が短くキャッチーで、SNS投稿文に使いやすい（例: 「私はライオン型でした！」）
- 特性のパーセンテージグラフ（数値の見える化）がシェアコンテンツとして機能

---

### 3-3. 離脱を防ぎ回遊を促す要素

- **相性情報** - 「友達・彼氏彼女はどのタイプ？」という興味から他タイプページへ誘導
- **有名人・キャラクター同一タイプ** - 「あのキャラと同じ！」という発見と共有
- **再診断ボタン** - 「もう一度試す」のCTA
- **他の診断への内部リンク** - 関連診断コンテンツへの回遊
- **Q&A / 深掘りコンテンツ** - 結果下部に関連コラム

## 4. 受検者本人と第三者（シェアリンク経由）の体験差別化

### 4-1. 現状の問題点

多くの診断サイトでは、受検者本人とシェアリンク経由の訪問者に同一のページを見せている。これにはいくつかの問題がある:

- **受検者本人への問題:** 自分が受けたフローを経由せずに結果URLを見ると、文脈が失われる（まれに）
- **第三者への問題:** いきなり結果ページを見せられても「なぜこの動物なのか」がわからない。診断への参加動機が生まれにくい

### 4-2. 差別化すべき体験の設計方針

**受検者本人（診断完了直後の閲覧）**

目的: 結果への納得感、自己理解の深化、シェア促進

優先コンテンツ:

1. 「あなたは〇〇型でした！」という強いパーソナライズメッセージ
2. 診断結果サマリー（どの軸がどのスコアだったか）
3. 「なぜこの結果になったか」の説明（回答との整合性を示す）
4. 深掘りコンテンツへの誘導（詳細説明ページへのリンク）
5. シェアCTA（目立つ位置に。「友達にも教える」ボタン）
6. 再診断や関連診断への誘導

**第三者（シェアリンク経由）**

目的: 診断への参加誘導、コンテンツの理解と興味喚起

優先コンテンツ:

1. 「○○さんは〇〇型です」という第三者視点の紹介文（または汎用的な説明）
2. このタイプの概要説明（コンテキストなしでも読める設計）
3. 強い診断参加CTA（「あなたは何型？診断してみよう！」）
4. 他のタイプの存在を示し、「自分は違うかも」という好奇心を刺激

### 4-3. 実装上の工夫

**URLパラメータによる状態管理:**

```
/result/lion?source=self    # 受検者本人
/result/lion?source=share   # シェアリンク経由（または source パラメータなし）
```

**コンテンツの動的切り替え:**

- 受検者本人: スコアバー、「あなたの結果：X%」などパーソナライズ要素を表示
- 第三者: 「○○さんがシェアしました」バナー＋「自分も診断する」CTA を目立たせる

**OGP設計:**

- SNSカード画像には「動物名＋キャッチコピー＋動物イラスト」を含める
- タイトルタグに「私は〇〇型でした！あなたは？」という誘い文句を入れる
- descriptionには「〇〇タイプの特徴：...」という情報を含め、クリック前から価値を伝える

## 5. 結果ページの推奨構成（総合）

以下は上記調査を踏まえた、動物性格診断の結果ページの推奨構成:

```
[ヒーローセクション]
- 動物イラスト（大きく、印象的に）
- タイプ名（例: "ライオン型"）
- キャッチコピー（例: "天性のリーダー"）
- タイプの希少度（例: "全体の約8%"）

[SNSシェアCTA] ← 最初に配置（「まず見せたい」心理を活用）
- X、LINE、Facebook シェアボタン
- シェア文テンプレート付き

[スコア表示]（受検者本人のみ）
- 各軸のパーセンテージバー
- 「外向型 67%」などの数値

[性格説明（メイン）]
- 3〜5段落
- 強み → 弱み → 人間関係 の順
- 日常シーンの具体例を含む
- バーナム効果を意識した普遍的共感ポイント

[有名人・キャラクター同一タイプ]
- 3〜6名
- 「あの人と同じ！」という発見感

[相性情報]
- 相性の良いタイプ（1〜2個）
- 相性の悪いタイプ（1個）
- 各タイプへのリンク（回遊促進）

[深掘りセクション]
- 恋愛・仕事・友人関係それぞれの傾向
- アコーディオン形式（読む量の制御）

[SNSシェアCTA] ← 2回目（読了後のタイミング）
- 「友達にシェアする」ボタン
- 「友達の結果も見てみよう」誘導

[他タイプへの誘導]
- 「このタイプと相性がいい〇〇型も見てみる」
- 全タイプ一覧へのリンク

[再診断・関連診断CTA]（第三者向けに特に重要）
- 「あなたは何型？診断してみる」ボタン
```

## 6. 調査から得られた主要な知見

1. **読み物としての価値が来訪者を増やす**: 16Personalitiesが圧倒的な人気を誇る理由は、結果ページが独立した価値を持つ読み物になっているため。シェアリンク経由のPVも高い。

2. **動物という媒体は「絵になる」**: テキストタイプ（INTP等）より動物名の方が視覚的に伝わりやすく、SNSシェア時のOGP画像が効果的。

3. **相性情報が口コミを促進する**: 「友達も診断して相性を確かめたい」という動機は、診断コンテンツへの最強の流入源。

4. **「希少性」の演出が有効**: 「あなたのタイプは全体の〇%」という情報は「自分だけ感」と「友達に話したい感」の両方を刺激する。

5. **シェアボタンは2回設置する**: 「すぐシェアしたい人」と「読んでからシェアしたい人」両方に対応するため、ページ上部と下部に配置。

6. **第三者誘導CTAが収益性に直結**: シェアリンク経由の訪問者を診断受検者に転換することが、コンテンツの最大のKPIである。このCTAの設計が最も重要。

7. **バーナム効果の意識的な活用**: 「まさに自分だ」と思わせる文章は技術であり、汎用性と具体性の絶妙なバランスで実現する。完全な個別化より、共感できる普遍性の方がシェアを促進する。
