# 「四字熟語」検索ユーザーの属性・意図の徹底分析

## 概要

四字熟語は日本語コンテンツの中でも特に検索需要が安定・多様なジャンルである。学習・実用・エンタメの三軸にわたる幅広いユーザー層が存在し、それぞれが異なる意図を持って検索している。本レポートでは、検索キーワードの種類と意図、ユーザー属性、競合サイトの構成、需要の季節変動、「面白い」検索の特性を整理する。

## 3. 競合サイト分析

- **規模**: 9,241語収録（全日本四字熟語普及連盟公式）、毎日平均22,000人アクセス
- **コンテンツ構成**:

  - 50音順検索
  - 人気ランキング（デイリー・週間・月間・年間）

  - 著名人33人との関連付け
  - SPI頻出100選、季節別一覧等の特集コンテンツ
- **差別化ポイント**: 業界最大規模のデータ、公式団体運営の権威性、ランキング機能

- **差別化ポイント**: 場面・用途別の逆引き機能の充実、漢検級別の網羅性

### 3-3. 四字熟語の百科事典（idiom-encyclopedia.com）

- **規模**: 多数の特集記事（1000選・中学受験TOP100・SPI TOP100等）

- **コンテンツ構成**: 学術的・辞典的な詳細解説、文学からの用例が豊富
- **差別化ポイント**: 信頼性・権威性が高い学術寄り辞典、出典・語源の詳細解説

### 3-5. goo辞書（dictionary.goo.ne.jp/idiom）

- **収録内容**: 新明解四字熟語辞典
- **注記**: 2025年6月25日にサービス終了（競合が1つ減少した状況）
- **コンテンツ構成**: カテゴリ検索、場面別分類
- **差別化ポイント**: 大手ポータルの集客力（サービス終了により消滅）

### 3-6. 競合の共通弱点

調査を通じて見えてきた競合の共通的な弱点：

1. **例文が教科書的・形式的**: 実際の会話・文章に使えるリアルな用例が少ない
2. **AIの視点がない**: 現代的・デジタル時代の文脈での解釈や用例がない
3. **エンタメ性と学術性のどちらかに偏っている**: 両立しているサイトが少ない
4. **インタラクティブ要素が薄い**: 一覧・辞書型が多く、ゲーム・クイズ等の体験型が少ない
5. **SNS・現代語との接続が弱い**: Z世代が四字熟語に触れるきっかけを作れていない

## 5. 「四字熟語 面白い」で検索するユーザーの詳細分析

### 5-1. 検索の動機

「四字熟語 面白い」で検索するユーザーは、純粋な知的好奇心・娯楽目的が主体である。具体的な動機は以下に分類できる：

- **暇つぶし・娯楽**: 「面白い漢字ってないかな」という気軽な関心
- **雑学・トリビア収集**: 人に話して驚かせたい、知識として蓄えたい
- **SNS・会話ネタ探し**: 投稿・チャットに使えるユニークな表現を探している
- **行事・スピーチのアクセント**: 面白い言葉を取り入れてユーモアを出したい
- **子どもへの学習動機付け**: 「面白い四字熟語で子供の興味を引きたい」という親・教師

### 5-2. 求めているコンテンツの特性

面白い四字熟語を求めるユーザーが満足するコンテンツの特性：

**面白さの種類別分類への期待**

- 意味が面白い（逆説的・皮肉・奇妙な意味の言葉）
- 語感・音が面白い（「魑魅魍魎」のような独特の響き）
- 見た目が面白い（珍しい漢字・字面の印象）
- 現代的に笑える（皮肉が効いた言葉、共感できる言葉）
- 創作四字熟語（現代語・日常語を四字に見立てた遊び）

**コンテンツ形式への期待**

- ランキング形式（「絶対に面白い四字熟語TOP10」等）
- 説明に笑いやひねりがある（堅い解説より親しみやすい表現）
- 具体的な使用例・シチュエーションの説明
- 画像・ビジュアルとの組み合わせ（字面を見せる工夫）

### 5-3. このユーザー層の属性

- **年齢層**: 幅広いが10〜30代が中心
- **目的の一時性**: 学習系と違い、その場のエンタメ消費が多い
- **シェア意欲**: 面白いと感じたらSNSでシェアしやすい層
- **回遊性**: 「面白い」から「かっこいい」「難しい」へと他カテゴリへの遷移が起きやすい
- **漢字への障壁**: 学習目的の層と違い、難しい漢字を苦手とする場合もある

### 5-4. 満足度が高いコンテンツ設計の要件

1. **一覧性**: まず30〜200個の選りすぐりリストを見られること
2. **分かりやすい解説**: 意味だけでなく「なぜ面白いか」の説明
3. **適度なユーモア**: 解説文自体に笑いの要素がある
4. **意外性**: 「知らなかった」「そんな意味なの」という驚きを提供
5. **共感性**: 日常に重ねて読める現代的な補足説明
6. **用途の提示**: 「これはXで使えます」「スピーチのつかみに使えます」等のガイダンス

## 6. yolos.net四字熟語辞典への示唆

### 6-1. 差別化の核心

競合が提供できていない価値として、以下がある：

**AI視点の例文・解説**

- 人間が書いた紋切り型の例文でなく、AIが現代的・ユニークな視点で書いた例文
- 「AIがこの四字熟語をどう理解し、どう使うか」という独自の視座
- これはすでに実装されており（B-223）、最大の差別化ポイントである

**現代文脈との橋渡し**

- SNS・デジタル時代における四字熟語の使い方の提示
- 若者世代が共感できる現代的な解釈

### 6-2. 優先すべきコンテンツ拡充の方向性

検索意図の多様性を踏まえると、以下の優先順位でコンテンツを充実させる価値がある：

1. **用途別特集ページ**（実用・選定系の需要が大きい）
   - 「座右の銘にしたい四字熟語」「年賀状の四字熟語」「体育祭スローガン」等
2. **面白い四字熟語の特集**（エンタメ需要・シェア誘発）
   - AI視点の「面白さ」の解説が差別化になる
3. **学年・試験種別リスト**（学習需要・定期的なアクセス）
   - 中学受験頻出・SPI頻出等の網羅的リスト
4. **季節連動コンテンツ**（年賀状・受験・入学シーズンへの対応）

### 6-3. 注意すべき競合優位性
