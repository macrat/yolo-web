# ブログ・Webコンテンツにおける画像の重要性とSEOベストプラクティス

## 1. サムネイル画像がCTRに与える影響

### Google Discover

- 幅1200px以上の大きな画像を使用することが掲載の前提条件
- 大きな画像カードは小さなサムネイルと比較してCTR **5%向上**（Google公式）
- ケーススタディでは **79%のCTR向上** や **300%のトラフィック増加** の事例が存在
- ある記事がWeb検索353クリックに対してDiscoverで **21,700クリック** を獲得（61倍差）

### SNSシェア（OGP画像）

- OGP画像あり/なしでCTRが **2〜3倍** の差
- X（Twitter）での横長大画像はCTR **30〜60%向上**
- 適切なOGP設定でSNS流入が **30%以上増加** したケースが多数

### ブログ一覧ページ

- ECの類似研究では画像1枚追加で転換率が **2倍**（eBay調査）
- 67%の消費者が「画像品質」を意思決定の最重要要素として評価

## 2. 画像とSEO

- 上位ランクのGoogleページの **97%** が画像を含む
- Core Web VitalsのLCPは画像最適化で最大改善可能。LCP改善がモバイル収益を **42%向上** させた事例も
- Article schemaでは16:9・4:3・1:1の3アスペクト比での画像提供が推奨
- `max-image-preview:large` メタタグがDiscover掲載に必須

## 3. AI生成画像のSEO上の扱い

- **Googleはペナルティを課さない**（Gary Illyes公式発言）
- C2PA/IPTCメタデータによる出所情報付与が業界標準化しつつある
- Google「About this image」機能でC2PAメタデータが活用される
- Imagen 4はSynthID/C2PAを自動付与するため、透明性要件を自然に満たす

## 4. 診断・占い系コンテンツでの画像活用

- ユーザーごとに異なる **動的OGP画像生成** がバイラル拡散の鍵
- 「自分の結果をシェアしたい」心理と組み合わせることで自発的拡散が起こる

## 5. テキストのみ vs サムネイル付きカード

- 視覚コンテンツはテキストより **94%多く閲覧** される
- 脳の画像処理速度がテキストより **60,000倍** 速いという神経科学的根拠がある

## yolos.netへの示唆

現在のyolos.netは：

- ブログカードにサムネイル画像なし（テキストのみ）
- OGP画像は単色背景+テキスト+絵文字の簡素なデザイン
- `public/` に静的画像アセットなし

**改善の優先度が高い項目：**

1. ブログ記事へのサムネイル画像追加（Google Discover対応、一覧ページのCTR向上）
2. OGP画像のデザイン品質向上（SNSシェア時のCTR向上）
3. `max-image-preview:large` メタタグの確認・追加
4. Article構造化データへのimage属性追加
