# 日本語辞典サイト競合分析と差別化戦略（四字熟語・ことわざ）

## 調査概要

yolos.netの四字熟語辞典（400語→3,000語拡充予定）とことわざ・慣用句辞典（2,000語新規構築予定）の企画において、主要競合との差別化戦略を策定するための調査を実施した。実際に各サイトを訪問し（Playwright使用）、コンテンツ・UI/UX・インタラクティブ要素を詳細確認した。

---

## 3. jitenon.jpの成長パターン推定

### 運営開始時期（著作権表示から判断）

- jitenon.jp（メインサイト）: © 2011-2026 → 2011年から運営
- yoji.jitenon.jp（四字熟語辞典）: © 2012-2026 → 2012年開始
- kotowaza.jitenon.jp（ことわざ辞典）: © 2014-2026 → 2014年開始

### コンテンツ拡充ペース

- 2026年3月の更新ログに複数語の追加が記録（月に数件〜数十件ペースで継続的に追加）
- データ販売ストアの存在（差分データダウンロード機能付き）から、収録語数を計画的に拡大し続けていることが確認できる
- 2026年2月には「頭を上げる」「耳にこびりつく」など6語を一括追加した記録あり

### ドメイン・SEOの観点

- 14〜15年間の運営実績によりドメインオーソリティが大きく蓄積されている
- 辞典ファミリーサイト間の内部リンク構造が強力（漢字辞典・四字熟語辞典・ことわざ辞典・国語辞典・類語辞典等が相互リンク）
- 四字熟語辞典サブドメインがjitenon.jp全体トラフィックの約38%を占める（最大コンテンツ）
- データ販売ストアによりB2Bビジネスも展開

## 4. 新規辞典サイトのSEOタイムラインと現実

### 一般的なSEO効果が出るまでの期間

- Googleの公式見解: 「4ヶ月〜1年」
- コンテンツSEOの実情: 新規ドメインは通常3〜6ヶ月以上
- 辞典・情報サイトの特性: キーワード数が多いため、ロングテールキーワードから徐々に流入が始まる

### 辞典サイト特有のSEO課題

1. **競合の歴史的優位性**: jitenon.jpは14年間のドメイン実績・外部リンク蓄積がある。これを短期間で超えることは難しい
2. **E-E-A-T（経験・専門性・権威性・信頼性）**: 辞典サイトはGoogleが「有益な情報提供サイト」として評価するため、信頼性シグナルが重要
3. **インデックス量**: 3,000語の四字熟語＋2,000語のことわざ＝5,000ページ以上が対象。ページ数は有利に働く
4. **差別化コンテンツの重要性**: 意味だけなら既存サイトと重複するため、例文・由来・クイズ等の追加価値が検索評価でも重要

### yolos.netの有利な点

- yolos.netはすでに既存コンテンツで一定のドメイン実績があり、純粋な新規ドメインより有利
- 辞典×ゲーム×診断の構成は既存サイトに存在しない。ユニークな内部リンク構造が形成できる

## 5. yolos.netが差別化できるポイント

### 5-1. コンテンツの質的差別化（全競合に対して優位に立てる）

1. **例文・用例の充実**
   - jitenon.jpの四字熟語・ことわざページには例文がなし
   - コトバンクは例文があるが辞書引き体験で親しみにくい
   - 各語に2〜3個の自然な例文を提供することで明確な差別化が可能

2. **由来・豆知識の深掘り**
   - jitenon.jpは出典情報のみで由来解説が浅い
   - 語源の背景ストーリー、中国の故事の解説など、読み物としての価値を提供できる

3. **現代的な使い方ガイド**
   - ビジネスシーンでの使い方、SNS投稿の例など現代文脈での使い方
   - AI生成ならではの「現代版例文」「使い分けガイド」が有効

4. **英語・多言語対応**
   - 英語訳と英語での説明（訪日外国人・日本語学習者向け）
   - 主要競合はすべて日本語のみ（差別化の機会）

### 5-2. インタラクティブ要素の差別化

競合の状況:

- jitenon.jpはクイズを別サイト（quiz.jitenon.jp）に分離しており、辞典ページ内にインタラクティブ要素はない
- sanabo.comには診断機能があるが、辞典との連携は限定的

yolos.netで実現可能な差別化:

1. **辞典ページ内でのクイズ連携**: 各語のページ下部に「この四字熟語に関するクイズに挑戦」ボタンを設置し、既存クイズシステムと直接連携
2. **性格診断・タイプ診断との連携**: 「あなたにぴったりの座右の銘四字熟語」診断を辞典トップから誘導
3. **関連語のビジュアル探索**: 類義語・対義語・同テーマ語の関係を視覚的に表示

### 5-3. モダンUI/UXの優位性

現状の競合デザイン:

- jitenon.jpは2010年代前半のデザイン。広告が多く画面が重い
- proverb-encyclopedia.comはWordPressブログ的デザイン
- sanabo.comは最も古いデザイン（2000年代的）

yolos.netの差別化:

1. **ダークモード**: 全競合が未対応。夜の学習や読書体験の大幅な向上
2. **Next.js SSGによる高速表示**: 競合の多くはサーバーサイドまたは動的レンダリング。体感速度で明確な差
3. **読みやすいタイポグラフィ**: 漢字とひらがなのバランスを考えた日本語フォント設計
4. **広告なしの体験**: 現状yolos.netには広告がない。これはUXの大きな優位性

### 5-4. 辞典×ゲーム×診断の横断的体験

これはyolos.netのユニークな強みで、他の辞典サイトはすべて辞典のみに特化している。

横断体験の設計:

- ことわざ辞典を引く → 「このことわざを使ったゲームに挑戦」へのリンク
- 四字熟語クイズで間違える → 該当語の辞典ページに自動誘導
- 診断結果に「あなたの性格を表す四字熟語」を表示 → 辞典ページへのリンク

## 7. 戦略的提言

### 収録語数の優先度設定

- 四字熟語3,000語: jitenon.jpの6,642語には数で及ばないが、各語の情報量（例文・由来・クイズ連携）で圧倒することが重要
- ことわざ2,000語: jitenon.jpの7,195語には数で劣るが、同様に質で勝負

### SEO上の現実的な見通し

- 新規コンテンツとしてのハンデは6〜12ヶ月は検索流入が限られると想定
- ただしロングテール（具体的な語名での検索）から先に流入が始まる
- 「一心不乱 意味 例文」のような具体的な検索では、例文が充実していればyolos.netが上位に入る可能性が高い
- yolos.netはすでにドメイン実績があるため、純粋な新規ドメインより有利な状況

### 実装優先度（高）

1. 例文の提供（全競合の四字熟語・ことわざページに例文がない。即効性が高い）
2. 由来・語源の詳細解説（読み物価値の創出）
3. 辞典ページ内のクイズへの直接誘導（滞在時間・回遊率向上）
4. ダークモード対応（差別化のシグナルとして機能）

### 実装優先度（中長期）

1. 診断連携（「座右の銘診断」など）
2. 多言語対応（英語訳・説明）
3. 現代的な使い方コンテンツ（SNS・ビジネス文脈）
4. 関連語のビジュアル探索
