# コンセプト来訪者価値計測: 道具箱はGoalに向かっているか

- 調査日: 2026-06-21（cycle-257 / B-535）
- 目的: サイトコンセプト「日常の傍にある道具（道具箱）」が憲法のGoal（来訪者価値→PV）へ向かっているかを、初めて来訪者価値で実測する。道具箱は cycle-167 で核に据えられ cycle-226 で「不変項」化されたが、その正当化は技術的実機実証であり来訪者価値の計測ではなかった（B-510 初回分析は本調査まで未実施）。
- データ: GA4 BigQuery `yolo-web-gcp.analytics_524708437.events_*`（2026-03-28〜06-20）、Search Console BigQuery、GA4 MCP。3本の foreground 計測サブエージェントで取得。生データと実SQLは `tmp/cycle-257/{value-by-section,toolbox-usage,recent-traffic-growth}.md`。
- データの限界（正直に明記）: サイト全体は約119PV/週と母数が小さい。多くのセル n<50 は参考値。道具箱イベントは観測8日と短い。過度な一般化は禁物。以下は「頑健な事実」と「参考値/暫定」を区別して記す。

## 1. 来訪者価値の在り処（セクション別・頑健）

| セクション            | organic着地(全期間) | 中央エンゲージ               | 回遊(別ページ遷移) | 評価                                              |
| --------------------- | ------------------- | ---------------------------- | ------------------ | ------------------------------------------------- |
| `/play`(診断・ゲーム) | 184/201（最大）     | 約109〜117秒（他の15〜20倍） | 中                 | **集客・滞在の両軸で1位**。頑健                   |
| `/`(道具箱トップ)     | ほぼ0               | 短い（一桁秒）               | 53%（最強）        | organic集客なしの**回遊ハブ**。価値の生産者でない |
| `/dictionary`(辞典)   | organic 2番手       | 精読中央値が全セクション最高 | 低〜中             | 地味な良コンテンツ                                |
| `/blog`               | organic 3番手       | 最下位クラス                 | 最下位クラス       | 読み捨て傾向                                      |
| `/tools`(道具詳細)    | n小                 | 二極化                       | n小                | **判断保留（参考値）**                            |

- **頑健な事実**: 来訪者価値（集客＋滞在の深さ）は `/play`（診断・ゲーム）に明確に集中。これは小母数でも頑健で cycle-256 観測とも一致。
- **頑健な事実**: トップ `/`（道具箱）は全期間を通じて organic 集客がほぼ0。直接来た既知/内部ユーザーを送る回遊ハブとしては機能するが、来訪者価値を新規に生んでいない。
- **頑健な事実**: 再訪はサイト全体で機能していない（returning が全セクション一桁人）。**「毎日使う・反復利用」を前提にした設計は現データでは支持されない**。価値は「単発訪問の質」で測るべき。

## 2. 道具箱の実利用（B-510 初回分析・暫定）

- 道具箱カスタムイベントの初出現 2026-06-13。**観測8日**（実発火3日）。
- `tile_first_interaction`: 5件 / 3セッション / 3ユーザー。
- `toolbox_preset_select` / `toolbox_tile_add` / `toolbox_tile_remove` / `toolbox_reset`: **全履歴でゼロ件**。道具箱の中核機能（並べる・プリセット・カスタマイズ）の利用は観測ゼロ。
- トップ `/` 着地10セッション中、タイル操作は1のみ（その1ユーザーは active 24日で**開発者/owner本人の可能性が高い**）。除外すると一般訪問者の道具箱実利用は**実質0**。
- 同8日間の比較: `/play` の `level_end` は44セッション（道具箱実利用の40倍以上）。
- **サンプル十分性の正直な評価**: 8日では「使われている/いない」を統計的に断定できない。ただし「整備後8日で並べ替え・プリセット系が完全ゼロ・実利用は開発者疑い1件」というシグナルの薄さは事実。§1の「`/`は全期間organic集客ほぼ0」（頑健）と合わせると、道具箱コンセプトに来訪者トラクションがある兆候は現時点で見られない。

## 3. 直近の流入増（成長機会・実在）

- **2026-06-17 から organic 約2.3倍**（直近4日 organic 14.5/日 vs 直前10日 6.4/日）。GA4 と Search Console の独立2ソースが同じ立ち上がり日を示す＝ノイズでない。母数は1日十数〜25人規模。
- 出どころ: ほぼ全量 google/organic。SCで伸びたクエリは「似てるキャラ診断／自分に似てるキャラクター診断／あなたに最も近いアニメキャラ診断」系。
- 着地: **`/play/character-personality` への一点突破**（organic着地 約12倍）。SC impression が 06-16→06-17 で急拡大（~20/日→100超/日）。**6月にコード変更なし**（git確認）＝Google側の自然な順位上昇が起点。
- 着地後の行動（38 organicセッション）: 診断完了79%・平均滞在約2.9分・モバイル80%・全員日本＝**単体エンゲージは良好**。しかし**回遊ゼロ（PV/セッション1.24）・観測窓内の再訪ゼロ**。「遊んで満足して帰る」。
- 判定: 増加は実在。トレンドか減衰かは順位変動由来のため数日の追計測が必要（過大解釈は禁物）。回遊・再訪ゼロ＝伸びしろが大きい点で成長の芽として有望側。

## 4. 計測の含意（事実の範囲で）

- データは「**診断・ゲーム中心**」を強く支持。`/play` が集客・滞在の両軸で頑健に1位、かつ今まさに organic が診断へ流入を増やしている。
- **道具箱コンセプト（道具を核・診断を息抜きに格下げ）は、来訪者価値のトラクションを示していない**。トップは organic 集客ほぼ0、中核機能の利用は観測ゼロ。最も価値が届いている診断を格下げする設計は、データと逆を向いている。
- 辞典・色は「地味だが精読される良コンテンツ」で、文化系の支え。ブログはテコ入れか縮小の判断対象。
  - 注（metric 依存・reviewer 再検算）: 辞典は**イベント単位の精読**中央値が最高だが、**セッション滞在**ではブログ(中央33.5秒) > 辞典(15.7秒)。評価が metric で逆転するため、「ブログ縮小」の具体化時は両 metric を見ること（片方だけで縮小判断しない）。
- 「反復利用（日常の傍にある）」前提は現データで支持されない。価値は単発訪問の質。
- 次に必要な計測: `/play` 内のどの個別診断が効いているかの深掘り、直近流入増の06-21〜06-25 追計測（定着確認）。
