## 4. 現状の四字熟語メタ生成（事実の記録）

`src/lib/seo.ts` の `generateYojiPageMetadata`（四字熟語 400 ページが共通利用）が出力する内容（例: 至誠通天）:

- title: `「至誠通天」の意味・読み方 - 四字熟語辞典 | yolos.net`
- description: `四字熟語「至誠通天」（しせいつうてん）の意味: 誠を尽くせば天に通じること`

データ `src/data/yoji-data.json`（400 件）は yoji / reading / meaning / difficulty / category / origin / structure / sourceUrl / example を保持している（cycle-246 計画時に sourceUrl/difficulty の列挙抜けを訂正）。

## 留意点・限界

- メタ改善の SEO 効果は遅効性（再クロール待ち）であり、施策後すぐの SC データには反映されない（効果確認は数週間後）。
- 「1 ページ目に出ているのにクリック 0」という観測から順位ではなくタイトル/スニペット側の要因と切り分けられるが、無名ドメインの信頼性差（権威辞書サイトとの並列）も CTR に影響しうる（推定）。
- 診断系（/play/yoji-level）は四字熟語関連で唯一クリックが立っているクエリ群（CTR 2.0〜3.5%）。
