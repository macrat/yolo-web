## 4. URL構造変更の影響評価

### 4-1. 影響範囲の定量評価

URL構造変更の対象となりうるカテゴリ（`/blog/`, `/games/`, `/quiz/`, `/fortune/`, `/tools/`, `/dictionary/`）の合計SEO資産:

| 指標                   | 数値    |
| ---------------------- | ------- |
| インデックス済みURL数  | 95件    |
| 検索クリック数（90日） | 20件    |
| 検索表示回数（90日）   | 1,777回 |

これらのURLのパスが変更された場合、Googleが新URLを再インデックスするまでの間、検索流入が一時的にゼロになるリスクがある。301リダイレクトを適切に設定すれば、数週間〜数ヶ月で回復が見込まれるが、インデックス再登録には時間を要する。

### 4-2. カテゴリ別リスク評価

| カテゴリ     | リスク     | 根拠                                                                                                     |
| ------------ | ---------- | -------------------------------------------------------------------------------------------------------- |
| /blog/       | **高**     | インデックス26件、検索クリック14件（全体67%）。変更による影響が最も大きい                                |
| /quiz/       | **中**     | インデックス3件。`/quiz/yoji-level`はクリック4件・表示90回と安定した流入がある                           |
| /dictionary/ | **中**     | インデックス63件と最多。ただし現状の検索クリックは2件にとどまる。今後のSEO成長に向けた資産として扱うべき |
| /tools/      | **低〜中** | インデックス3件、クリックゼロ。ただし表示回数186回あり、成長余地がある                                   |
| /games/      | **低**     | インデックスなし、検索クリックなし。SEO資産の損失はなし                                                  |
| /fortune/    | **低**     | インデックスなし、検索クリックなし。SEO資産の損失はなし                                                  |

### 4-3. 変更時の注意事項

1. **301リダイレクト必須**: すべての旧URLから新URLへ301リダイレクトを設定することで、被リンクのPageRankを引き継ぎ、Googleへの変更シグナルを伝える
2. **サイトマップの即時更新**: 変更後すぐにサイトマップを更新し、Search Consoleから再クロールをリクエストする
3. **変更のタイミング**: トラフィックが現状少ないため、影響は比較的軽微。ただし、/blog/が成長中であることを考慮し、blogカテゴリの変更は慎重に行う
4. **モニタリング**: 変更後4〜8週間はSearch ConsoleとAnalyticsで流入を監視する

## 5. 結論

### 現状のSEO資産評価

サイト開設から約3ヶ月が経過し、SEOは成長の初期段階にある。検索クリック21件という数字は絶対値として小さいが、以下の点が重要:

- **blogが検索流入の主力**: クリックの67%、表示回数の44.6%がblogから発生
- **dictionary最大のインデックス資産**: 63件がインデックス済みで、将来的な検索流入の基盤になりうる
- **/quiz/yoji-levelが最も安定した検索流入源**: クリック4件・表示90回・CTR 4.44%と、クイズカテゴリで唯一の有力SEOページ

### URL変更の総合判断

現時点でSEO資産が小さいため、URL変更による損失は絶対値として限定的（クリック最大20件/90日の一時的喪失）。ただし、以下の点に注意:

- `/blog/`カテゴリは検索流入が成長中であり、変更後のリダイレクト設定を確実に行うことが重要
- `/dictionary/`は将来的な資産として63件のインデックスURLがあり、丁寧な移行対応が求められる
- `/games/`、`/fortune/`はインデックスなしのため、URL変更のSEOリスクは実質ゼロ

適切な301リダイレクト設定を前提とすれば、URL構造変更のSERPへの長期的な影響は軽微と評価できる。
