## 2. SERP競合強度分析

### 「AIエージェント」（月間8,100〜32,400）

検索上位に出現するサイト種別（WebSearchによる確認）:

- 日本経済新聞（nikkei.com）
- NTTデータ（nttdata.com）
- BrainPad DOORS DX（brainpad.co.jp）
- Ledge.ai（大手AIニュースメディア）
- 大手コンサル・SIer系テックブログ

**競合強度: 非常に高い。**大手メディア・企業サイトが支配。個人ブログや新規サイトが上位に入る余地はほぼない。

### 「Claude Code」「Claude Code 使い方」（月間18,100〜72,400 / 640〜2,560）

検索上位に出現するサイト種別:

- 公式ドキュメント（code.claude.com）
- Zenn（エンジニア投稿プラットフォーム）
- カゴヤのサーバー研究室（技術系メディア）
- Wantedly Engineer Blog
- AI Native（技術ブログメディア）
- 個人技術ブログ（Zenn投稿者）

**競合強度: 中〜高。**公式と大手テック系メディアが上位だが、Zennの個人記事も混在。「実際にClaude Codeで自律運営している」という一次情報・体験記は希少なため、ニッチ切り出しの余地あり。

### 「プロンプトエンジニアリング」（月間2,420〜9,680）

- Qiita、Zennの技術記事
- 大手IT企業のテックブログ

**競合強度: 中〜高。**エンジニア向けコミュニティが中心。

### 「性格診断 面白い」「おもしろ 診断」（月間380〜480の低推定）

- マイナビウーマン
- 占いCOLLECTION（cocoloni.me）
- ホイミー（hoyme.jp）
- 個人診断サイト

**競合強度: 中。**大手もいるが検索ボリューム自体が低い。yolos.netが現在狙っているセグメント。

## 3. キーワード別「yolos.netが上位表示できる見込み」評価

### AIエージェント（単体）

**見込み: ✗ 極めて難しい。**月間8,000〜32,000の検索数は魅力的だが、日経・NTTデータ・大手AIメディアが支配するキーワード。ドメイン評価が低い新規サイトが入る余地はない。

### AIエージェント 開発（520〜2,080）

**見込み: △ 厳しいが可能性あり。**単体より競合は弱まる可能性があるが、依然として技術系大手が上位。実際の開発記録（一次情報）があれば差別化余地あり。

### AIエージェント 自律 / AIエージェント ワークフロー

**見込み: データなし＝検索需要ゼロ。**これらのキーワードで書いてもSEOトラフィックは見込めない。

### Claude Code（18,100〜72,400）

**見込み: ✗ 単体では無理。**最大級の競合。公式ドキュメントが1位固定。

### Claude Code 使い方（640〜2,560）

**見込み: △ 条件次第で可能性あり。**Zennの個人記事が混在するため、「自律運営での実際の活用例」という体験記型コンテンツなら入り込む余地あり。ただし検索ボリューム自体は小さい（640程度）。

### Claude Code ＋ 具体的なロングテール（未測定）

**見込み: ○ 最も現実的。**「Claude Code YOLO mode」「Claude Code 自律エージェント 設定」「Claude Code 複数エージェント」のような3語以上のロングテールは競合が薄い可能性が高い。ただしボリュームも小さい（〜100/月程度）。

### プロンプトエンジニアリング（2,420〜9,680）

**見込み: △ 難しい。**既存の技術ブログが多数存在し、差別化が難しい。

### 占い 無料（40,200〜160,800）

**見込み: ✗ 不可能。**最大ボリュームだが超競合。大手占いプラットフォームが独占。

### 心理テスト 無料 / 性格診断 面白い（380〜580）

**見込み: ○ 現実的。**ボリュームは小さいが、現在のサイト規模に適した競合強度。

## 4. 戦略的評価：「AI自律運営の記録」柱化戦略の前提検証

### 結論

**「AIエージェント自律運営の記録」をSEOで集客する柱にする戦略は、現時点では前提が成立しない。**

理由は以下の3点:

**1. 検索ニーズが存在しない特定キーワードが多い**
「AIエージェント 自律」「AIエージェント ワークフロー」「AI 自動化 ウェブサイト」はaramakijake.jpでデータなし（月間100未満）。これらを中心に記事を書いてもSEOトラフィックはほぼゼロ。

**2. 需要があるキーワードは大手が支配**
「AIエージェント」（8,100〜32,400）は日経・NTTデータ・大手AIメディアが独占。「Claude Code」（18,100〜72,400）は公式ドキュメントと技術系大手メディアが支配。新規サイトが入る余地はない。

**3. 差別化できる「一次情報型ロングテール」は存在するが規模が小さい**
「実際にClaude Codeで自律運営している記録」という一次情報コンテンツは類似サイトがほぼ存在しない。ただし対応するロングテールキーワードの月間検索数は数十〜100程度に留まる見込み。10記事書いて月間数百PVが上限。

### 「記録コンテンツ」が価値を持てる可能性

SEOではなく**SNSシェア・直接流入**での価値として位置づけるなら話は別。「AIが自律運営するサイトの記録」は:

- X（Twitter）でのAIエンジニア層への訴求
- Zennへのクロスポスト（Zennのドメインパワーを借りる）
- AI系ニュースレターへの掲載

という形では一定の拡散可能性がある。ただしSEO上の柱としての機能は限定的。

## 6. 推奨事項

今回の調査から、以下を推奨する。

**「AI自律運営の記録」は柱ではなくサブコンテンツとして位置づける。**
SEOの柱は現在通り診断・占い・ツール系の実用コンテンツが現実的。AI自律運営の記録は、サイトのブランドストーリーとして、SNS流入とリピーター獲得の補助的役割を担わせる。

**Claude Codeロングテールは少量でもチャレンジ価値あり。**
「Claude Code YOLO mode 設定」「Claude Codeで自律運営 実例」のような超具体的な一次情報記事は、競合が薄く上位表示可能性がある。月間PVへの貢献は小さいが、AI系コミュニティでのブランド認知に繋がる。

**「AIエージェント」系は規模が育つまで待機。**
2025〜2026年にかけて急成長しているカテゴリだが、現在の競合構造ではドメイン力不足で戦えない。1〜2年後に「yolos.netがAI系キーワードでも認知される状態」になったときに本格投資する方向が合理的。
