# コンテンツ価値評価レポート — B-165フェーズ4-1判断材料

## 1. レポートの目的と前提条件

### 1.1 調査の目的

本レポートは、PMがB-165フェーズ4-1（低価値コンテンツの削除）の実施判断を下すための客観的な材料を提供することを目的とする。具体的には以下を明らかにする。

- 各コンテンツカテゴリが検索経由でどの程度の価値を生んでいるか（現在の実績）
- 各カテゴリの成長トレンドが上向きか、横ばいか、停滞かの傾向
- 低パフォーマンスのカテゴリがサイト全体の評価に悪影響を与えている兆候の有無
- 現時点のデータで判断可能な範囲と判断が時期尚早な範囲の区分

本レポートは推奨アクションを含まない。PMが推測によるバイアスなくデータを評価できるよう、判断材料の提示に徹する。

### 1.2 対象期間とデータの制約

**サイト公開日**: 2026-02-13
**本レポート作成日**: 2026-03-27
**経過日数**: 42日

この42日という期間は、SEOの文脈では極めて短い。一般的に、Googleが新規サイトを「信頼できるサイト」として安定的に評価するまでには3〜6ヶ月程度かかるとされている（「Googleサンドボックス効果」として知られる現象）。そのため、現時点のGSCおよびGAデータは、各カテゴリの長期的な検索価値を判断する根拠として十分ではない可能性がある。

特に注意すべき制約:

- **絶対数が少ない**: GSCの総クリック数は29件（42日間）。統計的に有意な判断を下すには少すぎる
- **初期効果の影響**: サイト公開直後のDirectトラフィック（告知効果）はすでに減衰しており、オーガニック検索への移行期の途中にある
- **インデックス進行中**: 一部のコンテンツはまだ完全にインデックスされていない可能性がある（GSCデータより、dictionary/kanjiだけで推定1,111ページ以上がインデックス済みだが、実際のページ数2,136を大きく下回る）
- **新コンテンツが含まれる**: 診断8種は2026-03-08〜17の公開（サイト公開後23〜32日目）であり、評価期間が特に短い
