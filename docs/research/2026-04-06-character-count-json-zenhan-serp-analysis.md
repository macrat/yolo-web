# 「文字数カウント」「JSON整形」「全角半角変換」キーワード検索競合SERP分析

## 調査背景

yolos.netには文字数カウント（/tools/char-count）、JSON整形、全角半角変換の各ツールが存在するが、過去30日間の検索流入はほぼゼロ（文字数カウントはオーガニック検索2PV、その他は0PV）。「文字数カウント」は月間122万件という高需要キーワードにもかかわらず、流入がない原因を競合SERP分析により解明する。

## 5. 競合比較と勝機評価

### 5-1. 3キーワードの競合強度比較

| キーワード     | 月間検索数 | 大手の存在             | 個人参入余地                   | 難易度総合 |
| -------------- | ---------- | ---------------------- | ------------------------------ | ---------- |
| 文字数カウント | 約122万    | ラッコ・LUFT・Evernote | あり（個人も複数上位）         | 非常に高い |
| JSON整形       | 約2.7万    | ラッコのみ             | あり（開発者向け・多様な競合） | 高い       |
| 全角半角変換   | 約6,600    | ラッコサーバーのみ     | あり（個人サイトが上位多数）   | 中程度     |

### 5-2. 競合に共通する強み（yolos.netに欠けているもの）

1. **ドメイン年齢・被リンク**: 競合の多くは2015〜2020年代に運営開始し、5年以上の実績がある。yolos.netは新参（2024年〜）。
2. **ツール機能の充実度**: ラッコやLUFTは文字数だけで9種類以上の指標を表示。yolos.netは「文字数・バイト数・単語数・行数」の4指標のみ。
3. **サイト全体のコンテンツ量**: ラッコ（130+ツール）、so-zou（多数のテキストツール）など、サイト全体のコンテンツ量が多いためドメイン権威が高い。
4. **外部被リンク**: 競合はブログ記事・SNS等での紹介を通じて多数の被リンクを保有。

### 5-3. yolos.netが勝てる可能性の評価

**「文字数カウント」（月間122万）: 勝機ほぼなし（短〜中期）**

- 月間170万PV超のラッコツールズが圧倒的な強さ
- LUFTは35年超の法人・ドメイン歴
- サルワカは月間300万PVの超大型個人メディア
- yolos.netが新参サイトとして上位10位以内に入るには、ドメイン年齢・被リンクの面で5〜10年単位の時間が必要
- ただし現状でもインデックスはされており、長期的に改善する余地はある

**「JSON整形」（月間2.7万）: 勝機わずか（長期的に可能性あり）**

- ラッコツールズが強いが、その他は個人〜中小サイト
- 開発者コミュニティからの被リンクを獲得できれば上昇の可能性
- ツール機能に「バリデーション」「圧縮」「ツリー表示」など付加価値を追加することで差別化可能

**「全角半角変換」（月間6,600）: 最も勝機あり**

- so-zou.jpのような個人サイトが1位に入っている
- 大手の参入が少ない
- 月間ボリュームは小さいが、個人サイトが上位に入れる可能性が3キーワード中最も高い

## 6. 根本的な問題と改善方向性

### 6-1. 現状の本質的問題

yolos.netのツールが検索上位に入らない根本的な理由は次の通り。

1. **ドメイン権威の不足**: yolos.netは2024年〜の新参サイト。被リンクがほぼない。Googleは新規ドメインに対して「サンドボックス」効果があるとされており、初期は順位が上がりにくい。
2. **ツールのコモディティ化**: 文字数カウントは競合が機能的に飽和状態。同じ機能では差別化できない。
3. **コンテンツの薄さ**: 各ツールページに使い方の解説がほとんどない。Googleが評価するE-E-A-Tの観点でも薄い。

### 6-2. 現実的な改善策

**短期（1〜3ヶ月）で効果が期待できる施策:**

1. **ツールページへの解説コンテンツ追加**: 「文字数カウントとは」「使い方」「FAQ」などを500〜1000文字程度追加。ロングテールキーワード（「文字数 カウント スペース除く」「文字数 バイト数 違い」等）での流入増が期待できる。
2. **全角半角変換ページへの注力**: 3キーワードの中で最も競合が弱い。so-zou.jpと同等以上の機能と解説を追加する。
3. **構造化データ（FAQ schema）の実装**: 検索結果でのリッチスニペット表示によりCTRを向上させる。

**中長期（3ヶ月〜1年）で効果が期待できる施策:**

1. **被リンク獲得**: GitHubや技術ブログへのツール紹介記事投稿、開発者コミュニティへの参加。
2. **JSON整形ページの機能強化**: バリデーション・構文エラーハイライト・ツリービューなど競合に匹敵する機能追加。
3. **テキスト処理ツール群の充実**: 全角半角変換・文字数カウント・JSON整形を「テキスト処理ツール」群として関連付け、内部リンクでまとめる。

## 7. まとめ・結論

### 三大キーワードの勝機総括

**結論: 短中期では「文字数カウント」での上位表示はほぼ不可能。**

「文字数カウント」は月間122万という魅力的な需要があるが、ラッコツールズ（企業運営、170万PV超）、LUFT（35年超の法人）、サルワカ（月間300万PVの大型個人メディア）が上位を占めており、新参のyolos.netが数ヶ月で上位に入れる余地は現実的にない。

一方、「全角半角変換」（月間6,600件）は個人サイトが上位に入っており、最も参入しやすい。ただし月間ボリュームも小さい。

### 最も効果的な施策の優先順位

1. **全角半角変換ページの強化**: 機能・解説ともに競合水準まで引き上げる（最も達成可能性が高い）
2. **各ツールページへの解説コンテンツ追加**: ロングテールキーワードでの流入を狙う（地味だが継続的な効果）
3. **JSON整形ページの機能強化**: バリデーション付加などで差別化し、開発者コミュニティへのアプローチ

「文字数カウント」での検索流入を増やすことへの期待値を下げ、競合の弱い別のキーワードや別コンテンツタイプ（ブログ記事、診断・占い系）でのトラフィック獲得を優先するのが現実的な戦略である。
