# 英語圏市場キーワード需要・競合調査

## 調査の背景と目的

yolos.netはcycle-158でOwnerから「既存コンテンツの1つを英語圏向けにローカライズする小さな実験」が提案された。cycle-162では競合調査は行ったが「英語圏の検索ボリューム実測」が不足と指摘された。本レポートはその欠缺を補う。

フルi18n（B-056）は無期限延期だが、1コンテンツの英語版実験は前向きな方針。

**重要注意:** 本調査の検索ボリューム数値はすべて「推定値」である。有料ツール（Ahrefs・Semrush）の非公開データへの直接アクセスはなく、公開されているレポート・ブログ記事・ツールページから間接的に収集したデータを使用している。信頼度は各項目に明記する。

## 6. 結論と推奨

### 英語圏実験の最有望候補: 日本の伝統色辞典

以下の理由から、過去のリサーチ（2026-04-05-partial-english-localization-experiment.md）の結論を本調査のデータで裏付けられた。

1. **需要の存在**: 月間1〜5万件の推定需要、Wikipedia記事・複数英語サイトの存在が証明
2. **競合の弱さ**: 大手メディアは参入していない。既存の英語専業サイト（GetZenQuery）はデザインが質素
3. **差別化の余地**: 視覚的に美しいUIと日本語サイトとしてのオーセンティシティで勝負できる
4. **コンテンツ量**: 数百色の英語解説として適切な1ページ規模

### 診断系・ツール系英語KWへの参入は非推奨

- 診断系（personality quiz等）は16Personalities、BuzzFeed等の超大手が支配
- ツール系（character count、JSON formatter等）は特化サイトが月間数百万ビジットを獲得済みで新規参入の余地なし
- "remove line breaks"(2,900/月)等の小ボリュームKWは競合が弱いが、得られるトラフィックも微小

### 長期的視野での可能性

英語圏の日本文化への関心は2024〜2025年にかけて継続的に高まっている（訪日外客3686万人記録、アニメ産業海外収益56%）。この追い風に乗った日本文化特化コンテンツの英語版は、SEO競合が低い現在が参入の好機と言える。

ただし、検索ボリュームの数値が低い点は現実的に評価する必要がある。英語版実験は「大きなトラフィック獲得」より「仮説検証と学習」として位置づけることが適切。

## 付記: データの信頼性について

本調査では有料SEOツール（Ahrefs・Semrush）への直接アクセスがないため、以下の方法で推定値を導出した。

| 推定方法                               | 信頼度 | 使用箇所                                      |
| -------------------------------------- | ------ | --------------------------------------------- |
| Similarweb/Semrush公開データからの逆算 | 中     | character count、JSON formatter               |
| KeySearch公開キーワードリスト          | 高     | daily horoscope（823,000/月）                 |
| ケーススタディ記事の明示的なデータ     | 高     | remove line breaks（2,900/月）、TextFixer収益 |
