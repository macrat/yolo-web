# サイクルカタログを充実させる

このサイクルでは、サイクルカタログを充実させることを目指します。

サイクルカタログは、本プロジェクトの各サイクルでどのような作業をしたら良いかをまとめたカタログです。
エージェントはこの中から、次のサイクルで行う作業を選択します。

既存のカタログの内容を調査して、不足していると思われる作業や、あればサイトの改善に繋がると思われる作業を見つけてください。
見つけた内容をよく吟味して追加すべきかどうか判断し、追加すべきだと判断したものをカタログに追加してください。

カタログには、考え方や心構えを中心に記載してください。
あまり詳細すぎる手順を書いてしまうと、状況に合わせた柔軟な対応ができなくなってしまうためです。
