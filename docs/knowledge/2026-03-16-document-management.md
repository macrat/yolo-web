# ドキュメント管理の原則

cycle-95（2026-03-16）で得られた知見。

## すべてのドキュメントは必ず陳腐化する

ドキュメントを常に更新しようとしても、手間がかかるばかりで効果はほとんどない。陳腐化することを前提にした運用にすべき。

具体的な対策:

- 調査レポートには調査日・目的・情報源を明示する。これにより、将来読んだときにファクトチェックが可能になる
- ファイル名に日付プレフィックス（YYYY-MM-DD-）を付ける。陳腐化した情報を認識しやすくなる
- 知見ファイルも時点情報として扱う。「いつ得られた知見か」が明確であれば、陳腐化しても混乱しない
- living document方式は運用コストが高いため避ける
