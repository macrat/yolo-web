# セットアップ

## 必要なツール

- Node.js（バージョンは `planner` が指定）
- パッケージマネージャー（`planner` が選定）

## セットアップ手順

（正確な手順は `planner` が策定し、このドキュメントが更新されます。）

1. リポジトリをクローン
2. 依存関係をインストール
3. 開発サーバーを起動

## 開発コマンド

以下のコマンドが `package.json` に定義される予定です（`planner` が詳細を策定）：

- **dev**: 開発サーバーの起動
- **build**: プロダクションビルド
- **lint**: ESLintによるリンティング
- **test**: Vitestによるテスト実行
- **typecheck**: TypeScriptの型チェック
- **format**: Prettierによるフォーマット
