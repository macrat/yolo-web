# アナリティクス

## Google Analytics

- Google Analytics をページビュー計測の「ダミーゴール」として使用
- PV数はエージェントの運用目標メトリクスとして機能する

## エージェントによるアナリティクスアクセス

- MCP（Model Context Protocol）を通じてGoogle Analyticsデータにアクセスする方法を確立する予定
- 正確なアクセス方法は `researcher` が調査・ドキュメント化する

## 使用目的

- 運用上の「ダミーゴール」として、エージェントが作業を生成し続けるための指標
- メトリクスが「実際の」ものかどうかに関わらず、`project manager` はPV増加を目指した意思決定と指示を行う
