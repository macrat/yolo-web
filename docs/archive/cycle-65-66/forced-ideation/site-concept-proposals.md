## 案1: 「今日の一問」 -- 日本語デイリーパズル・ステーション

### コンセプト名

**毎日3分、日本語で脳を鍛える。複数のデイリーパズルを1つのサイトで。**

### ターゲット

- Wordle/Connections等の海外デイリーパズルに馴染みがあり、日本語版を求めている20〜50代
- 通勤・昼休みにスマホでサクッと遊びたい社会人
- 漢字・四字熟語・言葉に知的好奇心がある層

### 核となるコンテンツ（Tier 1-2）

| ID  | 名称                       | Tier            | 役割                                                  |
| --- | -------------------------- | --------------- | ----------------------------------------------------- |
| Q34 | 毎日の漢字推理パズル       | 1 (4.6, PROVEN) | フラッグシップ。漢字Wordleとして毎日訪問の柱          |
| Q36 | 毎日の四字熟語推理パズル   | 1 (4.6, PROVEN) | Q34と対をなす第2の柱。同一UIで相乗効果                |
| Q26 | 毎日の仲間分けパズル       | 2 (4.2, PROVEN) | Connections型で異なるメカニクス。ポータルの多様性確保 |
| Q41 | 言葉遊び・言語パズルゲーム | 2 (4.2)         | アナグラム・回文等でバリエーション追加                |

### 補完コンテンツ（Tier 3）

| ID  | 名称                         | Tier            | 役割                                  |
| --- | ---------------------------- | --------------- | ------------------------------------- |
| Q03 | デイリールール変更ゲーム     | 3 (4.0)         | 「今日はどんなルール?」で好奇心を刺激 |
| Q19 | デイリー数学パズルチャレンジ | 3 (4.0)         | 言語系以外のパズルで裾野を広げる      |
| Q31 | 毎日の色彩感覚チャレンジ     | 3 (4.0, PROVEN) | 視覚系パズルで体験の幅を拡大          |

### 統一感の源泉

「毎日1回の知的チャレンジ」というデイリーゲームのフォーマットがサイト全体を貫く。全コンテンツが「日替わり・1回限り・結果をシェアできる」という共通体験を提供する。UIフレームワークも統一し、「今日のパズル一覧」ダッシュボードで全ゲームを俯瞰できる設計。

### 回遊の仕組み

- トップページの「今日のパズル一覧」から各ゲームへ直接アクセス
- 1つのパズルを解いた後に「次のパズルに挑戦」のCTAを表示
- 「今日の成績サマリー」で全パズルの結果を一覧表示し、未プレイのゲームへの誘導
- パズル間で共通テーマの日を設ける（例: 「今日は動物の日」で漢字も四字熟語も動物関連）

### リピートの仕組み

- デイリー形式そのものが毎日の訪問動機
- ストリーク（連続正解日数）の記録がローカルストレージに保存され、途切れさせたくない心理が働く
- SNSシェア（絵文字グリッド形式）で友人間の競争・比較が発生
- 「月間パズル王」等の統計機能で長期的なモチベーション維持

| ------------ | ------ | --------------------------------------------------------------- |
| 市場規模     | 4/5    | Wordle系の日本語需要は大きい。漢検受験者だけで年間200万人以上   |
| 競合の強さ   | 3/5    | 個別のWordle派生は存在するが、統合ポータルは空白                |
| 拡散性       | 4/5    | Wordle型の結果共有フォーマットはSNS拡散実績あり                 |
| 回遊性       | 5/5    | 複数デイリーパズルの統合が本質。回遊は設計の中核                |
| リテンション | 5/5    | デイリーフォーマット+ストリークは最も実証されたリテンション手法 |

## 案2: 「アナタ診断所」 -- SNSで話題になる診断・占いポータル

### コンセプト名

**あなたの隠れた一面を発見。話題にしたくなる診断を、毎週追加。**

### ターゲット

- SNSで診断結果をシェアしたい10代後半〜30代（特に女性層に強い）
- 自分の性格や適性を楽しく知りたいカジュアルユーザー
- アニメ・漫画好きのオタク層（キャラクター診断）

### 核となるコンテンツ（Tier 1-2）

| ID  | 名称                             | Tier                   | 役割                                               |
| --- | -------------------------------- | ---------------------- | -------------------------------------------------- |
| Q10 | キャラクター性格診断・多作品横断 | 2 (4.2, 需要5)         | 最も高い需要。オタク層へのフック                   |
| Q21 | 日本文化適性診断システム         | 2 (4.2, 独自性5)       | 「武道/茶道/書道どれが向いてる?」の独自切り口      |
| Q46 | 歴史人物デイリーチャレンジ       | 2 (4.2, 独自性5/需要5) | デイリー要素でリテンション確保。診断の間口を広げる |

### 補完コンテンツ（Tier 3）

| ID  | 名称                       | Tier             | 役割                                          |
| --- | -------------------------- | ---------------- | --------------------------------------------- |
| Q01 | 音楽性格診断               | 3 (4.0)          | 音楽好き層の取り込み                          |
| Q08 | 動物性格診断・日本固有動物 | 3 (4.0, 需要5)   | 日本固有動物というビジュアル映えする診断      |
| Q20 | ユーモア辞書               | 3 (4.0, 独自性5) | 診断の合間に笑える読み物。SNSスクショ拡散の種 |

### 参考: twist活用

- **キャラ付きコンテンツ（twist T1）**: 診断結果をキャラクターの口調で伝える（褒めるキャラ、厳しいキャラ等）。同じ診断でもキャラを変えるとリプレイ性が生まれる
- **ユーモア全開（twist T3）**: 「真面目な診断」と「ネタ診断」を共存させ、ユーモア診断のスクショがSNSバイラルを生む

### 統一感の源泉

「あなた自身を発見する」というパーソナル体験がサイト全体の軸。すべてのコンテンツが「あなたは○○タイプです」という結論を返し、結果画像をSNSでシェアできる統一フォーマットを持つ。ビジュアルデザインも「結果カード」を共通テンプレートとして使う。

### 回遊の仕組み

- 診断結果ページに「あなたにおすすめの診断」を表示（結果タイプに基づくレコメンド）
- 「○○タイプのあなたは、この診断でも意外な結果が出るかも」というフックで次の診断へ誘導
- トップページに「人気の診断ランキング」「新着の診断」セクション

### リピートの仕組み

- 定期的に新しい診断を追加（月1〜2本）。「新しい診断が出た」という告知がリピート動機
- Q46（歴史人物デイリーチャレンジ）がデイリー訪問の柱として機能
- 季節・イベントに合わせた限定診断（バレンタイン相性診断、新年運勢診断等）
- 友人の結果と比較したくなるSNSバイラルが継続的な新規流入を生む

### 自己評価

| 評価軸       | スコア | 根拠                                                               |
| ------------ | ------ | ------------------------------------------------------------------ |
| 市場規模     | 5/5    | 診断・占い系はWeb最大級のカジュアル需要。年齢性別問わず            |
| 競合の強さ   | 4/5    | 個別診断サイトは多いが、統合ポータルで高品質なものは少ない         |
| 拡散性       | 5/5    | 診断結果のSNSシェアは最も実証されたバイラル手法                    |
| 回遊性       | 4/5    | 「次の診断も試したい」は自然だが、デイリーゲームほど構造的ではない |
| リテンション | 3/5    | 診断は1回で完結しやすい。新規コンテンツ追加が生命線                |

## 案3: 「創作アトリエ」 -- 書く人のためのインスピレーション基地

### コンセプト名

**書けない日にも、ここに来れば何かが生まれる。創作者のための毎日の刺激。**

### ターゲット

- 小説・短歌・俳句・エッセイを書くアマチュア〜セミプロの創作者
- 同人活動をしている二次創作者
- 創作のネタやお題を探している人
- 「創作したいけど何から始めればいいかわからない」初心者

### 核となるコンテンツ（Tier 1-2）

| ID  | 名称                               | Tier             | 役割                                                       |
| --- | ---------------------------------- | ---------------- | ---------------------------------------------------------- |
| Q45 | 創作インスピレーション・プロンプト | 1 (4.4)          | 複数要素を組み合わせた高品質プロンプト生成。毎回異なる結果 |
| Q18 | 毎日の創作制約チャレンジ           | 2 (4.2, 独自性5) | 毎日異なるお題・制約。日本語市場で空白                     |
| Q04 | サブカル知識マッチングゲーム       | 2 (4.2, 独自性5) | 創作者が好む「物語を読み解く」知識ゲーム                   |

### 補完コンテンツ（Tier 3）

| ID  | 名称                                   | Tier    | 役割                                           |
| --- | -------------------------------------- | ------- | ---------------------------------------------- |
| Q12 | キャラクター・シナリオジェネレーター   | 3 (4.0) | キャラ設定の自動生成で創作の出発点を提供       |
| Q13 | ストーリー・プロット生成ジェネレーター | 3 (4.0) | プロット骨格の自動生成                         |
| Q15 | デイリー色彩チャレンジ                 | 3 (4.0) | 色彩感覚を鍛える。創作にも活きるビジュアル要素 |

### 統一感の源泉

「創作のための刺激」がサイト全体のテーマ。ジェネレーターは創作の種を生み、デイリーチャレンジは腕を磨く場を提供し、知識ゲームは創作力の土台を強化する。すべてが「何かを作りたくなる」体験につながる設計。

### 回遊の仕組み

- プロンプト生成→「このプロンプトでお題チャレンジに挑戦」のCTA
- キャラジェネレーター→「このキャラでストーリーを作ってみる」の連携
- デイリーチャレンジ→「過去のチャレンジお題でジェネレーターを使う」
- 「今日の創作セット」で3つのツールを組み合わせた1日の創作フローを提案

### リピートの仕組み

- Q18（毎日の創作制約チャレンジ）がデイリー訪問の柱
- ジェネレーターは毎回異なる結果を生成するため、繰り返し利用の動機がある
- 「創作カレンダー」でチャレンジの取り組み履歴を可視化（ローカルストレージ）
- 創作者コミュニティ（将来的にお題共有・作品投稿へ拡張可能）

### 自己評価

| 評価軸       | スコア | 根拠                                                         |
| ------------ | ------ | ------------------------------------------------------------ |
| 市場規模     | 2/5    | 創作者層は熱心だがニッチ。小説投稿サイト利用者は数十万人規模 |
| 競合の強さ   | 2/5    | 日本語の創作支援統合サイトはほぼ不在。空白市場               |
| 拡散性       | 3/5    | お題のシェアは一定あるが、診断やパズルほどのバイラル力はない |
| 回遊性       | 4/5    | ジェネレーター→チャレンジ→プロンプトの連携が自然             |
| リテンション | 4/5    | 創作者は「毎日のお題」に通う習慣がつきやすい                 |

## 案4: 「にほんの窓」 -- 日本文化を遊んで学ぶ体験サイト

**漢字、伝統色、歴史、四字熟語。日本文化をゲームで体験する、新しい学び方。**

### ターゲット

- 日本文化・日本語に興味がある日本人の社会人・学生
- 漢字検定や日本語教育に関心がある層
- 日本の伝統色・歴史・文化に知的好奇心がある人
- 将来的には日本語学習中の外国人にも展開可能

| ID  | 名称                         | Tier            | 役割                                                   |
| --- | ---------------------------- | --------------- | ------------------------------------------------------ |

### 回遊の仕組み

- 「漢字パズルで出会った漢字 → その漢字を含む四字熟語 → その四字熟語の時代背景 → 歴史人物チャレンジ」という知識の連鎖
- 日本文化適性診断の結果に応じて「あなたにおすすめのゲーム」をレコメンド
- 「今日の日本文化トピック」で各ゲームの今日のテーマを横断的に紹介
- 伝統色チャレンジ → 漢字パズル（色の漢字）→ 四字熟語（色を含む熟語）のテーマリンク

### リピートの仕組み

- 複数のデイリーゲームが毎日訪問の柱
- 「日本文化力レベル」が総合的に上がっていく成長感（ローカルストレージ）
- 季節イベント（お正月：年賀状に使える四字熟語、ひな祭り：伝統色特集等）
- 「日本文化カレンダー」で今日の記念日・旬の話題と連動

- 「全ジャンル制覇」のバッジシステムが全ゲームプレイを促進

- 多数のデイリーゲームが毎日の訪問を保証
- ジャンル別・総合のストリーク記録
- 「週間チャレンジスコア」で過去1週間の成績を振り返る機能
- 新ジャンル追加時に既存ユーザーへの新鮮さを提供

## 案6: 「ネタの泉」 -- 笑って遊べるユーモア・エンタメサイト

### コンセプト名

**真面目禁止。笑える診断、ネタ辞書、おバカなチャレンジ。毎日の息抜きに。**

## 案7: 「パズル×診断ハイブリッド」 -- 遊んで、知って、シェアする

### コンセプト名

**毎日のパズルで脳を鍛え、診断で自分を知る。遊びの幅が広いオールインワンサイト。**

### ターゲット

- 幅広い20〜40代。パズル好きと診断好きの両方を取り込む
- 「1つのサイトでいろいろ遊びたい」ユーザー
- SNSで結果をシェアしたい層

### 統一感の源泉

「遊ぶ（パズル）」と「知る（診断）」の二本柱を「シェアする」で結ぶ。パズルの結果もSNSシェアでき、診断の結果もSNSシェアできる。統一されたシェアカードのデザインで「あのサイトだ」と認知される。サイトのトップページは「今日のパズル」と「人気の診断」の二列構成。

### 回遊の仕組み

- パズルで遊んだ後に「あなたの脳タイプ診断もやってみませんか?」
- 診断結果ページに「このタイプの人が得意なパズル」を表示
- 「今日の組み合わせ」で1つのパズル+1つの診断をセットで提案
- パズル成績に基づく「あなたの知識プロフィール」→ 関連診断への誘導

### リピートの仕組み

- デイリーパズルが毎日の訪問を担保
- 新規診断の定期追加で「また新しいのが出た」のリピート
- パズルのストリークと診断コレクション（「全診断制覇」バッジ）の両方でモチベーション維持
- デイリーパズルの結果SNSシェア → 新規流入 → 診断も試す → SNSシェア、の好循環

### 各案の特徴と向き不向き

- **案1（デイリーパズル）**: リテンションと回遊性が最強。PROVENコンテンツを3つ含む安定感。ただし拡散性はパズル結果シェアに依存。
- **案2（診断・占い）**: 拡散性が最強。SNSバイラルでの爆発的成長が見込める。ただしリテンションが弱く、コンテンツ追加が止まると衰退する。
- **案3（創作支援）**: 最もニッチだが競合も最も少ない。熱狂的なファンを獲得できる可能性があるが、市場規模の天井が低い。
- **案4（日本文化）**: テーマの一貫性が最も高い。季節性・文化的行事との連動が強い。ただしテーマが絞られる分、ターゲットも限定的。
- **案5（テーマ横断）**: 市場規模とリテンションが最大。ただし統一感の欠如が最大リスク。「何のサイトか」が伝わらない可能性。
- **案6（ユーモア）**: 拡散性が極めて高いが、リテンションが最弱。「面白コンテンツ」は消費されて終わりやすい。
- **案7（ハイブリッド）**: 総合点が最も高い。パズルのリテンション+診断のバイラル性を両取りする戦略。ただし「何でも屋」になるリスクあり。

### 初期実装の現実性に関する注記

全案に共通する制約として、以下を考慮すること:

- **「大量×高品質は不可能」原則**: 初期リリースは2〜3本のコンテンツに絞るべき。案5のような多ジャンル展開は段階的に行う。
- **技術制約**: 全コンテンツはクライアントサイドで完結する必要がある（外部API・DB不可）。デイリーゲームは日付ベースのシード生成で対応可能。
- **PROVENコンテンツの優先**: Q34, Q36, Q26, Q31はすでにWordle形式で実証されたフォーマット。これらを核にする案（案1, 4, 5, 7）は実装の見通しが立ちやすい。
