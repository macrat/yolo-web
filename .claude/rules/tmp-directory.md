# 一時ファイルの管理ガイド

`./tmp/` ディレクトリは、本サイクルでだけ利用する一時的なファイルを保存するための場所です。
ここに保存したファイルはコミットされず、いつまで残るかは保証されません。

将来にわたって必要となるドキュメントは `./docs/` ディレクトリに保存してください。
その他のファイルで将来も必要なものは、適切な場所に保存してください。
ただし、スクリーンショットなどは容量が大きいので、ウェブサイトに掲載するものでない限りは `./tmp/` に保存してください。

サイクル終了時にすでに不要なファイルはなるべく削除してください。

## Do/Don't

**Do**: 作業中のコマンドの出力を保存する。

**Do**: サブエージェント間でやりとりするための一時的なファイルを保存する。

**Do**: スクリーンショットなどの一時的なファイルを保存する。

**Don't**: 将来にわたって必要なドキュメントを保存する。

**Don't**: 他のディレクトリにあるファイルから `./tmp/` に保存したファイルについて言及またはリンクする。
