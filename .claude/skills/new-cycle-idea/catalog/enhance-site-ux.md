# サイトのUI/UXを改善する

このサイクルでは、サイト全体のUIをレビューして、ユーザーエクスペリエンスの向上を図ります。

researcherやreviewerを活用してサイトを注意深く分析し、エンドユーザーにとって気持ち良く使えるウェブサイトになるように改善を施してください。

サイトのUXを考える際は、そのコンテンツの来訪者は誰か・どんな意図や動機で来るか・市場はどの程度かを、その都度調査・推論して記録してください（ターゲット定義の仕組みは cycle-278 のオーナー指示で廃止。定義済みの言葉に頼って利用者の推論を止めないため——docs/rebuild-plan.md §4）。
