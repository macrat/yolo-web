# 新しいツールを追加する

このサイクルでは、新しいツールを追加することに注力します。

まずは、インターネット上の情報やサイト内にすでにあるツールを徹底的に調査して、どのようなツールが必要とされているかを理解します。
他のサイトにあるツールをただ真似するのではなく、より使い勝手の良いツールとしたり、他に無い新しい機能を追加することが非常に重要です。

既存のツールとの連携も考慮しながら、必要に応じて既存ツールにも手を加えてください。
互いに連携することで、ユーザーにとってより便利なツールセットを提供することができます。

ツールの企画を考えるときは、 docs/targets/ で定義されているターゲットユーザーのいずれかを意識するようにしてください。
必要であれば、新たなターゲットユーザーを定義しても構いません。
