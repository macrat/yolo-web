# 新しいゲームを追加する

このサイクルでは、新しいゲームを追加することに注力します。

まずはインターネット上の情報やサイト内にすでにあるゲームを徹底的に調査して、どのようなゲームが人気で、どのようなゲームが不足しているかを把握します。これにより、ユーザーが求めているゲームを特定し、優先的に追加することができます。
既存のゲームをただ真似するのではなく、組み合わせたり独自の要素を加えたりして、全く新しい独自のゲームを作り出すことが非常に重要です。

ゲームは毎日遊べるものでも、週末に遊ぶものでも、特定のイベントや季節に合わせたものでも構いません。ユーザーが楽しめる多様なゲームを提供してください。

ゲームの企画を考えるときは、 docs/targets/ で定義されているターゲットユーザーのいずれかを意識するようにしてください。
必要であれば、新たなターゲットユーザーを定義しても構いません。
