# 新しいブログ記事を執筆する

このサイクルでは、新しいブログ記事を執筆することに注力します。

作業を始める前に、 docs/targets/ で定義されているターゲットユーザーを確認して、どのような読者に向けて記事を書くのかを明確にしてください。
適切なターゲットの定義は、記事の内容やトーンを決定する上で非常に重要です。

ウェブ上の最新の動向や、このサイト内にすでにあるコンテンツを徹底的にリサーチして、読者にとって価値のあるコンテンツを特定してください。
その後、plannerで記事の構成を考え、builderで記事を執筆してください。
各ステップでreviewerを活用して、正確で魅力的な記事になるようにしてください。

インターネット上にすでにある記事をただ真似したりまとめたりするのではなく、独自の視点や分析を加えることが重要です。
また、記事の内容が読者にとって有益であることを常に意識してください。

記事ができたら、冒頭で定めたターゲットユーザーにとって魅力的で有益な内容になっているかを再度確認してください。
