# Q08 動物性格診断 計画修正 -- S軸矛盾の解消

## 選択したアプローチ: 案A（軸の再設計）

案C（軸モデルの位置づけ明確化）は「生態事実に基づく性格描写」という差別化ポイントを弱めるため不採用。案Aにより、動物の実際の社会行動とS軸の割り当てを一致させる。

## 変更の全体像

5種の動物の軸割り当てを変更する。3種をG→S（実際に単独行動）、2種をS→G（実際に群れ/集団行動）に修正し、生態事実との整合性を確保する。

**変更前**: G6種:S6種（うち3種のG割り当てが事実と矛盾）
**変更後**: G5種:S7種（全種が生態事実と整合）

5:7の偏りは、日本固有動物には単独行動の種が多いという生態的事実を反映した結果であり、無理に6:6にして事実と矛盾するより誠実な設計である。

## 各変更の生態的根拠

### G→Sに変更する3種

1. **ムササビ**: 単独行動の動物。モモンガとは異なり群れを形成しない（Wikipedia「ムササビ」、愛知県環境調査センター資料）。S軸のみの変更で他軸は維持。

2. **ホンドテン**: 繁殖期以外は基本的に単独行動（Wikipedia「ホンドテン」、野生動物保護管理事務所資料）。S軸のみの変更で他軸は維持。

3. **ニホンリス**: 普段は単独で生活している（動物図鑑）。S-A-H-Iはニホンキツネと重複するため、A軸もC（慎重型）に変更。食料を堅実に貯蓄する行動は「慎重型」に合致する。

### S→Gに変更する2種

4. **エゾシカ**: 群れを形成する動物。オスとメスで別々の群れを作り集団で生活する（札幌市資料）。G-C-L-Iはホンドタヌキと重複するため、A軸もA（行動型）に変更。大地を群れで駆ける行動力は「行動型」に合致。キャッチコピーも「群れで駆ける」に修正。

5. **ニホンモモンガ**: 巣内同居の習性があり、学術研究も存在する（CiNii「ニホンモモンガ成獣による巣内同居の誘発要因の分析」）。冬場に複数個体が巣穴に集まって体温調節する。結果文では「冬には仲間と身を寄せ合う一面も」のように、事実に基づいた描写を行う。S軸のみの変更で他軸は維持。

## 変更理由の要約

レビュアー指摘（19cd11036bb）により、ムササビ・ホンドテン・ニホンリスの3種が実際には単独行動であるにもかかわらずG（群れ型）に割り当てられていた矛盾を解消する。PM判断（19cd110a308）に従い、案A（軸の再設計）を採用。「生態事実に基づく性格描写」という差別化ポイントを維持しつつ、全12種の軸割り当てが生態事実と整合する状態にした。

副次的な変更として、エゾシカとニホンモモンガを生態事実に基づきS→Gに移動し、G:S比を4:8ではなく5:7に改善した。完全な6:6にはできないが、日本固有動物には単独行動の種が多い事実を反映した設計として妥当である。
