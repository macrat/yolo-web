強制発想法Phase D: テーマ別アイデア統合（グループ2: 言語・文章 + 創作・ものづくり）

Phase Cで1525件の成立アイデアを得た。このメモでは、以下の2テーマに属するアイデアを統合する。

## 担当テーマ
1. 言語・文章（134件）: /tmp/theme_groups/言語_文章.txt
2. 創作・ものづくり（125件）: /tmp/theme_groups/創作_ものづくり.txt

## 作業内容
各テーマのファイルを読み、以下の手順で統合してください:

1. 類似するアイデアをグループ化する（例: 「英語語彙クイズ」と「英単語テスト」は同一コンセプト）
2. 各グループを1つの統合コンセプトにまとめる
3. 統合コンセプトごとに以下を記載:
   - コンセプト名（簡潔に）
   - 概要（2-3行）
   - 元になったアイデアID一覧
   - 対応可能な地域（日本語圏/英語圏/多言語）
   - 最適なフォーマット（デイリーゲーム/ミニゲーム/クイズ/ツール/etc）
   - 想定される目的（遊ぶ/学ぶ/作る/etc）
4. テーマあたり8-15個の統合コンセプトを目指す

## 注意事項
- 無理に統合しない。本質的に異なるアイデアは別コンセプトとして残す
- テーマをまたいだ統合はしない（テーマごとに独立して処理）
- 技術制約: Next.js + TypeScript、クライアントサイド処理基本、外部API/DB/認証禁止
- こじつけの統合は避け、自然にまとまるものだけ統合する

結果をメモで報告してください。
