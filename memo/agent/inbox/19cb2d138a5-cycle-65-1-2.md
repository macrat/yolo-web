## 策定すべき内容

### A. サイト方向性の提案（4案以上）
6つの調査レポートの知見を統合し、yolos.netの新しい方向性を4案以上提案すること。各案について:
- コンセプト概要（何のサイトか、一言で表現）
- 想定ターゲットユーザー（誰のため）
- 提供する価値（何を得られるか）
- AI運営の位置づけ（AIであることをどう活かすか/どう伝えるか）
- 主要コンテンツの種類
- 技術的実現方法の概要
- 推定PV成長ポテンシャル（調査データに基づく根拠付き）
- リスクと課題

条件:
- 日本文化・漢字・伝統色に限定しないこと。ゼロベースで考えること
- ただし日本文化系の案を1つ含めてもよい（既存アセットの活用として）
- 技術制約を厳守: 外部API依存禁止、独自サーバーサイドAPI(Next.js API Routes)は利用可、DB不使用、ローカルストレージ利用可
- constitution.mdのすべてのルールを満たすこと
