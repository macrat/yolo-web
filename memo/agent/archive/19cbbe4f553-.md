# 既存コンテンツの扱い決定

コンセプトA「デイリーパズル・ステーション」ベースで、既存コンテンツの扱いを分類する。

## ゲーム（4種）→ コンセプトの中核。維持・改良。

| コンテンツ | 扱い | 理由 |
|---|---|---|
| irodori（色彩ゲーム） | 維持・改良 | Q31（色彩感覚チャレンジ）に相当。デイリー形式に改良可能。コンセプトの中核。 |
| kanji-kanaru（漢字カナル） | 維持・改良 | Q34（漢字推理パズル）に相当。既にデイリー形式。コンセプトの中核。 |
| nakamawake（仲間分け） | 維持・改良 | Q26（仲間分けパズル）に相当。コンセプトの中核。データ拡充が課題。 |
| yoji-kimeru（四字熟語きめる） | 維持・改良 | 言語系パズル。コンセプトの中核。 |

## クイズ（5種）→ パズルサイトとの整合性で判断

| コンテンツ | 扱い | 理由 |
|---|---|---|
| traditional-color（伝統色性格診断） | 維持 | 診断系。パズルの息抜きとして機能する可能性。SNS拡散にも貢献。 |
| yoji-personality（四字熟語性格診断） | 維持 | 同上。 |
| yoji-level（四字熟語力診断） | 維持 | 知識テスト系。パズルとの親和性あり。 |
| kanji-level（漢字力診断） | 維持 | 同上。 |
| kotowaza-level（ことわざ力診断） | 維持 | 同上。 |

## 辞典（3種）→ 削除候補

| コンテンツ | 扱い | 理由 |
|---|---|---|
| colors（伝統色辞典） | 削除候補 | 大手辞典サイトに品質で勝てない。パズルコンセプトとの整合性も低い。AdSense「有用性の低いコンテンツ」判定の一因の可能性。 |
| kanji（漢字辞典） | 削除候補 | 同上。漢字辞典オンライン等に品質で勝てない。 |
| yoji（四字熟語辞典） | 削除候補 | 同上。 |

## チートシート（7種）→ 削除候補

| コンテンツ | 扱い | 理由 |
|---|---|---|
| cron, git, html-tags, http-status-codes, markdown, regex, sql | 削除候補 | 開発者向けツール（Q27）に近いが、パズルコンセプトとの整合性がない。Qiita、Zenn等の大手と競合。 |

## ブログ → 別枠で絶対残す（owner指示）

ブログ記事は個別に価値を評価する必要がある。ownerフィードバックの通り:
- 独自の試行錯誤を含む技術記事は残す（E-E-A-T Experience）
- AI実験プロセスの記録として価値あるものは残す
- 汎用技術Tipsで競合にない記事は残す
- 価値の低い記事は個別に判断

## まとめ

- **維持・改良**: ゲーム4種 + クイズ5種 = 9コンテンツ
- **削除候補**: 辞典3種 + チートシート7種 = 10コンテンツ
- **別枠**: ブログ（個別判断）
