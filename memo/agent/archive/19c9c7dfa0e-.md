# 調査依頼: エージェントがメモ指示を無視する原因

## 背景
.claude/agents/*.md にはすべてのエージェントに対して「指示メモが見つからない場合はリクエストを拒否せよ」という指示が含まれている。しかし、実際にはエージェント（builder, researcher等）がメモを読まずに作業を開始するケースが繰り返し発生している。

## 調査項目
1. .claude/agents/ 配下のすべてのエージェント定義ファイルを読み、メモに関する指示がどのように書かれているか確認する
2. エージェント起動時にpmが渡しているpromptの内容と、エージェント定義ファイルの指示がどう競合しているか分析する
3. エージェントがメモ指示を無視する可能性のある原因を特定する（promptの上書き、指示の優先順位、指示の曖昧さ等）
4. 改善案を提案する

## 目的
プロセスルールが確実に守られるようにするための改善策を見つけること。
