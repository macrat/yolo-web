# 既存コンテンツの扱い決定と最終サイトコンセプト文書作成

## 依頼内容

選択されたコンセプトB「おもしろ占い・診断パーク」に基づき、以下の2つの成果物を作成してください。

### 成果物1: 既存コンテンツの扱い方針

以下の既存コンテンツそれぞれについて、新コンセプトとの関係から「保持」「修正して保持」「削除（410）」のいずれかを判断し、理由を明記してください。

**既存コンテンツ一覧:**

#### ゲーム（/games/）
- irodori: 色の微差を当てるゲーム（デイリーモードあり）
- kanji-kanaru: 漢字の読み・意味を当てるゲーム
- nakamawake: 仲間分けゲーム
- yoji-kimeru: 四字熟語を完成させるゲーム

#### 辞典（/dictionary/）
- colors: 日本伝統色辞典
- kanji: 漢字辞典
- yoji: 四字熟語辞典

#### ツール（/tools/）  
35種以上の開発者向け・テキスト変換ツール（JSON formatter, base64, color-converter, kana-converter, keigo-reference, traditional-color-palette等）

#### チートシート（/cheatsheets/）
cron, git, html-tags, http-status-codes, markdown, regex, sql

#### ブログ（/blog/）
記事群（個別評価が必要。Owner指示: 独自の試行錯誤=E-E-A-Tを満たす記事は保持。一律削除せず1記事ずつ評価）

#### クイズ（/quiz/）
クイズ機能

**判断基準:**
- 新コンセプト（占い・診断パーク）に直接含まれるコンテンツ → 保持
- 新コンセプトとは異なるが、独立して来訪者に価値を提供し、SEOやサイトの多様性に寄与するもの → 保持（ただし、サイト全体の一貫性を損なわないか検討）
- 来訪者への価値が低い、または大手に勝てないもの → 削除（410）
- Owner指示: 辞典は大手に勝てない。チートシートもQiita/Zenn等の大手がある。ただし独自性がある場合は別。
- Owner指示: コンテンツ削除時は410エラーを返す（301リダイレクトは不適切）

### 成果物2: 最終サイトコンセプト文書（site-concept-v3.md改訂案）

以下の構成で最終サイトコンセプト文書を作成してください:

1. **サイトコンセプト**: コンセプトBの内容を基にした一文のコンセプト定義
2. **ターゲットユーザー**: コンセプトBから導かれるペルソナ
3. **提供価値**: 来訪者にとっての価値（Why visit?）
4. **コンテンツ戦略**: 
   - 新規コンテンツ（コンセプトBの8コンテンツ）の概要
   - 既存コンテンツの扱い方針（成果物1の要約）
5. **差別化戦略**: 競合（診断ドットコム、ShindanMaker等）との差別化方針
6. **成長戦略**: PV獲得のためのチャネル（SNSバイラル、SEO等）
7. **品質基準**: 「面白さ」の品質をどう定義し担保するか（constitution.md Rule 4との整合）
8. **リスクと対策**: 評価時に認識されたリスクへの具体的対策
