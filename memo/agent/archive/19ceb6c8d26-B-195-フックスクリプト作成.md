### 新規作成: `.claude/hooks/block-agent-escape.sh`

CLAUDECODE環境変数の改ざんとclaudeコマンドの直接実行をブロックするフックスクリプトを作成してください。

実装要件:
- block-destructive-git.shと同様のパターンでstdinからjqでtool_input.commandを取得
- early return: コマンド文字列にCLAUDECODEもclaudeも含まれない場合はexit 0で即時終了
- heredoc本文は除外する（block-destructive-git.shのremove_heredocs関数を参考）

ブロックすべきパターン:
1. CLAUDECODE環境変数操作:
   - unset CLAUDECODE
   - export CLAUDECODE=
   - CLAUDECODE= （代入）
   - env -u CLAUDECODE
2. スクリプト言語インライン:
   - python3 -c / python -c でCLAUDECODEを含むもの
   - perl -e でCLAUDECODEを含むもの
   - node -e でCLAUDECODEを含むもの
3. claudeコマンド実行:
   - claude コマンド（先頭語またはパイプ・セミコロン後）
   - npmやnpx経由のclaude実行
   - フルパス指定（パス末尾が/claudeのパターン）
   - コマンド置換経由（$(which claude) 等）
   - exec経由（exec claude 等）

ブロック時はexit 2でエラーメッセージを表示。

### settings.json の更新

`.claude/settings.json` のPreToolUse Bashフックのhooks配列に block-agent-escape.sh を追加してください。

以下のテストケースで動作確認してください（スクリプトに直接テストコマンドを渡して確認）:

ブロックされるべき:

- env -u CLAUDECODE command
- claude --agent
