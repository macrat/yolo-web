# 方向性の決定

## 選定: コンセプトA「デイリーパズル・ステーション」をベースとする

### 選定理由

1. **評価スコア最高（4.1）**: 一貫性(5)と成長ポテンシャル(5)が突出
2. **既存コンテンツとの高い親和性**: 既存ゲーム4種（irodori, kanji-kanaru, nakamawake, yoji-kimeru）がデイリーパズル路線にそのまま活かせる
3. **ownerフィードバックとの整合**: 「ゲーム差別化の容易さ」「デイリーでリテンション向上」「少量×高品質」すべてに合致
4. **リテンション最強**: PV最大化というGoalに最も直結

### コンセプトBの要素を部分的に取り込む

コンセプトAの弱点「ターゲットがやや狭い」を補うため、占い・診断を1〜2件追加する案を検討する。ただしパズルの一貫性を損なわない範囲で。

### 注意点

- Q26（仲間分けパズル）: 既存のnakamawakeがあるため活用可能。ただしデータ調達コスト要確認。
- 言語系パズル（Q41）: ownerの「言語の正確性が求められるツール」警告に注意。
- 既存のdictionary（辞典）とcheatsheets: コンセプトとの整合性要検討。

## 既存コンテンツとの照合（暫定）

### ゲーム（4種）
- irodori → Q31（色彩感覚チャレンジ）に相当。維持・改良。
- kanji-kanaru → Q34（漢字推理パズル）に相当。維持・改良。
- nakamawake → Q26（仲間分けパズル）に相当。維持・改良。
- yoji-kimeru → 四字熟語パズル。維持・改良。

### 辞典（3種: colors, kanji, yoji）
- コンセプトAとの直接的な整合性は低い。
- 辞典としての品質が「有用性の低いコンテンツ」判定の一因の可能性。
- 削除候補だがowner判断待ち。

- 削除候補だがowner判断待ち。
