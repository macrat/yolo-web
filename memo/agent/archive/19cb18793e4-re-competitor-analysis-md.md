## 調査1: ことわざ・慣用句の競合サイト

### 主要競合サイト

#### 1. 故事・ことわざ・慣用句辞典オンライン (kotowaza.jitenon.jp)
- **URL**: https://kotowaza.jitenon.jp/
- **収録数**: 7,195語（2026年3月時点）
- **強み**:
  - 五十音順・漢字画数・場面用途・検索ランキングなど多様な検索方法
  - 意味・出典・類語を網羅した詳細な情報提供
  - 継続的なコンテンツ更新
  - 漢字・四字熟語・ことわざのデータセット販売も提供
- **弱み**: インタラクティブ要素やゲーム性がほぼない、視覚的な工夫が少ない

#### 2. ことわざ・慣用句の百科事典 (proverb-encyclopedia.com)
- **URL**: https://proverb-encyclopedia.com/
- **運営者**: 北澤篤史（ことわざ学会理事）
- **強み**:
  - 専門家監修による信頼性
  - 意味からの逆引き検索が可能（「上手な人でも失敗することもある」→ 弘法も筆の誤り）
  - 小学生向け・中学受験・高校入試・大学受験と学習段階別に構成
  - 場面別・感情別の逆引き分類が充実
  - クイズ機能・オリジナルイラスト素材を提供
  - コラム連載あり
- **弱み**: 個人運営のため更新ペースが不安定な可能性

#### 3. goo国語辞書 ことわざカテゴリ (dictionary.goo.ne.jp)
- **URL**: https://dictionary.goo.ne.jp/jn/category/%E6%85%A3%E7%94%A8%E5%8F%A5%E3%83%BB%E3%81%93%E3%81%A8%E3%82%8F%E3%81%96/
- **強み**: NTTレゾナントという大手プラットフォームの信頼性・SEO優位性
- **弱み**: あくまでも国語辞書の一カテゴリで、専門性に欠ける

#### 4. ことわざ・四字熟語辞典 (kotowaza-dic.com)
- **URL**: https://kotowaza-dic.com/
- **強み**: 四字熟語を含む複合カテゴリで展開

#### 5. クイズ・検定系サイト
- **ことわざ・慣用句 四択クイズ** (kanji.kuizu100.net/kotowaza/): シンプルな四択クイズ
- **こと検・ことわざ検定** (kotowaza-kentei.jp/quiz/): 初級・中級・上級の3レベルのクイズ

### 市場の規模感
- 「ことわざ」は日本語学習・教育の定番カテゴリであり、受験シーズン（秋〜冬）に検索数が増加するシーズナルな傾向がある
- 具体的な月間検索ボリューム数値は公開情報からは取得できなかったが、kotowaza.jitenon.jp がSimilarWebに掲載されていることからそれなりのアクセス規模があると推定される
- モバイルアプリ分野でも「ことわざクイズ - はんぷく一般常識」（App Store）などが人気

### 参入余地の分析
- **辞書系は競合が確立されており参入余地は限定的**: 既存サイトがSEOを抑えており、同様の辞書コンテンツでは差別化が難しい
- **インタラクティブ・ゲーム要素での差別化は有望**: 既存競合はほぼ静的な辞書形式。ことわざを題材にしたゲーム（Wordleのことわざ版、穴埋めパズルなど）は競合が薄い
- **ことわざ × SNS拡散（診断形式）** の組み合わせはほぼ空白地帯
- yolos.netのコンセプト（インタラクティブ・エンタメ性）との相性は良い

## 調査2: 診断コンテンツの競合

### 主要競合サービス

#### 1. 診断メーカー / ShindanMaker (shindanmaker.com)
- **URL**: https://shindanmaker.com/
- **特徴**: 誰でも無料で診断を作成・公開できるプラットフォーム
- **規模**: HOT診断はTop200を毎時更新、人気診断は数百万〜数千万回利用のものがある
- **SNS連携**: X（旧Twitter）との高い親和性。ハッシュタグがトレンド入りした実績多数
- **強み**: プラットフォームとして確立しており、UGC（ユーザー生成コンテンツ）の量が圧倒的

#### 2. 16Personalities (16personalities.com)
- **URL**: https://www.16personalities.com/
- **特徴**: MBTIベースの性格診断。世界で10億回以上利用、45言語対応
- **日本での人気**: 非常に高い。「MBTI診断」「16パーソナリティ」として日本でも定着
- **強み**: ブランド力・科学的根拠・結果の詳細さ
- **弱み**: 日本特化ではない、SNS拡散はしにくい

#### 3. ホイミー (hoyme.jp)
- **URL**: https://hoyme.jp/
- **特徴**: 面白い・よく当たると評判の無料診断コンテンツ集
- **強み**: 日本語特化、エンタメ性重視

#### 4. 診断ドットコム (4ndan.com)
- **URL**: https://4ndan.com/
- **特徴**: 性格・好みから診断する各種テスト、アニメキャラに例える診断など
- **強み**: 多様なジャンルの診断、人気ランキング機能あり

#### 5. Makko診断ポータル (ma-kko.com)
- **URL**: https://ma-kko.com/
- **特徴**: 診断作成ツール「Makko」で作られた診断のポータル
- **強み**: オリジナル診断の手軽な作成・公開

### 診断コンテンツがSNSで拡散する理由（ガイアックス調査より）
1. **自己表現欲求**: 「こんな自分がいた」を他者に伝えたい
2. **シェアインセンティブ**: 結果共有でキャンペーン参加、抽選応募など
3. **娯楽性**: 気軽に短時間で楽しめる
- 人気診断は1,000万回近い利用回数に達することも

### 2025年以降のトレンド
- Xのアルゴリズム変化（機械学習による12種類のユーザー行動予測モデル）により、単純な「拡散狙い」は効果が薄れている
- 「素材配布 × 大喜利」のような参加型コンテンツが新たなバズパターンとして台頭

### yolos.netのコンセプトとの相性
- **高い相性**: yolos.netはインタラクティブコンテンツが得意で、診断系はその延長線上にある
- **差別化ポイント**: 漢字・四字熟語・ことわざなど日本語知識系の診断（「あなたの語彙力は？」「あなたに合う四字熟語は？」など）はほぼ空白地帯
- **注意点**: 診断メーカーという強力なプラットフォームがあるため、「場所」より「コンテンツの質とオリジナリティ」での差別化が必要

## 調査3: 検索ボリュームデータ

注意: 本調査では公開されている無料ツールや信頼できる記事からの間接的な情報をもとにしており、正確なAPI取得値ではありません。

### 漢字辞典・四字熟語関連

| キーワード | 推定規模 | 備考 |
|-----------|---------|------|
| 漢字辞典 | 大（月間数万〜数十万規模） | goo辞書・漢字ペディアなど大手が上位を占める |
| 四字熟語 | 大（月間数万規模） | 辞典サイト群が多数存在、競合激しい |
| 四字熟語 一覧 | 中〜大 | 学習用途で安定的な需要 |
| 四字熟語 意味 | 中〜大 | 長尾キーワードで比較的狙いやすい |
| ことわざ | 大 | 受験シーズンにスパイク傾向 |
| ことわざ 意味 | 中〜大 | 個別ことわざのLong-tailキーワードが豊富 |

競合状況: 「四字熟語辞典オンライン」（yoji.jitenon.jp）が6,642語収録と圧倒的なコンテンツ量。「四字熟語の百科事典」（idiom-encyclopedia.com）も競合として存在。漢検（kanken.or.jp）も上位表示。

### Webツール関連（JSON整形・QRコード生成等）

| キーワード | 推定規模 | 備考 |
|-----------|---------|------|
| JSON 整形 | 中〜大 | 開発者が日常的に使用する実用ツール |
| JSON フォーマット | 中 | 上記の類義語 |
| QRコード 作成 | 大 | 非エンジニアも含む一般ユーザーの需要が高い |
| QRコード 生成 | 大 | 「作成」とほぼ同規模 |
| URL エンコード | 中 | エンジニア向け、安定した需要 |

競合状況: QRコード作成ではAdobe Express・QRのススメ（20年以上の老舗）・Chrome拡張（週70万人以上のアクティブユーザー）が強い。JSON整形はChromeデベロッパーツールやVS Codeに機能が内包されつつも、手軽なWebツールへの需要は継続。

### チートシート関連

| キーワード | 推定規模 | 備考 |
|-----------|---------|------|
| git コマンド | 中〜大 | Qiita記事が上位を占める |
| git チートシート | 中 | 学習者・入門者向けに安定需要 |
| 正規表現 | 大 | 非常に広いキーワード |
| 正規表現 チートシート | 中 | 実用性重視のエンジニアに需要 |
| 正規表現 一覧 | 中 | 「チートシート」と類似 |
| Markdown 記法 | 中 | エンジニア・ライターに定常需要 |

競合状況: Qiita・GitHub Docs・Zenn・個人ブログ（yunabe.jp等）が多数存在。検索上位はQiitaのいいね数が多い記事が占めることが多い。動的・インタラクティブなチートシート（入力してその場で試せる）は差別化余地あり。

### 単語ゲーム関連（Wordle系）

| キーワード | 推定規模 | 備考 |
|-----------|---------|------|
| Wordle | 大（ピーク時）→現在は中 | 2022年のバズから落ち着いている |
| ワードル | 中 | 日本語版の需要 |
| 日本語 Wordle | 中 | 複数の非公式日本語版が存在 |
| 言葉遊びゲーム | 中 | より広いカテゴリ |
| タイピングゲーム | 中〜大 | Wordleより安定した長期需要 |

競合状況: 公式Wordleには日本語版がなく、非公式日本語版（WORDLE ja、wordle.mottox2.com等）が複数存在。SNSシェア率で人気度を測る傾向がある。2022年の爆発的人気から落ち着きつつあるが、日本語ゲームとしての安定需要はある。

### 総括
- **即効性が高いカテゴリ**: Webツール（QRコード・JSON整形など）は実用的で反復利用率が高く、安定したアクセスを見込める
- **学習需要が安定**: 漢字・四字熟語・ことわざは受験シーズン問わず一定の需要
- **差別化余地が最大**: インタラクティブ型チートシート（入力すると結果が即表示されるもの）は競合が薄い
- **エンタメ × 知識**: ことわざ・四字熟語 × ゲーム・診断の掛け合わせは未開拓領域
