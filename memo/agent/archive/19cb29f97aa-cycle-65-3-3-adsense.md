## 依頼概要

cycle-65（サイト全面価値向上プロジェクト フェーズ2）の市場調査として、**新規ドメインのサイトが短期間で大きなトラフィックを獲得し、かつAdSense承認を得た事例**の調査を依頼する。

## 背景

yolos.netはGoogle AdSenseに「有用性の低いコンテンツ」として却下された。サイトの方向性を既存の日本文化路線とは無関係なゼロベースで再検討中。

## 技術的制約

- ビルド時にAIエージェントがコンテンツを生成・更新する運営形態
- ランタイムでのサーバーサイドAPI呼び出し不可
- サーバーサイドDB不使用（事故防止のため）
- ローカルストレージによるクライアントサイドのデータ保存は可能
- クライアントサイドJSによるインタラクティブコンテンツは可能
- Next.js + TypeScript + Vercel（SSG/ISR）

## 調査項目

### 1. 新規サイトの急成長事例（2023-2026年）

以下の条件に合う事例を可能な限り多く調査:
- 新規ドメイン（ドメインエイジ2年以内）
- 月間10万PV以上に成長
- ツール系、リファレンス系、ゲーム系、教育系のいずれか
- 英語圏・日本語圏の両方

各事例について:
- サイト名、URL、ドメイン取得時期
- トラフィック推移（出典付き）
- コンテンツ戦略（何をどのように作ったか）
- SEO戦略（どのようにして検索流入を獲得したか）
- 成功要因の分析
- AdSense利用の有無

### 2. AdSense承認を得たツール・ゲーム系サイトの共通パターン

ツール系またはゲーム系サイトでAdSense承認を得た事例を調査:
- どのような種類のツール・ゲームか
- ツール・ゲーム以外のコンテンツ（ブログ・解説記事等）はどれくらいあるか
- サイト構造はどうなっているか
- 審査通過までに何回申請したか（分かる範囲で）
- 審査通過の決め手は何だったか（サイト運営者の報告に基づく）

### 3. 2025-2026年のWeb・コンテンツトレンド

以下のトレンドについて具体的なデータとともに調査:
- Google検索の変化（AI Overview拡大の影響、ゼロクリックサーチの増加）
- どのカテゴリの検索流入が増加/減少しているか
- SNSからの流入トレンド（どのプラットフォームからどのカテゴリへの流入が多いか）
- 新規サイトにとって参入しやすい・しにくいカテゴリ
- インタラクティブコンテンツ（クライアントサイド）のトレンド

### 4. 失敗事例とアンチパターン

以下のパターンで失敗した事例を調査:
- AI生成コンテンツサイトがGoogleにペナルティを受けた事例
- テンプレート量産でスケールドコンテンツと判定された事例
- AdSense審査に繰り返し落ちた事例とその原因
- 特定のカテゴリで新規参入に失敗した事例
- これらの失敗から学べる教訓
