cycle-65 タスク2（サイトコンセプトの再定義）のための調査依頼。

## 調査目的

site-value-improvement-plan.md のフェーズ2-2「サイトコンセプトの再定義」を実施するための情報収集。
サイトの存在意義と独自の価値提案を策定するために、AIが運営するウェブサイトの成功事例、サイトコンセプト設計のベストプラクティス、AI運営の効果的な位置づけ方を調査する。

## 調査内容

1. **AIが運営・生成するウェブサイトの成功事例調査**
   - AI生成コンテンツサイトで高い評価やトラフィックを得ている事例の調査
   - AI運営であることをどのように開示・位置づけしているかの調査
   - Google検索で高い評価を得ているAI活用サイトの特徴分析

2. **サイトコンセプト設計のベストプラクティス**
   - Value Proposition（独自の価値提案）の効果的な策定方法
   - 複数カテゴリのコンテンツを統合するサイトコンセプトの成功例
   - ユーザーに「なぜこのサイトを使うべきか」を明確に伝えるためのフレームワーク

3. **AI運営の位置づけに関する調査**
   - AI運営であることの開示がユーザー信頼性に与える影響の研究・事例
   - constitution.md Rule 3（AI運営・不正確性の通知義務）を遵守しつつ、信頼性への悪影響を最小化する表現方法の調査
   - 「AI運営」を差別化要因として活用している成功例

4. **サイト方向性の選択肢の調査**（ownerメモ 19cb257b560 の視点を踏まえて）
   - 日本文化 + AI という方向性（root-cause-analysis.md の推奨方針B）の妥当性を検証するためのデータ収集
   - それ以外の方向性（例: 特定分野に特化した高品質ツールサイト、AIの実験記録サイト、など）の可能性と市場性の調査
   - AIが最も価値を発揮できるコンテンツ領域の調査
