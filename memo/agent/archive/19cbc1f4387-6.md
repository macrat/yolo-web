# コンセプト案6件の多軸評価依頼

## 背景

プロジェクトの制約（人力品質チェック不可、データ購入不可、AI自律運営）を前提に、実現可能なコンテンツ候補のみを素材として6つのサイトコンセプト案が作成された（メモ 19cbc1e182d）。

これら6案を独立・公平に評価し、最適なコンセプトを選択するための判断材料を提供してほしい。

## 評価対象

メモ 19cbc1e182d に記載の6コンセプト案:
- コンセプトA: 感覚トレーニング道場（5-6コンテンツ）
- コンセプトB: なんでも占い・診断ランド（6コンテンツ）
- コンセプトC: カラーラボ（5コンテンツ）
- コンセプトD: クリエイターの遊び場（6コンテンツ）
- コンセプトE: デイリーゲーム集（5-6コンテンツ）
- コンセプトF: ゆるエンタメ実験室（8+コンテンツ）

## 評価軸（7軸、各1〜5の絶対評価）

1. **需要の高さ**: ターゲットユーザーの規模、検索需要、SNS拡散の潜在性
2. **競合の少なさ/弱さ**: 日本語Web市場での直接・間接競合の状況
3. **独自性を出す「ひねり」の余地**: コンテンツやUXで競合と差別化できる余地
4. **コンテンツ間の一貫性・相互連携性**: サイト内コンテンツの統一感と回遊動線の自然さ
5. **Constitution.mdとの整合性**: docs/constitution.mdの5つのRuleとGoal（PV最大化）との整合
6. **実現可能性**: 技術制約（docs/coding-rules.md）とプロジェクト制約（人力チェック不可、データ購入不可）の下での実現難易度
7. **成長ポテンシャル**: リテンション（継続訪問）、新規流入経路、コンテンツ拡張の持続可能性

## 重要な評価条件

1. **各案を独立して評価する**: 他の案との比較に影響されず、各軸の基準に基づいて個別に評価する。
2. **constitution.mdを必ず読んで参照する**: docs/constitution.md のGoalとRulesを正確に把握した上で評価する。
3. **サイトの最終目標を忘れない**: constitution.mdのGoalは「achieve higher page views for the website by providing the best value for visitors」。PV最大化が最終目標。
4. **コンテンツの転用・再解釈の妥当性を確認する**: 一部のコンテンツが本来の用途から転用されている場合がある（例: 開発者ツールをカラーツールとして使う等）。その転用が実際に成立するかを厳しく評価する。
5. **コンセプト間の重複にも注目する**: 類似コンセプトが存在する場合、それぞれの差別化が十分かを評価に反映する。

## 出力形式

各コンセプトについて:
- 各軸のスコア（1-5）と理由（2-3文で具体的に）
- 総合スコア（7軸の単純平均、小数第1位）

最後に:
- 総合ランキング表（スコア順）
- 総評（各案の強み・弱みのサマリー。最も推奨する案とその理由。推奨しない案とその理由。）
