## 2. 問題1（重大）: 相性マトリクスにsado-shodo組が欠落

計画では「28組（7*(7+1)/2）」と記載しているが、実際に記載されている異タイプ相性は20組であり、**sado-shodo（茶道-書道）** の相性記述が存在しない。合計は同タイプ7組 + 異タイプ20組 = 27組にとどまる。

必要な異タイプ21組を検証した結果:
- budo-haiku, budo-kado, budo-rakugo, budo-sado, budo-shodo, budo-togei: 6組 OK
- haiku-kado, haiku-rakugo, haiku-sado, haiku-shodo, haiku-togei: 5組 OK
- kado-rakugo, kado-sado, kado-shodo, kado-togei: 4組 OK
- rakugo-sado, rakugo-shodo, rakugo-togei: 3組 OK
- **sado-shodo: 欠落**
- sado-togei: 1組 OK
- shodo-togei: 1組 OK

**修正必要**: sado-shodo の相性テキスト（label + description）を追加すること。茶道と書道は日本文化として親和性が高く、「静かな空間での集中」「所作への美意識」等の共通点から自然な相性テキストが書ける組み合わせ。

## 3. 問題2（中）: QuizMetaの必須フィールドが未記載

types.tsのQuizMetaインターフェースに対し、以下のフィールドが計画に明示されていない:

- **title**: 推測可能（「日本文化適性診断」）だが明示すべき
- **description**: SEO用の長い説明文。未記載。
- **shortDescription**: 一覧表示用の短い説明文。未記載。
- **type**: "personality"。推測可能だが明示すべき。
- **questionCount**: 18。推測可能だが明示すべき。
- **icon**: クイズ一覧に表示されるアイコン絵文字。未記載。
- **accentColor**: テーマカラーhex。未記載。
- **keywords**: SEO用キーワード配列。未記載。
- **publishedAt**: ISO 8601形式の公開日時。未記載。

「builderはこのメモのみを参照して実装すること」と記載されている以上、これらは全て明示的に記載されるべき。animal-personality.tsを参考にすれば推測は可能だが、特にdescription、shortDescription、icon、accentColor、keywordsはbuilderの判断に委ねるべきでない。

**修正必要**: 上記フィールドを全て計画に追記すること。

## 4. Q17cの自然さ: 許容範囲（軽微な懸念）

修正後の「言葉ではなく、気持ちを込めた差し入れやちょっとした手作りのものを届ける」は、前回指摘のあった「手紙を書く」よりは改善されている。ただし、Q17の設問「困っている人を見かけたとき」は即時的な対応を想起させるのに対し、この選択肢は「差し入れを届ける」という事後的な行動を描写しており、他の3選択肢（さりげなく手助け / 声をかけて元気づける / 問題解決を手伝う）と比べてやや時間軸がずれている。

ただし、「華道/俳句タイプの人は言葉より行動で気持ちを伝える」という性格特性を表現する選択肢としては意味が通る。修正は必須ではないが、もし修正するなら「言葉にはしないが、さりげなく相手が必要としているものを用意する」のような即時的な表現にすると4選択肢の時間軸が揃う。

## 5. ターゲットユーザー視点の評価: 良好

ターゲット（手軽で面白い占い・診断を楽しみたい人）の観点から:

**良い点:**
- 7タイプは推奨範囲（6-8）内で、「外れ感」が少ない
- 日本文化の知識不要で日常行動ベースの質問 → 前提知識なしで楽しめる
- タイプ説明文は300-500文字の範囲で読み応えがある
- 各タイプ説明に歴史的エピソード（千利休、嘉納治五郎、松尾芭蕉等）を含め、「なるほど」感がある
- 相性診断でリピートやシェアの動機付けがある
- 相性テキストがユーモアを含み、友達との会話ネタになる

**懸念点:**
- 18問はcontent-categories.mdの推奨（8-12問）を大幅に超える。7タイプの弁別には必要な数だが、ターゲットが求める「1回2-5分で手軽に遊べる」からは外れる可能性がある。ただし、各質問が短く4択なので実際の所要時間は3-4分程度と推測される。
- Q4にbudo:2が2回あることで、budo以外のタイプになりにくい設計意図がやや透ける。ただし1問のみなので影響は軽微。

**総合**: ターゲットユーザーにとって十分楽しめるコンテンツになっている。
