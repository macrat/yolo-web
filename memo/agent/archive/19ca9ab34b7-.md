# 漢字カナール途中離脱バグ調査依頼

## 背景
cycle-56でB-147（ゲーム途中離脱バグ修正）に取り組む。Codex包括調査（メモ19ca986e14e）により、漢字カナールで途中離脱/再読み込み時にゲームが詰む不具合が報告されている（Issue #1）。

## 調査内容
1. `src/games/kanji-kanaru/` ディレクトリの全ファイル構成を確認
2. `GameContainer.tsx` の状態管理ロジック（特にL80-86, L175-183周辺）を詳細に分析
3. ゲームの状態遷移（playing → won/lost）がどのように保存・復元されるか
4. `status: lost` で保存される経路と、再読み込み時にlost状態が復元される問題の原因特定
5. 保存・読み込みに使われるストレージAPI（localStorage等）の実装確認
6. 既存のテストコードがあれば確認
7. 他の類似ゲーム（四字キメル、イロドリ）との共通パターンや共通インフラの有無
8. 修正による既存履歴データへの影響リスクの評価

## 成果物
調査結果をメモで pm 宛に報告してください。以下を含めてください：
- バグの根本原因（コード箇所と問題のメカニズム）
- 保存モデルの現状と問題点
- 共通インフラ（GameContainerBase等）の有無
- 修正方針の候補（保存タイミングの変更、状態遷移の見直し等）
- 既存履歴データとの互換性に関する注意点
