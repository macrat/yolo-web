# PM判断基準書

R1〜R3のレビューサイクルで意見の相違が発生した原因を分析し、以下の判断基準を定めます。
R4以降のレビューでは、この基準に従って判定してください。

## 1. origin（出典区分）の判断基準

### 定義
- **漢籍**: 中国の古典文献（四書五経、史記、荘子、老子、韓非子、孫子など）に直接出典がある
- **仏典**: 仏教経典（般若心経、法華経、涅槃経など）に直接出典がある
- **故事**: 中国の故事・逸話から生まれた（ただし漢籍・仏典に文献が特定できる場合はそちらを優先）
- **日本語由来**: 日本で作られた、または日本の文献・文化から生まれた
- **その他**: 上記いずれにも該当しない（西洋由来の翻訳語など）
- **不明**: 出典が特定できない

### 判断ルール
1. **文献特定優先**: 具体的な文献名が特定できる場合、「漢籍」「仏典」を優先する。「故事」は文献が特定できないが逸話由来であることが明確な場合に使用。
2. **漢語であることだけでは漢籍にしない**: 漢字で構成されているだけでは「漢籍」としない。日本で作られた漢語は「日本語由来」とする。
3. **複数出典**: 漢籍にも仏典にも出典がある場合、より古い・より直接的な出典を採用する。
4. **迷った場合**: 「不明」ではなく、最も蓋然性の高い選択肢を選ぶ。「不明」は本当に手がかりがない場合のみ使用する。

### R1-R3で揺れた例と確定判断
- **猪突猛進**: R1で「漢籍→日本語由来」、R2で「→漢籍」、R3で「→不明」と揺れた。**確定: 日本語由来**。理由: 「猪突」は日本独自の用法（中国では猪=ブタ）。
- **安心立命**: 読みが揺れた（あんしんりつめい vs あんじんりゅうめい）。origin自体は「漢籍」で安定。

## 2. structure（構造パターン）の判断基準

### 定義
- **並列**: 2つの二字熟語が対等に並ぶ（A+B型）。例: 花鳥風月（花鳥+風月）
- **修飾**: 前の要素が後の要素を修飾する（A→B型）。例: 大器晩成（大器が晩成する→晩く成る大器）
- **主述**: 主語+述語の文構造（SV/SVO型）。例: 万物流転（万物が流転する）
- **類義**: 同じ意味の要素を重ねて強調する。例: 温厚篤実（温厚≒篤実）
- **対義**: 反対の意味の要素を組み合わせる。例: 古今東西（古⇔今、東⇔西）
- **因果**: 原因と結果の関係。例: 因果応報（因果→応報）

### 判断ルール
1. **主述 vs 修飾の区別**: 「Aが/はBする」と自然に読める場合は主述。「AのB」「AなB」と読む方が自然な場合は修飾。
2. **主述 vs 因果の区別**: 結果を述べる構造なら主述、原因→結果の関係を表すなら因果。
3. **並列 vs 類義の区別**: 2つの要素が異なる概念なら並列、ほぼ同じ意味なら類義。
4. **迷った場合**: 四字熟語を文に展開して、最も自然な読みに基づいて判断する。

## 3. category（意味カテゴリ）の判断基準

### カテゴリ定義（確定版）
- **life**: 人生の在り方、生き方全般、人生の哲学
- **effort**: 努力、忍耐、精進、頑張ること
- **nature**: 自然現象、風景、自然の美しさ
- **emotion**: 感情、心理状態、気持ちの動き
- **society**: 社会関係、人間関係、集団の中での振る舞い
- **knowledge**: 知識、学問、知恵、思考
- **conflict**: 対立、争い、競争、戦い
- **change**: 変化、転換、推移、移り変わり
- **virtue**: 道徳、美徳、徳のある行い
- **negative**: 否定的な行為・状態への戒め、悪い行い

### 判断ルール
1. **主たる意味で判断**: 四字熟語が最も強く伝えるメッセージに基づいて1つだけ選ぶ。
2. **用途コンテキストを考慮**: このカテゴリは(a)ゲームのヒント（5回失敗後に表示）、(b)辞書のブラウジング/グルーピングの両方に使われる。ユーザーが「このカテゴリなら、この四字熟語がありそう」と推測できる分類が望ましい。
3. **emotion vs negative**: 否定的な感情でも、感情そのものを表す場合はemotion。行為への戒めや否定的な状況の描写はnegative。
4. **effort vs virtue**: 努力することそのものを表す場合はeffort。人格的な美徳を表す場合はvirtue。
5. **life vs 他カテゴリ**: lifeは他のどのカテゴリにも当てはまらない「人生全般」に使う。より具体的なカテゴリに当てはまる場合はそちらを優先。

### 現在のカテゴリ設計の評価
現在の10カテゴリ・単一ラベル設計は以下の理由で妥当と判断する:
- 分布が30〜61で比較的均等（極端な偏りがない）
- ゲームヒントとして適度な粒度（10種類なら「当たれば絞り込める」程度）
- 辞書ブラウジングとして適度なグループ数（多すぎず少なすぎず）
- 多ラベルはゲームヒント表示時に複雑化するため不採用

## 4. meaning（意味）の判断基準

1. **簡潔かつ正確**: 1〜2文で四字熟語の核心的な意味を伝える。
2. **辞書的な説明**: 比喩的な使い方がある場合、元の意味と比喩的な意味の両方を含める。
3. **既存の辞書と矛盾しない**: sourceUrlの辞書と大きく異なる場合は辞書を優先。

## 5. reading（読み）の判断基準

1. **最も一般的な読み**: 複数の読みがある場合、最も広く使われている読みを採用。
2. **全てひらがな**: カタカナや漢字を含めない。
3. **sourceUrlの辞書と一致**: 迷った場合はsourceUrlの辞書の読みを採用。

## 6. レビュアーへの指示

- 上記基準に照らして判断し、基準に反する値がある場合のみ指摘してください。
- 「こちらの方が良い」という主観的な好みによる変更提案は行わないでください。
- 指摘する際は、上記基準のどの項目に基づいているかを明記してください。
