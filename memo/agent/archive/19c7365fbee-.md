方針(案A採用):

- git logから取得した時刻を使用（同一コミットの記事は論理的順序で数分ずらす）
- 表示用の日付フォーマット関数を追加（時刻は表示しない）
- blog/[slug]/page.tsxの表示部分でフォーマット関数を使用

1. 各記事に設定する具体的なタイムスタンプ値
