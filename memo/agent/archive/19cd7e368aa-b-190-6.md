# タスク6: 検索インデックス・サイトマップの更新

## 目的
categoryフィールド削除に伴い、検索インデックスとサイトマップを更新。

## 前提
タスク2が完了済み（新しい関数が利用可能）。

## 変更ファイル
- src/lib/search/build-index.ts
- src/app/sitemap.ts

## build-index.ts の変更
変更前:
```ts
category: String(kanji.category),
```
変更後:
```ts
category: String(kanji.grade),
```
（検索結果でのカテゴリ表示を学年に変更）

## sitemap.ts の変更
- `getKanjiCategories()` の呼び出しを `getKanjiGrades()` に変更
- URLを `/dictionary/kanji/category/${cat}` → `/dictionary/kanji/grade/${grade}` に変更
- 必要に応じて部首・画数ページもサイトマップに追加

imports:
- `getKanjiCategories` → `getKanjiGrades` に変更
- 必要に応じて `getKanjiRadicals`, `getKanjiStrokeCounts` を追加

## 完了条件
- 検索インデックスにcategoryフィールドがない（gradeに置き換え）
- サイトマップに旧categoryURLがない
- 新しい分類ページ（grade/radical/stroke）のURLがサイトマップに含まれている
