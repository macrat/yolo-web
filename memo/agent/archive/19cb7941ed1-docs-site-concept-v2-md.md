最高優先（Phase 1）:
- 日本都道府県チャレンジ（Worldle型）: 47問の少量作り込み、地図シルエット+方角ヒント
- 既存4ゲーム（漢字カナール・四字キメル・ナカマワケ・イロドリ）の強化: 解説ページ追加

高優先（Phase 2）:
- 日本文化テーマ性格診断: SNS拡散力高、少量作り込みで実現可能
- 伝統色連動色彩チャレンジ: 既存イロドリとの連携

中優先（Phase 3）:
- 英語UI対応: 外国人ターゲット訴求強化

低優先（将来検討・条件付き）:
- 歴史人物チャレンジ: 著名50人限定・アーカイブ型（前提: ファクトチェック体制確立）
- サブカルチャレンジ: テキストベース限定（前提: J-PlatPat商標確認フロー策定、商標登録済み名称は不使用）

除外:
- 音楽チャレンジ（楽曲著作権問題が根本的に解決不能）
- 画像使用のサブカルゲーム（著作権侵害リスク）
- 大規模辞典拡充（大量×高品質問題）
