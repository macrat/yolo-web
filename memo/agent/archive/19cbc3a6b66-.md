# 既存辞典・チートシート・クイズの評価

## 依頼内容
以下の既存コンテンツを調査し、新コンセプトとの関係から「保持」「修正して保持」「削除（410）」を判断してください。

## 対象コンテンツ

### 辞典（src/app/dictionary/配下）
1. colors: 日本伝統色辞典
2. kanji: 漢字辞典
3. yoji: 四字熟語辞典

### チートシート（src/app/cheatsheets/配下）
1. cron
2. git
3. html-tags
4. http-status-codes
5. markdown
6. regex
7. sql

### クイズ（src/app/quiz/配下）
- クイズ機能

## 各コンテンツについて調査すること
- 該当ディレクトリのファイルを読んでコンテンツの概要・品質を把握する
- 独自性があるか（日本語Web上で同種のサービスがあるか）
- 来訪者にとっての価値
- 新コンセプト「おもしろ占い・診断パーク」との整合性

## 判断基準
- Owner指示: 辞典は大手に勝てない（漢字辞典オンライン等）
- Owner指示: チートシートもQiita/Zenn等の大手がある。ただし独自性がある場合は別。
- Owner指示: コンテンツ削除時は410エラーを返す
- 新コンセプトとは異なるが、独立して来訪者に十分な価値を提供するもの → 保持
- 大手サイトに品質・量で勝てないもの → 削除

## 出力形式
各コンテンツについて:
- コンテンツ名
- 概要
- 独自性と競合状況
- 来訪者にとっての価値
- 判定（保持/修正して保持/削除）と理由
