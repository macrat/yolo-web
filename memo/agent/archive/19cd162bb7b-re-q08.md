# Q08動物性格診断 タイトル マーケット調査レポート

## 1. ターゲットユーザーの特性分析

docs/targets/手軽で面白い占い・診断を楽しみたい人.yaml より確認した特性：

- **行動動機**: 「自分の意外な一面を発見すること」「友達との会話のネタ」「1回2〜5分の手軽さ」
- **シェア欲求**: 診断結果を友達と比較して盛り上がること、SNSシェアしやすい仕組みを求める
- **嫌うこと**: ありきたりで面白みのない診断結果、こじつけで納得できない判定
- **検索インテント**: "動物診断 無料"、"性格診断 面白い"、"おもしろ 診断" など

**診断コンテンツを選ぶ際に惹かれる要素（重要度順）:**
1. 「自分のこと」を知れる感（自己開示欲求の充足）
2. 友達に見せたくなる/シェアしたくなる意外性
3. ユーモアと独自性（ありきたりでないこと）
4. 手軽さ（タイトルを見た瞬間に「面白そう」と直感できること）

このユーザーにとってタイトルは「面白そうかどうかを3秒で判断する材料」。タイトルで独自性・意外性・「自分に関係ある感」が伝わらないとクリックされない。

## 2. 競合の診断コンテンツタイトル調査

### 日本語動物診断の主なタイトルパターン（診断メーカー・競合サイトより）

| タイトル例 | パターン |
|---|---|
| あなたをどうぶつに例えると！ | 「あなたを〇〇に例えると」 |
| あなたを動物に例えたら | 「あなたを〇〇に例えたら」 |
| 動物タイプ診断：あなたの性格は何系アニマル？ | 疑問形 + 「あなた」 |
| 動物キャラ占い｜全60タイプ診断 | タイプ数の具体的提示 |
| 恋愛どうぶつ診断 | シンプル + テーマ（恋愛） |
| あなたは動物系でいうと何系か（20.7万アクセス） | 疑問形 + 「あなた」 |
| 動物人相診断占い！16タイプの顔タイプ | 診断方法の明示 |
| 【動物タイプ診断】動物タイプ別の性格や特徴 | 括弧でジャンル明示 |

### SNSでバズっている診断コンテンツの特徴

調査から判明したバズりやすいタイトルの共通パターン：

1. **「あなた」を主語に置く**: ユーザーが「自分のこと」と瞬時に認識できる
2. **疑問形の活用**: ツァイガルニク効果で「答えを知りたい」心理を刺激。疑問形タイトルはそうでないものよりクリック率200%高いという調査結果もある
3. **タイプ数・種類数の明示**: 「全60タイプ」「16タイプ」など、多様性・詳細さを予告しクリック意欲を高める
4. **「例えると」「に例えたら」形式**: 比喩的表現で「面白い発見がありそう」という期待感を生む
5. **短くシンプル**: 長いタイトルよりも短く直感的なものがSNS拡散しやすい

## 3. 診断コンテンツのタイトル マーケティングベストプラクティス

### CTR（クリック率）を高めるタイトルの書き方

- **自己関連性の強調**: 「あなた」「あなたの」を冒頭に置くと自分ごと化しやすい
- **希少性・独自性の予告**: 「上位10%」「日本だけ」「珍しい」など、唯一性が感じられる表現
- **結果の想像を促す**: 「〜がわかる」「〜を暴く」「〜が丸わかり」などの表現
- **疑問形 vs 断定形**: 疑問形がCTR向上に有効（「〜は？」「〜なの？」）

### SEOの観点

- 検索インテントの "動物診断 無料"、"性格診断 面白い" に対応するキーワードを含める
- タイトルは30文字前後が理想（検索結果での表示切れを避ける）
- 「日本」「動物」「性格診断」「無料」などの主要キーワードを自然に含める

### SNSシェア時の設計

- OGPタイトルとして表示されるため、短くても情報密度の高い表現が重要
- 「これやってみた」と投稿したくなる、ユーモアや意外性のある切り口を予告するタイトルが有効

## 4. 「日本にしかいない動物」の特徴をどう伝えるか

### 「固有種」という言葉の問題

「固有種」は生物学・環境教育の専門用語で、ライトユーザーには馴染みが薄い。ターゲットユーザー（占い・診断を手軽に楽しみたい層）にとって、難しい専門用語はハードルになる可能性が高い。

### ライトユーザーでも直感的に理解できる代替表現

| 表現 | 特徴 |
|---|---|
| 日本にしかいない動物 | 最も平易で直感的。「図鑑」などのコンテンツでも使われる一般的な表現 |
| 日本だけの動物 | シンプルで短い |
| 日本生まれの動物 | 親しみやすさあり。ただし「日本で生まれた」という意味にも取れる |
| 日本の森にいる動物 | 情景が浮かぶが、固有性は弱い |
| 純日本産の動物 | ユニークだがやや仰々しい |
| ニッポンの動物 | カジュアルで親しみやすい |

「日本にしかいない」が最も平易かつ明確。PHP研究所の子供向け図鑑タイトルにも使われており（「日本にしかいない生き物図鑑」）、幅広い年代に通じる表現であることが確認できた。

### 既存コンテンツで「日本固有」を訴求している事例

- 「日本にしかいない生き物図鑑」（PHP研究所）：タイトルに直接「日本にしかいない」を使用
- テレビ番組「実はその生物、日本にしかいないんです」（テレしずWasabee）：番組タイトルでも「日本にしかいない」を採用
- これらのメディアがライト層向けに「固有種」ではなく「日本にしかいない」を選んでいることは、この表現の有効性を裏付ける

## 5. タイトル候補 3〜5案

以下の3つの軸でタイトル候補を提示する：
- **軸A**: 独自性（日本にしかいない）を前面に出す
- **軸B**: ユーザーの自己関連性（あなた）を前面に出す
- **軸C**: 両者を組み合わせる

### 案1（現状維持）: 「日本の動物性格診断」
- 評価: 「日本の動物」という表現では固有性が伝わらない（日本に生息するすべての動物と混同される）。独自性の訴求が弱く、他の動物診断との差別化が難しい。ターゲットユーザーのクリック動機には直接働きかけていない。
- 推奨度: 低

### 案2（独自性強調）: 「日本にしかいない動物で性格診断」
- 評価: 「日本にしかいない」という唯一性が明確。「なんで？」という好奇心を刺激する。30文字以内でSEO・SNS表示に適している。
- 推奨度: 高

### 案3（自己関連性強調）: 「あなたの性格は何の動物？〜日本にしかいない動物で診断〜」
- 評価: 「あなた」「何の動物？」で疑問形と自己関連性を確保。独自性も副題で補足。ただし全体がやや長い（28文字）。
- 推奨度: 中〜高（文字数調整が必要）

### 案4（ユーモア・意外性強調）: 「あなたの性格、日本の秘密の動物に例えると？」
- 評価: 「秘密の動物」という表現が好奇心を強く刺激する。「あなた」で自己関連性を確保。「秘密」がやや大げさという見方もあるが、ライト層のクリック動機には合致する。
- 推奨度: 中

### 案5（最もシンプル・キャッチー）: 「あなたは日本にしかいない動物で言えば？」
- 評価: 「あなた」で自己関連性、「日本にしかいない」で独自性、疑問形でクリック動機を全て網羅。最もバランスが良い。診断メーカーで人気のある「あなたを〇〇に例えたら」パターンに沿っており、SNSシェア時にも映える。
- 推奨度: 最高

---

## 6. 総合推奨

**推奨案: 案5「あなたは日本にしかいない動物で言えば？」または案2「日本にしかいない動物で性格診断」**

理由：
1. ターゲットユーザーの「自分のことを知りたい」動機に直接訴える（「あなたは」）
2. 競合の動物診断との明確な差別化（「日本にしかいない」＝他にないコンテンツという予告）
3. 「固有種」という専門用語を使わず、ライトユーザーでも瞬時に理解できる
4. 疑問形がクリック心理を刺激する
5. SEOキーワード（動物診断、日本、性格）を自然に含む
6. SNSシェア時のOGPタイトルとしても映える長さ・表現

案2は簡潔さとSEO適性、案5はエンゲージメントとSNS拡散性に優れる。コンテンツの目標（SNSシェア重視 vs 検索流入重視）によって最終選択を判断することを推奨する。
