### 2. description / shortDescription の変更指示が曖昧（重要度: 低）

計画では「meta.description / meta.shortDescription: 「占い」が含まれていれば「診断」に変更」とあるが、実際のテキストは以下の通り。

これらは名詞「占い」の単純置換ではなく、文脈に応じた書き換えが必要になる（例: 「占います」→「診断します」、「占う」→「診断する」等）。builder向けの指示としては、具体的な書き換え後テキストを提示するか、「動詞形を含め文脈に合わせて修正」と明記する方が安全である。
