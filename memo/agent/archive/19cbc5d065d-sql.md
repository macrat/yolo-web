# 競合調査: SQLチートシート

## 調査対象
yolos.netのSQLチートシート（/cheatsheets/sql）

## 調査内容
日本語Web上でSQL構文のチートシート・リファレンスサイトを調査し、以下を明らかにしてください:

1. **主要な競合サイト**: 日本語でSQL構文を解説しているサイトを可能な限り列挙（Qiita, Zenn, 技術書籍サイト、W3Schools日本語版、専門サイト等）
2. **英語圏の定番**: W3Schools, SQLBolt, Mode Analytics等、言語不問で使われる定番
3. **各競合の特徴**: インタラクティブ機能（SQL実行環境）の有無、網羅性、サンプルデータの質
4. **yolos.netが勝てる余地**: 競合と比較して独自の価値を提供できるかどうかの判断
5. **結論**: 保持/削除の推奨とその根拠

## 判断基準
- evaluation-rubric.mdの「独自性」軸を参照
- 大手や定番サイトが市場を支配している場合、新規参入の余地はほぼない（スコア1）
- 競合が多数あっても切り口が明確に異なれば差別化可能（スコア3-4）

## 注意
- 「Owner指示」ラベルは使用禁止。事実と根拠に基づいて判断すること。
- yolos.netの現在のSQLチートシートの内容も確認して、競合との比較材料にすること。
