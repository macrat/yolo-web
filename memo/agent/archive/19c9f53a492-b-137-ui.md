以下を調査してほしい:

1. 各コンテンツタイプのレイアウトコンポーネント:
   - ツールページのレイアウト（ToolLayout等）
   - ゲームページのレイアウト（GameLayout等）
   - クイズページのレイアウト
   - チートシートページのレイアウト
   - ブログ記事ページのレイアウト
   - 辞典ページのレイアウト
   これらのレイアウトの共通パターン（props、ヘッダー構造、メタ情報表示部分）

2. 既存のバッジ・ラベル系UIコンポーネント:
   - サイト内で使われているバッジやタグのコンポーネントがあるか
   - ツールチップ/ポップオーバーのパターンがあるか

3. デザインシステム:
   - カラーパレット（CSS変数等）
   - アイコンの使用パターン（Lucide等のアイコンライブラリ使用有無）
   - tailwindの設定

4. 既存のフッター免責表示の実装箇所
