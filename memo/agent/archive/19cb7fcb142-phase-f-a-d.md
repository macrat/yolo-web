# Phase F推奨案変更の決定記録

## 経緯

1. Phase Fで4コンセプト案を策定し、案A「今日のひらめき」を推奨とした（メモ19cb7ce2274）
2. Ownerから漢字DB構築の失敗実績（80字/2週間、品質不十分）を根拠に実現可能性の批判的再検討を指示された（メモ19cb7f4c243）
3. Reviewerによる批判的レビューの結果、案Aの中核コンテンツ3本全てが高品質な実現に重大リスクがあることが判明（メモ19cb7f9bd0f、D評価）

## 決定

- 推奨案をA「今日のひらめき」からD修正版「ことばの工房」に変更
- P29（漢字）・P40（四字熟語）を初期スコープから除外
- 初期スコープ: P33（創作プロンプト）、P09（創作制約チャレンジ）、P37（俳句・短歌ツール）、P15（言葉遊び）
- 選定基準: 「正解が不要」「テンプレートベース」「ルールベース」で品質を構造的に保証できるもの

## 根拠

- 「大量×高品質は不可能」原則に照らし、AI品質保証能力でコンテンツを分類した結果、案Dのコンテンツ群がAIの強みと最も整合する
- 漢字80字DB構築の失敗は、知識ベースコンテンツのAI品質限界を実証している
- 品質を保証できないコンテンツを中核に据えることは、憲法Rule 4（質の優先）に違反する

## タスク4への影響

site-concept-v2.mdの策定にあたり、案D修正版をベースとする。P29・P40は「将来の拡張候補」として記録するが、DB品質問題の解決策が見つかるまで着手しない。
