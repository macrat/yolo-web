## 調査項目

1. **技術調査**: Next.js (App Router) でRSSフィードを実装する方法を調査してください。
   - route handler (app/feed/route.ts) 方式
   - 静的生成方式
   - 既存のライブラリ（feed, rss等）の比較

2. **RSS仕様**: RSS 2.0 / Atom の仕様の違いと、どちらを採用すべきか（または両方）。

3. **既存コードの確認**: 当サイトのブログ投稿の管理方法（src/lib/blog.ts等）を確認し、RSSフィードに必要なデータがどこから取得できるか調べてください。

4. **SEO/ディスカバビリティ**: RSSフィードのHTMLへの埋め込み方法（linkタグ）、検索エンジンへの通知方法等。
