# 追加コンテンツ候補10件の市場調査・5軸評価

## 目的

以下の10件のコンテンツ候補について、Phase Eと同じ手法で市場調査を実施し、5軸評価を行ってください。これらは強制発想法の過程で生成されなかった類型の候補であり、既存候補との公平な比較のために同一基準で評価する必要があります。

## 評価基準（Phase Eと同一）

| 軸 | 説明 |
|---|---|
| 独自性 | 日本語圏に同種のWebサービスが存在しないか。5=競合なし、1=飽和市場 |
| 需要 | ターゲット層の規模と来訪頻度の期待値 |
| 実装可能性 | Next.js静的サイト、外部API/DB/認証なし、クライアントサイド処理で実装可能か |
| 継続性 | 定期更新や新規コンテンツ追加の容易さ |
| 品質達成可能性 | AIエージェントが高品質に作成・維持できるか。漢字80字DB構築に2週間以上かかり品質も不十分だった実績を踏まえること |

総合 = 5軸の平均。3.8以上をAランク（有力候補）とする。

## 追加の評価軸: AIエージェント品質保証分類

各候補について、以下の3群に分類してください:
- **A群**: 高品質保証可能（テンプレートベース、ルールベース、正解不要、データ不要など）
- **B群**: 条件付きで品質保証可能（条件を明記）
- **C群**: 品質保証困難（大量の正確なデータが必要、AIハルシネーションリスクが高い等）

## 候補一覧

### Q01: 都道府県×アニメ聖地マッチングクイズ
概要: 都道府県とアニメ・漫画の舞台（聖地）をマッチングするクイズゲーム。地図上で聖地を探すインタラクティブ要素付き。
技術要件: 47都道府県×主要聖地のデータベースが必要。

### Q02: 和歌×歴史背景インタラクティブ体験
概要: 百人一首等の和歌を、詠まれた時代背景・作者の人生と合わせてインタラクティブに体験するコンテンツ。歌の意味を現代語訳・歴史コンテキストとセットで提供。
技術要件: 百人一首100首分の解説データ。

### Q03: 伝統色×日常物カラーマッチングゲーム
概要: 日本の伝統色が実際にどこで使われているか（着物、食器、建築、季節の風景等）を当てるゲーム。色見本を見て「この色はどこで見かける？」を選択。
技術要件: 伝統色×使用場所/物のマッピングデータ。

### Q04: ○×ゲーム進化形（テーマ変更+タイムアタック+特殊ルール）
概要: 誰もが知る○×ゲーム(三目並べ)に独自のひねりを加えたバリエーション集。絵文字テーマ（犬vs猫、寿司vs🍜等）、タイムアタックモード、4×4盤面、重力モード（コネクトフォー風）などのルールバリエーション。CPU対戦あり。
技術要件: ゲームロジックのみ。データベース不要。

### Q05: ワードカジノ（トランプゲーム×日本語）
概要: ブラックジャック風のカードゲームだが、カードに数字ではなく日本語の文字（ひらがな/カタカナ/漢字）が書かれており、引いた文字で単語を作ると得点になる。文字の難しさがカードの「点数」に相当する。
技術要件: ゲームロジック＋基本的な日本語辞書データ（単語存在チェック用）。

### Q06: スライドパズル×伝統色グラデーション
概要: タイルスライドパズルのタイルが色のグラデーションになっており、正しい色順に並べるパズル。日本の伝統色を使った美的体験。難易度別（3×3から5×5）。
技術要件: パズルロジックのみ。データベース不要。

### Q07: 統合カラーパレットビルダー
概要: ユーザーが自ら色を選んで組み合わせるカラーパレット作成ツール。配色ルール提案、アクセシビリティチェック（コントラスト比）、CSSエクスポート、日本の伝統色からの選色モード、画像からの色抽出など、複数機能が有機的に統合された高度なツール。
技術要件: 色彩理論のアルゴリズム。データベース不要。

### Q08: キャラ付きデイリーおみくじ
概要: 毎日異なるおみくじを引けるが、普通のおみくじではなく独自のキャラクター性を持つ。「やけに具体的なラッキーアイテム（例: 左から3番目のコンビニのおにぎり）」「達成困難なアドバイス（例: 今日は空を3時間見つめると吉）」「方言バージョン」「やたら褒めてくれるバージョン」「やたら厳しいバージョン」など、複数のキャラクターから選べる。安価なLLM（Haiku等）で文面を大量事前生成。
技術要件: 事前生成済みテキストデータ。ランタイムAPI不要。

### Q09: 職業別ユーモア占い
概要: 「エンジニア占い」「営業占い」「学生占い」など、職業・立場別に特化したユーモア占い。業界用語を使った運勢（「今日のコードレビューは吉」「今日のプレゼンは大凶だが資料は神」等）。日替わりで毎日更新。
技術要件: 職業別テンプレート＋事前生成済みテキスト。

### Q10: AI目線の四字熟語用例集
概要: 四字熟語の用例をAI・ロボット目線で書いた面白い例文集。「一期一会」→「サーバー再起動のたびに揮発するセッションデータ。まさに一期一会である」のようなネタ例文。四字熟語辞典としての正確な解説＋ユーモア例文の二本立て。
技術要件: 四字熟語×ユーモア例文のデータ。LLMで生成可能。

## 各候補の評価手順

1. 日本語圏の既存競合サービスを調査
2. 5軸スコアを付与
3. A/B/C群の分類と理由
4. Aランク(3.8以上)かどうかの判定
