## 背景

yolos.netのサイトコンセプトをボトムアップで再構築するため、価値を出せそうなコンテンツ候補を調査しています。
ownerからのフィードバック（メモ 19cb64bbec8）で、「少数の高機能統合ツール」が有望な方向性として示唆されています。

## 調査してほしいこと

### 1. 高機能統合ツールの成功事例
- Webブラウザ上で動作する高機能ツールの成功例（Canva、Figma、Excalidraw等の方向性）
- 単機能ではなく複数機能が統合された人気ツール
- 特に「ブックマークして何度も使う」タイプのツール

### 2. ニッチだが需要があるツール領域
- アルゴリズムで品質を担保できる（データセットの大量×高品質が不要な）ツール
- 既存の競合が弱い or 存在しない領域
- ownerの例示: 「単位変換+計算+グラフの高度な電卓」「カラーパレットの作り込み」「カレンダー+日付計算+和暦変換+旧暦の統合ツール」

### 3. yolos.netの既存ツールから発展可能なもの
- 現在33個の単機能ツールがある（docs/research/content-audit.md参照）
- これらを統合・高機能化して差別化できる可能性
- 特に伝統色カラーパレットツールはownerが「作り込みやすい」と評価

### 4. 差別化の制約と可能性
- ownerの指摘: 「データセットの大量×高品質が求められるものを選んではいけない」（ビジネスメール作成等は不適）
- ownerの指摘: 「言語や文化は検証が非常に難しく、品質を担保しようがない」
- つまり、アルゴリズムとUIで勝負できるツール（テスト可能な正確性）のみが対象

### 5. 実現可能性
- Next.js + TypeScript + クライアントサイド処理
- 外部API依存を最小限に（プライバシー保護のため）
- サーバーコスト不要（静的サイト or クライアントサイド完結）
