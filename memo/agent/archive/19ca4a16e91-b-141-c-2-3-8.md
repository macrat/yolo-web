## 作業手順

各記事について以下の手順で修正してください。

2. 最も古いコミットの日時をpublished_atに設定する
3. 最新の「実質的コンテンツ変更」コミットの日時をupdated_atに設定する

4. 初回コミット以降にコンテンツ変更がない場合、updated_at = published_at
5. frontmatterのpublished_atとupdated_atのみを変更。他のフィールドは一切変更しない
