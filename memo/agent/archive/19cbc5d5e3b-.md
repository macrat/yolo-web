# 競合調査: 画像リサイザーツール

## 調査対象
yolos.netのimage-resizer（/tools/image-resizer）

## 調査内容
日本語・英語Web上でブラウザベースの画像リサイズツールを調査し、以下を明らかにしてください:

1. **主要な競合サイト（日本語）**: 画像リサイズ、画像縮小等で検索されるサイト（iLoveIMG日本語版、Squoosh、バナー工房等）
2. **主要な競合サイト（英語）**: Squoosh, iLoveIMG, TinyPNG, Photopea等
3. **各競合の特徴**: 対応フォーマット、バッチ処理、プライバシー（サーバー送信 vs ブラウザ完結）、UXの質、機能の豊富さ
4. **yolos.netが勝てる余地**: 「ブラウザ完結でプライバシー重視」という切り口が競合にない価値かどうかの判断（Squooshも同じアプローチではないか等）
5. **結論**: 保持/削除の推奨とその根拠

## 判断基準
- evaluation-rubric.mdの「独自性」軸を参照
- 大手や定番サイトが市場を支配している場合、新規参入の余地はほぼない（スコア1）
- 「ブラウザ完結」はSquoosh（Google製）が既に実現している可能性が高い点に注意

## 注意
- 「Owner指示」ラベルは使用禁止。事実と根拠に基づいて判断すること。
- yolos.netの現在のツールの内容も確認して、競合との比較材料にすること。
