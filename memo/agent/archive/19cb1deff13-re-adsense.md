## 調査結果: AdSense再申請待機期間の根拠

### 調査の結論

**Google公式ドキュメントには「X日待ってから再申請すること」という明示的な待機期間の規定は存在しない。**

ただし、複数回の不承認を受けた場合、Googleは自動的に審査リクエストをロックし、次回リクエスト可能な日付をアカウント内に表示する仕組みがある。

---

### 1. 公式ドキュメントに待機期間の記述はあるか

**結論: 明示的な「X日待機」規定は存在しない**

Google公式ヘルプ（support.google.com/adsense）を複数確認したが、「不承認後に何日待ってから再申請すること」という具体的な待機期間の記述は確認できなかった。

公式ヘルプページの中で確認できた関連する記述：
- AdSenseアカウントのサイト審査は「通常数日以内に完了するが、2〜4週間かかることもある」（審査所要時間についての記述）
- アカウント有効化は「6ヶ月以内に完了する必要がある」（申請の有効期限）

出典:
- https://support.google.com/adsense/answer/76228?hl=ja （AdSense アカウントを有効にする）
- https://support.google.com/adsense/answer/10246390?hl=ja （閉鎖したAdSenseアカウントを再開する）

### 2. 公式に推奨される待機期間

**結論: 公式の「推奨待機期間」は存在しない。ただし、審査リクエスト回数の上限は実装されている。**

公式ヘルプ（support.google.com/adsense/answer/12170222）の記述として複数のサイトが引用している内容：
> 「サイトが最近複数回の審査を受け、不承認となった場合、[審査プロセスを開始]ボタンは無効になります。いつサイトの審査を再度リクエストできるようになるかについては、提示された日付をご確認ください。」

また、ページレベルの審査については：
> 「30日間にリクエストできるページ単位の審査の数には上限があり、この上限は毎日更新される」
（出典: support.google.com/adsense/answer/7003627）

**これらが示す実態：**
- 単発の不承認後は即座に再申請可能
- 短期間で複数回不承認になると、システムが自動的に次回申請可能日をロックする
- その待機期間は固定ではなく、アカウント内に表示される日付で確認する仕組み

### 3. 「スパム判定」という懸念の根拠

**結論: 「スパム判定」という表現の公式根拠は存在しない。実態はより穏やかな「リクエスト回数上限」システム。**

インターネット上で広まっている「短期間での繰り返し申請がスパム判定される」という情報については、Google公式ドキュメントに直接の根拠は見当たらなかった。

実際には：
- 繰り返し不承認を受けると、Googleシステムが自動的に「審査プロセスを開始」ボタンをロックし、次回可能日を表示する
- これはスパム判定というよりも、審査リクエスト数の上限（rate limiting）として機能している
- 上限に達したユーザーに対して「サイトの審査のリクエスト回数が上限に達したため、まだサイトの再審査をリクエストできません」というメッセージが表示されるとのコミュニティ報告がある

出典:
- https://support.google.com/adsense/thread/322362181 （Googleコミュニティ: 審査リクエスト上限に達した事例）
- https://support.google.com/adsense/answer/7003627?hl=ja （広告配信に影響を与えるポリシー違反を解決する）

### 4. 「2週間待つ」という情報の出処

**結論: 2週間ルールはデマ（誤情報の拡散）であり、公式根拠はない。**

日本語圏でよく見られる「2週間空けてから再申請する」という情報の出処については、以下の説明が有力：

> 「アカウントをキャンセルして別のGoogleアカウントで新規申請する際に、古いアカウントの紐付け解除に最大2週間かかる」という文脈から「2週間」だけが独り歩きした

つまり、この「2週間」は「不承認後の再申請の待機期間」ではなく「アカウント切り替え時のキャッシュクリア待機期間」であり、通常の再申請には全く関係しない。

複数のブログ記事も「2週間待つ必要はない」と明示しており、不承認当日に修正して即日再申請し合格した事例も報告されている。

出典:
- https://support.google.com/adsense/thread/259241492 （コミュニティ: AdSense再申請の方法）

### 5. 実際のベストプラクティス（公式以外の知見）

公式の明示的な指針はないが、複数の信頼できる情報源が示す実態：
- 問題が修正できていれば即日再申請でも問題ない
- Googleクローラーがサイトを再クロールするまでの時間（数日〜2週間程度）を考慮すると、コンテンツ改善後2週間程度待つのは合理的
- ただし「X日待たなければならない」という義務的な規定はない

### 総括

| 質問 | 回答 |
|------|------|
| 公式の待機期間規定 | 存在しない |
| 推奨待機期間 | 公式に推奨はない（改善完了後すぐに申請可能） |
| スパム判定の根拠 | 存在しない（実態は回数上限によるロック） |
| 「2週間ルール」の根拠 | アカウント切り替え時の話が誤って伝わったデマ |
| 繰り返し申請のリスク | 回数上限に達すると一時的にロックされ、指定日まで待機が必要 |
