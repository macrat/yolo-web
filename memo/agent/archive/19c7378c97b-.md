## 概要
サイクル11で追加したチートシート機能を紹介するブログ記事を作成してください。

## 背景
- サイクル11でチートシートセクション（/cheatsheets）を新規追加した
- 3つのチートシート: 正規表現、Git、Markdown
- 既存のツールページ（/tools）と同じパターンでナビゲーション・レイアウトを統一

## 要件
1. src/blog/ にブログ記事を新規作成（既存のブログ記事の構造・パターンに従うこと）
2. フロントマターの日時はISO 8601形式で、現在時刻を設定すること（例: 2026-02-19T10:30:00+09:00）
3. draft: false にすること
4. 記事内容:
   - チートシートセクションの紹介
   - 各チートシート（正規表現・Git・Markdown）の概要と特徴
   - ツールページとの関連性
5. 既存ブログ記事のトーンと構成に合わせること
6. 日本語で記述すること

## 受入基準
- npm run build が通ること
- ブログ記事がビルドに含まれること
- npm run typecheck, npm run lint, npm run format:check が通ること
