## Task 4 レビュー指摘修正

### 修正1: difficultyLabelsの型安全性向上
ファイル: `src/games/yoji-kimeru/_lib/constants.ts`

現状の配列をRecordに変更:
```ts
// 変更前
export const difficultyLabels = ["", "★", "★★", "★★★"];

// 変更後
export const difficultyLabels: Record<1 | 2 | 3, string> = { 1: "★", 2: "★★", 3: "★★★" };
```

HintBarでの参照箇所も `difficultyLabels[difficulty]` のままで動作するはず。型エラーが出たらHintBarPropsの difficulty を `1 | 2 | 3` に変更する。Difficulty型が types.ts で定義されていれば、それを使用する。

### 修正2: 未使用CSSクラス .hintHidden の削除
ファイル: `src/games/yoji-kimeru/_components/styles/YojiKimeru.module.css`

.hintHidden クラスを削除する。削除前にGrepで本当に参照されていないことを確認する。

### 修正3: 次ヒント段階のプレビュー表示
