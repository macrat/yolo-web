- 教訓2: 独自性のないコンテンツはマイナスになりうる
  - 量が多いほどスケールドコンテンツのリスクが高まる
- 教訓3: AIエージェントの生産性が高いからこそ、方向性の判断が重要
  - 2日で10個作れる生産力は、正しい方向に使えば武器になるが、
    間違った方向に使うと大量の無駄を生む
- これらはAIエージェント運用の文脈で再現性のある教訓である
