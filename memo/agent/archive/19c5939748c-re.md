## Summary

オーナーから、漢字カナール（/games/kanji-kanaru）のデザインが他ページと異なり、ナビゲーション動線がない問題が指摘された。全ページのデザイン統一を行う計画を策定してほしい。

## Context

### 現状の問題

1. **漢字カナールページ（/games/kanji-kanaru）が独自レイアウト**:
   - `layout.tsx` で独自の `gameLayout` / `gameMain` / `gameFooter` を使用
   - 共通コンポーネント Header / Footer を使っていない
   - 他のページへのナビゲーション動線がゼロ

2. **独自CSSカスタムプロパティ**:
   - `KanjiKanaru.module.css` で `--kk-*` 独自変数を定義（`--kk-color-bg`, `--kk-color-text` 等）
   - サイト全体の CSS変数（`--color-text`, `--color-bg`, `--color-border` 等）を使っていない

3. **ゲーム一覧ページ（/games）は問題なし**:
   - 共通 Header + Footer を使用
   - カードグリッドレイアウトでツール一覧と同様のパターン

### 参考ファイル

- `src/app/games/kanji-kanaru/layout.tsx` — 独自レイアウト（Header/Footer なし）
- `src/app/games/kanji-kanaru/page.tsx` — GameContainer のみ render
- `src/components/games/kanji-kanaru/styles/KanjiKanaru.module.css` — 独自変数定義
- `src/app/games/page.tsx` — ゲーム一覧（正常、参考用）
- `src/components/common/Header.tsx` — 共通ヘッダー（ゲームリンクあり）
- `src/components/common/Footer.tsx` — 共通フッター

### ゴール

- 漢字カナールページに共通 Header + Footer を表示
- サイト全体のデザイン言語（CSS変数、レイアウトパターン）に統一
- ゲーム固有のCSS（ボード、タイル色など）は維持しつつ、基本レイアウト/色はサイト共通変数を使う
- パンくずリストなどのナビゲーション動線を追加
