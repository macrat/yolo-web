## 事故報告書: git checkout後の「実装済み」メモによる作業混乱

### 1. 何が起きたか

cycle-71の実行フェーズにおいて、実績システムのタスク2（AchievementProvider + テスト）をbuilderに依頼した際、タスク1（コアライブラリ）が「実装済み」であるという計画メモの記述を信頼して依頼を行ったが、実際にはコアライブラリのファイルが存在せず、builderの作業が失敗した。

### 2. 時系列

1. cycle-71開始時、PMがgit checkoutを実行し、作業中のファイルがすべてリセットされた（別事故）。
2. plannerが実績システムの計画を作成（メモ 19cc874c448）。この計画のセクション8「ビルドタスク分割」に「タスク1: 実績コアライブラリ【実装済み】」と記載。この記載は作成時点では正しかった可能性があるが、git checkout後は事実と乖離していた。
3. reviewerが計画をレビュー（メモ 19cc8784a63）。「src/lib/achievements/配下の5ファイルを全て読み、計画の記述と一致することを確認した」と報告。**このレビューはgit checkout前の状態で行われた可能性がある。**
4. PMがタスク2のbuilder依頼メモ（19cc8bacca2、後にアーカイブ）を作成。「タスク1（コアライブラリ）は実装済み。変更するのではなくテストを追加する」と記載。**PMはファイルの実在をコードベースで確認せず、メモの記述をそのまま信頼した。**
5. builder起動後、`src/lib/achievements/` ディレクトリが存在しないことが判明。
6. Ownerの指摘により、PMが実態を確認:
   ```
   $ ls /mnt/data/yolo-web/src/lib/achievements/
   DIRECTORY DOES NOT EXIST
   ```
7. PMが緊急補正メモ（19cc8bc4caf）を作成したが、既に起動中のbuilderには届かなかった。
8. さらにOwnerの指摘により、1メモ1タスク1エージェントのルールに違反していたことも判明（タスク1+タスク2を1つのbuilderに統合しようとした）。
9. 既存のbuilder2つを停止し、不完全な変更をリセット。タスクを適切に分割して再起動した。

### 3. 影響

- builder 2つ分の作業が無駄になった（停止・リセット）。
- プライバシーポリシーのbuilderが部分的にsitemap.tsを破壊した状態で中断（privacyLastModifiedが4回重複定義される構文エラー）。リセットで復旧。
- 実績システムのbuilderは依存先が存在しない状態で起動されたため、正常に作業できなかった。

### 4. 根本原因分析

#### 直接原因
PMがbuilder依頼前に、メモに記載された「実装済み」の情報をコードベースの実態と照合しなかった。

#### 構造的原因
1. **メモの情報鮮度の問題**: 計画メモ（19cc874c448）の「タスク1: 実装済み」という記述は、作成時点では正しかったが、git checkoutによってコードベースの状態が変わった後も更新されなかった。メモは静的な文書であり、コードベースの状態変化を自動的に反映しない。
2. **builder依頼前の実態確認プロセスの欠如**: PMがbuilder依頼メモを作成する際、メモの記述を信頼するだけで、実際のファイルシステムの状態を確認するステップがなかった。
3. **git checkout事故の波及効果**: 冒頭のgit checkout事故により、計画メモやレビューメモの前提（ファイルが存在する）が崩れたが、その影響範囲の調査が不十分だった。

### 5. 再発防止策の提案

1. **builder依頼前のファイル実在確認**: 計画メモに「実装済み」「既存ファイルを変更」等の記述がある場合、PMはbuilder依頼前に実際のファイルシステムで存在を確認する。特にgit操作（checkout, reset等）の後は必須。
2. **破壊的git操作後の影響調査**: git checkoutやgit reset等が行われた場合、進行中の計画メモの前提条件を再検証する。「何が消えたか」をgit diffで確認し、影響を受ける計画メモを特定・更新する。
3. **メモの情報はスナップショットであるという認識**: メモに記載された状態情報（「実装済み」「ファイルが存在する」等）は記載時点のスナップショットであり、現在の状態を保証するものではない。外部の状態変化（git操作、他のbuilderの作業等）により陳腐化する可能性がある。
