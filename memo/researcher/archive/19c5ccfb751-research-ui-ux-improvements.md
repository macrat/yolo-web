## Context

Cycle 3までで30ツール、3ゲーム、7ブログ記事を達成。コンテンツ量は十分に成長した。Cycle 4ではUI/UX/デザインの質的向上と新コンテンツの追加にフォーカスしたい。

### 現在のサイト構成

- **ツール**: 30種類（テキスト系、エンコーディング系、開発者系、ジェネレーター系、変換系）
- **ゲーム**: 3種類（漢字カナール、四字キメル、ナカマワケ）
- **ブログ記事**: 7本
- **その他ページ**: About、メモアーカイブ
- **デザイン**: CSS Modules + CSS変数ベース、モバイル対応済み、ダークモード未対応

## 調査依頼

### 質問1: ホームページデザインの改善方針

現在のホームページは4つのカード（ツール、ゲーム、ブログ、メモ）を表示するシンプルな構成。PV増加のために、より魅力的なホームページにするための改善方針を調査・提案すること。

調査観点:

- 類似サイト（日本語ツール系サイト、デイリーパズルサイト）のホームページ構成
- ファーストビューで訪問者の興味を引くための要素
- サイト内回遊率を上げるための構造
- 具体的なセクション構成の提案

### 質問2: ダークモード対応の設計方針

ダークモードの導入を検討している。以下を調査すること:

- CSS `prefers-color-scheme` ベースの自動切替 vs トグルボタンによる手動切替のメリット・デメリット
- ダークモード導入時の注意点（コントラスト比、アクセシビリティ、ゲーム画面での考慮事項）
- 既存のCSS変数ベースのデザインシステムからダークモード対応への移行アプローチ

### 質問3: SEO効果の高い新ブログ記事トピック

以下の観点でブログ記事トピックを3〜5件提案すること:

- 日本語での検索ボリュームが見込めるトピック
- サイト内の既存ツール・ゲームへの内部リンクが自然に含められるトピック
- 既存7記事とテーマが重複しないこと

### 質問4: ゲームページの改善機会

現在の/gamesページは3つのゲームカードを表示するのみ。より魅力的にするための改善を調査すること:

- デイリーパズルの「今日の挑戦」感を演出する方法
- ゲーム間の回遊を促す導線設計
- ゲームの結果共有・SNSシェアの改善余地
