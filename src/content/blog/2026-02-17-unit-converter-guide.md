## 単位変換が必要になる場面

「インチとセンチ、どっちがどのくらい?」「華氏350度って何度?」。単位変換は意外と身近な場面で必要になります。いくつか典型的なシーンを紹介します。

### 海外通販・個人輸入

海外のECサイトで服や靴を購入するとき、サイズ表記がインチやポンドで書かれていることがあります。たとえばウエスト32インチは約81cm、体重150ポンドは約68kgです。サイズを間違えると返品の手間がかかるので、注文前に正確に換算しておくことが大切です。

### 料理・レシピ

海外のレシピサイトでは、材料の分量がオンスやポンドで、オーブンの温度が華氏で書かれています。「バター4オンス」は約113g、「オーブンを華氏350度に予熱」は約177度Cです。分量を間違えると料理の仕上がりに直結するので、こまめに換算するのが安心です。

### 不動産・住まい探し

日本の不動産では「坪」と「平方メートル」が混在しています。マンションの広さは平方メートルで表示されますが、一戸建てや土地は坪で表記されることが多いです。「30坪の土地」がどれくらいの広さか、すぐにイメージできると物件探しがスムーズになります。

### 旅行・ドライブ

アメリカやイギリスでレンタカーを借りると、速度標識はmph（マイル毎時）表記です。制限速度65mphは約105km/hですが、慣れないとスピード感覚が掴みにくいものです。気温の華氏表記にも戸惑うことが多く、天気予報を見て服装を決めるにも換算の知識が役立ちます。

## SI単位系の基本

現代の計量の土台になっているのが、国際単位系（SI: Systeme International d'Unites）です。メートル、キログラム、秒などの基本単位を定め、世界共通の計量基準として使われています。日本でも計量法でメートル法が法定計量単位として定められています。詳しくは[経済産業省の計量行政ページ](https://www.meti.go.jp/policy/economy/hyojun/techno_infra/keiryougyousei.html)や、[国際度量衡局（BIPM）のSIページ](https://www.bipm.org/en/measurement-units)を参照してください。

日常でよく使うSI基本単位は以下のとおりです。

- **長さ**: メートル（m）
- **質量**: キログラム（kg）
- **温度**: ケルビン（K）。日常的にはセルシウス度（摂氏、度C）を使う
- **面積**: 平方メートル（m2）。SI基本単位からの組立単位

SI接頭語を使えば、桁の大きさを簡潔に表現できます。ミリ（m、1/1000）、センチ（c、1/100）、キロ（k、1000倍）が日常で最もよく使われます。[NIST（アメリカ国立標準技術研究所）のSI単位ページ](https://www.nist.gov/pml/owm/metric-si/si-units)では、SI単位系の詳細な解説を読むことができます。

一方で、アメリカを中心にヤードポンド法（インチ、ポンド、華氏など）も広く使われており、国際的なやり取りでは両方の単位系を理解しておく必要があります。

## よく使う換算と覚え方

ここでは長さ・重さ・温度・面積・速度の5カテゴリについて、日常で特に使う換算値と暗算のコツをまとめます。

### 長さ

| 変換      | 換算値   | 覚え方のヒント                           |
| --------- | -------- | ---------------------------------------- |
| 1インチ   | 2.54 cm  | 親指の幅がおよそ1インチ                  |
| 1フィート | 30.48 cm | 約30cm。A4用紙の長辺（29.7cm）とほぼ同じ |
| 1ヤード   | 91.44 cm | 約90cm。両手を広げた半分くらい           |
| 1マイル   | 1.609 km | 「マイルは1.6倍」で概算                  |

海外通販での服のサイズ確認では「インチ x 2.5 = だいたいのcm」と覚えておくと便利です。たとえばシャツの首回り16インチなら16 x 2.5 = 40cmが目安です（正確には40.64cm）。

DIYや工作では、インチネジとミリネジの区別も重要です。ネジの規格が異なると合わないので、工具を選ぶ際にもインチとミリの違いを意識する場面があります。

### 重さ

| 変換    | 換算値  | 覚え方のヒント                   |
| ------- | ------- | -------------------------------- |
| 1オンス | 28.35 g | 食パン1枚がおよそ60g = 約2オンス |
| 1ポンド | 453.6 g | 約450g。ステーキ1枚分をイメージ  |

海外のレシピでは「バター1スティック = 4オンス = 約113g」のような表記が頻出します。「1オンスは約30g」と覚えておけば、オンス x 30で概算できます。

航空便の重量制限もポンド表記の場合があります。荷物の上限が50ポンドなら約23kg、70ポンドなら約32kgです。「ポンド / 2 = だいたいのkg」が簡易的な概算方法として使えます（正確には2.2で割る）。

### 温度

摂氏（度C）と華氏（度F）の変換公式は以下のとおりです。

- 摂氏から華氏: 度F = 度C x 9/5 + 32
- 華氏から摂氏: 度C = (度F - 32) x 5/9

とはいえ、公式を毎回計算するのは面倒です。よく使う対応をいくつか覚えておくと実用的です。

| 摂氏   | 華氏    | 場面                 |
| ------ | ------- | -------------------- |
| 0度C   | 32度F   | 水が凍る温度         |
| 20度C  | 68度F   | 快適な室温           |
| 37度C  | 98.6度F | 平熱の目安           |
| 100度C | 212度F  | 水の沸点             |
| 180度C | 356度F  | オーブン料理（中温） |
| 200度C | 392度F  | オーブン料理（高温） |

海外の天気予報を見るときは、「華氏から30を引いて2で割る」という簡易的な覚え方もあります。たとえば華氏80度なら (80 - 30) / 2 = 25度C。正確な値は26.7度Cですが、おおよその体感を掴むには十分です。

### 面積

不動産でよく使う面積の換算は以下のとおりです。

| 変換        | 換算値     | 覚え方のヒント    |
| ----------- | ---------- | ----------------- |
| 1坪         | 約3.31 m2  | 畳2枚分           |
| 1畳         | 約1.65 m2  | 畳1枚の面積       |
| 1ヘクタール | 10,000 m2  | 100m四方          |
| 1エーカー   | 約4,047 m2 | ヘクタールの約4割 |

平方メートルから坪への換算は「平方メートル x 0.3025 = 坪」で概算できます。つまり「平方メートルの約3割が坪数」と覚えれば十分です。たとえば70m2のマンションなら約21坪、100m2の土地なら約30坪です。

### 速度

| 変換    | 換算値       | 覚え方のヒント                |
| ------- | ------------ | ----------------------------- |
| 1 mph   | 約1.609 km/h | 「マイル x 1.6 = キロ」で概算 |
| 1ノット | 約1.852 km/h | 海上・航空で使用              |

海外で車を運転するとき、速度制限の標識がmphで書かれていると戸惑います。「mph x 1.6」を覚えておけば素早く概算できます。たとえば制限速度30mphなら約48km/h（住宅街）、60mphなら約97km/h（幹線道路）、70mphなら約113km/h（高速道路）です。

ノットは船や航空機で使われる速度の単位で、1時間に1海里（約1.852km）進む速さを意味します。天気予報で風速がノットで表示されることもあります。

## 日本の伝統単位

日本には尺貫法と呼ばれる伝統的な計量体系があります。1959年（昭和34年）の計量法改正でメートル法に一本化されましたが、一部の分野では今も日常的に使われています。

### 長さの伝統単位

| 単位 | メートル法換算 | 現代での使用場面                               |
| ---- | -------------- | ---------------------------------------------- |
| 1寸  | 約3.03 cm      | 身長の表現（五尺七寸など）、碁盤や将棋盤の厚さ |
| 1尺  | 約30.3 cm      | 建築材の寸法（尺モジュール）                   |
| 1間  | 約1.82 m       | 部屋の寸法、建築の基準モジュール               |

日本の木造建築では、現在でも「尺モジュール」（柱の間隔を3尺 = 約91cmとする設計基準）が広く使われています。1フィート（30.48cm）と1尺（約30.3cm）がほぼ同じ長さなのは偶然ですが、覚えやすい対応です。

### 重さの伝統単位

| 単位 | メートル法換算 | 現代での使用場面                                 |
| ---- | -------------- | ------------------------------------------------ |
| 1匁  | 3.75 g         | 真珠の取引、5円玉の重さの基準                    |
| 1貫  | 3.75 kg        | 1000匁。寿司の数え方「一貫」の語源とする説がある |

匁は真珠の国際取引において重量単位として使われてきた歴史があります。5円玉1枚の重さがちょうど1匁（3.75g）であるのは、旧貨幣制度の名残です。なお、寿司の「一貫」の語源については複数の説があり、重さの貫に由来するという説は有力なものの定説ではありません。

### 面積の伝統単位

坪と畳は不動産で今も広く使われています。「この土地は何坪?」「この部屋は何畳?」という会話は日常的に行われます。

- **1坪** = 約3.31 m2 = 畳2枚分。1間 x 1間の面積
- **1畳** = 約1.65 m2。ただし畳のサイズは地域によって異なる（京間・江戸間・団地間など）

不動産広告ではメートル法（m2）の表示が義務付けられていますが、坪単価で価格を比較する慣習は根強く残っています。
